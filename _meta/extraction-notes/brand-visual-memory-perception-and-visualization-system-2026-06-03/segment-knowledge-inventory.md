## 关键方法资产

1. 品牌视觉升级先判断记忆资产，不先追求新风格。
2. 视觉规划用“传承、特色、效率”三要素判断。
3. 产品功能应转化为用户可感知的视觉符号。
4. 多 SKU 和多价格带品牌需要把产品结构变成可视分层。
5. 详情页和主图应区分 A 品牌态度、B 使用场景、C 功能卖点。
6. AI 生图之前应先建立空间、人物、元素、版式四模块规则。

## 后续可复用方向

这条路径可直接服务品牌策划 HTML 报告中的“视觉规划”板块，包括视觉资产盘点、ABC 分层、参考图 prompt、AI 生图约束和详情页视觉执行建议。
