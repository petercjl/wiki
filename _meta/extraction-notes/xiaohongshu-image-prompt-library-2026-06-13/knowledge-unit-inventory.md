# Knowledge Unit Inventory

## Core knowledge units

1. 小红书图片任务：点击、搜索承接、品类识别、场景代入、痛点呈现、证据展示、收藏价值。
2. 通用 prompt 字段：笔记类型、品类/产品、人群、场景、痛点、结果、标题、排版、负面约束。
3. 封面类型：痛点型、结果型、场景型、清单/攻略型、对比测评型、爆文首图、关键词封面。
4. 组图结构：封面、痛点/前情、使用场景、细节/证据、对比/方法、总结/购买理由。
5. 变量库：人群词、场景词、痛点词、效果词、风格词、图片逻辑词。
6. 流量场景：信息流素材与搜索承接素材的差异。
7. 卖点转译：兴趣型、功能型、场景型卖点转成画面语言。
8. 搜索关键词预埋：标题、标签、画面语义同时覆盖关键词。
9. 评论区/总结转化：适合谁、解决什么、怎么选、注意事项。
10. QA 约束：手机端可读、真实生活感、弱营销、避免错误中文和 AI 感。
