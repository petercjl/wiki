## Evidence Rule

For short-video ingestion, when a knowledge point depends on what an asset looks like, the screenshot is part of the knowledge unit. ASR alone is insufficient for:

- role / identity board layout
- storyboard panel structure
- motion-arrow encoding
- director-table spatial staging
- product / character / scene asset library examples
- UI workflow steps and prompt templates shown on screen
