# Course Reconstruction

山峰组259期6-15/6-16课程讲的是：在淘系存量市场、流量分散、AI和智能计划逐步改变运营方式的背景下，淘宝运营不能只盯推广按钮，而要先判断店铺增长模型、产品转化周期、人群资产质量和流量渠道，再选择标准计划、智能计划、搜索卡位、人群推广、流量金卡、相似品跟投等工具，并用报表判断计划是否拉对人、是否有成交机会、是否能放大。

## Core Model

```text
平台趋势 -> 店铺模型 -> 产品转化周期 -> 人群资产 -> 流量渠道 -> 计划类型 -> 指标归因 -> 放量/纠偏
```

## Relationship To Previous Courses

- 253期讲全站推广打款放量，是“款进入付费放量后怎么操作”。
- 256期讲产品类型与生命周期，是“什么款适合什么指标和打法”。
- 259期讲淘宝智能推广与人群运营，是“淘宝后台里如何看人群、选计划、控归因、判断计划健康”。

## Key Contributions

1. **曝光价值公式**：搜索曝光价值 = 搜索点击率 × 搜索转化率 × 客单价。搜索起不来时，先对比自己和同行的曝光价值，而不是只问关键词有没有流量。
2. **店铺模型**：721、172、333、127 不是绝对分类，而是帮助商家判断销售额来自拉新、潜客回访、老客复购还是均衡结构。
3. **人群资产诊断**：流转效率、人群资产规模、TA人群浓度共同决定成交规模。看见只是曝光问题，买单才是人群和转化问题。
4. **转化周期与归因**：短转化周期看直接成交；长转化周期看加购、潜客和转化周期回收；ToB/咨询型看询单和询单成本。
5. **计划类型匹配**：标准计划负责精准和人群纠偏；智能计划负责效率、宽度和收割；最大化适合抢量但会推高 PPC；控投产稳但难放大。
6. **工具层落地**：测图测款、趋势明星、好货快投、相似品跟投、搜索卡位、流量金卡、人群推广都要服务于产品阶段和人群模型。
