# Formal Page Plan

## Formal Pages

- `domains/量化交易/04-量化理论基础/index.md`：理论基础入口。
- `01-策略收益来源分解.md`：把收益拆为宇宙/选池、信号、组合、买卖器、执行、市场环境。
- `02-选池、信号与组合构建.md`：把 ETF 池和信号当成 alpha 研究对象，先做相关性/分组/换手验证。
- `03-买卖器、执行与风控.md`：定义买入卖出器的职责、资金利用率与风险控制边界。
- `04-策略评估指标与归因.md`：构建回测结果分析模板。
- `05-避免过拟合与回测陷阱.md`：建立参数搜索、时间序列验证、未来函数和选择偏差规则。

## Fusion Notes

- 与 `00-股票量化知识地图`：新页补充更可执行的研究方法。
- 与 `01-聚宽`：平台页负责代码/回测兼容；理论页负责研究判断。
- 与 `03-策略案例`：案例页后续应按本理论层复盘。
