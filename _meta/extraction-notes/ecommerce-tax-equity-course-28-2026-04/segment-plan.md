# Segment Plan｜财税股权 28 期课程知识系统

## Segments

1. S1/D1 上午：电商企业风险基因、四本账、股东账/业务账/管理账/风险账。
2. S2/D1 下午：税务合规路径、税种逻辑、税务风险排雷、主体与资金隔离。
3. S3/D2 上午：股权架构、持股主体、控制权、激励、分红与退出。
4. S4/D2 下午：战略管理、经营分析、预算管理、竞争战略与财务指标。

## Rebuild Logic

The formal learning path is organized by knowledge dependency, not classroom chronology:

风险认知 → 四本账 → 税务合规 → 税负模型 → 主体架构 → 业财核算 → 股权控制 → 激励治理 → 资源整合 → 战略财务 → Agent 诊断模板。
