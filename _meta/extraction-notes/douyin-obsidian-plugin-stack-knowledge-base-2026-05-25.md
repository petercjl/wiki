## 可稳定提取的核心知识

1. Obsidian 第三方插件可按四层理解：基础增强层、AI 增强层、导航与发现层、内容创作层。
2. 推荐插件清单：BRAT、Claudian、Terminal、Git、Notebook Navigator、Excalidraw、Local Images Plus。
3. BRAT 的核心价值：安装未进入社区市场的 Beta 插件，简化从 GitHub 手动拷贝 `main.js / manifest.json / styles.css` 的繁琐流程。
4. Claudian 适合文件数较少的单次操作；涉及较多文件、跑 coding CLI 或开发插件时，更适合 Terminal + Claude Code / Codex CLI / AI IDE。
5. Git 在知识库场景中的核心价值是备份、同步、自动提交与回滚。
6. Notebook Navigator 的价值是把文件夹、最近文件、标签等高频导航能力集中到更高密度的浏览体验中。
7. Excalidraw 的价值不是单独画图，而是深度融入 Obsidian 的双链、图谱、反链、标签与元素级链接。
8. Local Images Plus 的价值不仅是下载网页图片，还包括附件路径管理与离线图片资产沉淀。
9. 插件不是越多越好，应按长期持续使用价值筛选。
