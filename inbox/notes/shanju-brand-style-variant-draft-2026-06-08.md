# 杉居品牌风格变体推演草稿

> 这是一份待审阅草稿，不是正式知识库页面。  
> 目标：基于“极简北欧风”母体，结合杉居品牌情绪板，推演出一个可持续扩展、可用于 AI 生图的品牌化风格变体。

## 1. 先给结论

“杉居”不应该被定义成普通的“极简北欧风”。它更像是一个从极简北欧风里生长出来的品牌世界：

> **杉居轻厨房生活风**：以厨房工具为品牌产品母题，以明亮室内厨房、露台、庭院、野餐和户外烧烤为生活场域，以猫的陪伴、温柔女性日常、浅木白绿空间和轻料理分享为核心的北欧轻生活品牌风格。

这个风格的核心不是“家居陈列很干净”，也不是“只能发生在室内厨房”，而是：

- 用户在一个清晨、午后或傍晚，用厨房工具完成一件让生活变温柔的小事：烘焙、调味、煎烤、收纳、分享。
- 产品不是孤立货品，而是参与“光、厨房工具、食物、猫、你、生活”的关系。
- 品牌不是卖一个功能，而是卖“被温柔照料的日常秩序”。

如果通用极简北欧风的关键词是“白、浅木、自然光、留白、功能性”，那么杉居变体应该增加：

- **轻烘焙**
- **明亮厨房与户外轻料理**
- **橘猫陪伴**
- **年轻女性居家状态**
- **淡蓝衬衫**
- **鼠尾草绿小物**
- **刚出炉的松软**
- **安心托付**
- **轻烧烤 / 露台料理 / 野餐分享**
- **live gently**

## 2. 情绪板观察

情绪板由 4 条横向画面构成，左侧有编号、品牌名、英文小字和中文短句，右侧是明亮厨房中的人物、猫、烘焙动作和食物。

### 2.1 文字系统

画面中反复出现：

- `SHANJU / 杉居`
- `LIGHT HOME BAKING`
- `CAT & KITCHEN`
- `live gently.`

中文短句包括：

- 光与厨房 / 有猫 有你 有生活
- 怡到好处的温度 / 安心托付 每一份热爱
- 刚出炉的松软 / 分享给最重要的它
- 简单的幸福 / 生活本该这样温柔

这些文字说明杉居的表达不是促销型，也不是功能硬解释型，而是轻声生活方式型。文案的节奏很短，像生活手记或品牌画册，不像电商大字报。

### 2.2 空间系统

情绪板主要展示的是明亮白色厨房：

- 大面积白色墙面和台面。
- 浅木开放架和浅木凳。
- 自然窗光，整体高亮但不刺眼。
- 白色陶瓷、玻璃杯、花瓶、鲜花、绿色植物。
- 烤箱和水槽提供真实厨房功能，不只是背景。

这不是“装修效果图式北欧风”，而是“轻烘焙发生中的厨房”。空间必须有动作发生。

但从品牌产品线看，厨房不应被理解成一个固定房间，而应理解成“料理工具被使用的生活场域”。杉居的产品如果包括烧烤盘、调味罐、厨房工具、餐厨收纳等，它们天然可以从室内厨房延展到：

- 阳台/露台：傍晚轻烧烤、朋友小聚、猫在门边或窗边看着。
- 庭院/小花园：浅木桌、白色餐具、绿植和轻烤食物。
- 野餐桌/营地桌：调味罐、烧烤盘、食材盘和浅色布巾。
- 户外烧烤场景：不是粗犷露营风，而是干净、明亮、有秩序的北欧轻户外料理。

因此情绪板是杉居的“室内厨房基准”，不是空间边界。

### 2.3 人物系统

人物是年轻女性，状态松弛、干净、温柔：

- 浅蓝宽松衬衫。
- 自然盘发。
- 表情轻柔，不强摆拍。
- 动作是倒水、烘焙、吃面包、与猫互动。

她不是“精致女主人”或“网红模特”，更像一个愿意认真生活、也愿意把温柔分给猫和家的人。

### 2.4 动物关系

橘猫是高记忆资产。

它不是装饰道具，而是品牌关系的第三个角色：有猫、有你、有生活。猫让厨房不再只是功能空间，而变成陪伴空间。

橘猫在画面里承担三层作用：

1. 情绪：增加温暖、陪伴、松弛。
2. 记忆：让用户更容易记住杉居，不只是记住白色厨房。
3. 场景：把“家”具体化为有生命互动的空间。

### 2.5 色彩系统

主色：

- 哑光白 / 暖白：厨房、墙面、台面。
- 浅木色：开放架、托盘、凳子。
- 浅蓝：人物衬衫，是杉居区别于普通米白北欧风的重要识别色。
- 鼠尾草绿：手套、植物、小厨具。
- 橘猫暖橙：天然暖色记忆点。
- 面包奶油黄：烘焙场景的食物暖色。

这套色彩比通用极简北欧风更“有角色”。通用北欧风可能只有白木绿；杉居必须保留浅蓝人物和橘猫暖橙。

## 3. 用品牌策略方法推演

本节使用 `/Users/pechen/wiki/domains/品牌策略` 中的几个方法：

- ABC 视觉分层系统：A 层品牌态度、B 层使用场景、C 层功能卖点。
- 可视化世界观四模块：空间、人物、元素、版式。
- 心向感知四入口：品牌名称、slogan、品类情绪、形象气质。
- 场景营销与情景营销：从“在哪用”升级到“为什么这件事对用户有情绪意义”。
- 品牌视觉美学：先看起来像品牌，而不是只做单张好看的图。

## 4. 杉居的心向感知

### 4.1 品牌名称

“杉居”两个字本身有天然的家居感：

- 杉：木、自然、清新、浅木、温和。
- 居：家、日常、空间、陪伴、生活秩序。

这个名字不适合走高冷设计师路线，也不适合走强促销路线。它更适合建立一种“自然、温和、可托付的居家生活感”。

### 4.2 Slogan / 价值短句

情绪板上的 `live gently.` 是很重要的英文价值句。它不是口号式 slogan，而是语气系统。

中文方向可以围绕：

- 温柔地生活。
- 把生活放轻一点。
- 有光，有厨房，有猫，也有你。
- 每一份热爱，都被安心托付。

这些句子不一定直接作为最终 slogan，但它们定义了杉居的语言气质：短、轻、克制、有人情味。

### 4.3 品类情绪

如果杉居的产品围绕厨房工具、餐厨用品、轻烘焙、烧烤盘、调味罐、收纳和宠物共居厨房生活，那么基础情绪应该不是“高级”，而是：

- 干净
- 安心
- 温柔
- 松弛
- 有陪伴
- 可被托付

这里要特别注意：杉居不能只做“漂亮厨房”。它必须让用户感觉“这个品牌懂我的生活，也懂我对猫、对家、对日常小事的认真”。

进一步说，杉居也不能只做“室内厨房”。真正的品牌母题应该是“厨房工具让生活场景变得轻、净、有陪伴”。室内厨房是一个高频入口，户外烧烤、露台轻料理、野餐分享则是同一套工具和情绪的外延。

### 4.4 形象气质

杉居的形象气质可以定义为：

> 明亮、轻柔、会照顾料理细节的生活陪伴者。

它不是：

- 高冷极简。
- 日式侘寂。
- 奶油风网红家。
- 黑白灰高级公寓。
- 强烟火气家庭厨房。
- 粗犷露营烧烤品牌。
- 强宠物品牌。

它是“轻厨房工具 + 轻烘焙 + 轻烧烤 + 猫陪伴 + 北欧明亮感”的组合。

## 5. ABC 视觉分层推演

### 5.1 A 层：品牌态度

杉居的 A 层不应该是宏大品牌宣言，而是轻生活态度：

> 生活本该这样温柔。  
> 光与厨房，有猫，有你，有生活。

A 层画面可以是：

- 明亮厨房远景。
- 露台或庭院轻料理远景。
- 人物与猫同处一个空间。
- 品牌名和短句在大留白区域中轻声出现。
- 没有强卖点，没有促销，没有大标题压迫。

A 层的任务是让用户先觉得：这个品牌有稳定审美、有温度、值得慢慢看。

### 5.2 B 层：使用场景

杉居的 B 层最强，因为情绪板几乎全部在讲 B 层。

核心场景：

- 清晨厨房：阳光进来，准备早餐或烘焙。
- 午后轻烘焙：面包出炉，猫在旁边等待。
- 傍晚露台：烧烤盘上是轻煎蔬菜、面包或肉类，调味罐有序摆在浅木桌上。
- 周末庭院/野餐：厨房工具从室内被带到户外，料理和分享仍然保持清爽秩序。
- 厨房收纳：台面干净，工具和食物被温柔组织。
- 与猫共享的家：不是“给猫买东西”，而是“我和它一起生活得更舒服”。

B 层的关键词是“关系”：

- 人与厨房的关系。
- 人与户外料理场域的关系。
- 人与猫的关系。
- 人与食物的关系。
- 产品与这些关系的连接。

### 5.3 C 层：功能卖点

C 层不能消失，但要被 B 层包住。

如果产品是烧烤盘、调味罐、收纳篮、厨房用品、烘焙用品或宠物厨房周边，功能可以这样表达：

| 功能事实 | 直接卖点 | 杉居式表达 |
| --- | --- | --- |
| 烧烤盘适合多场景 | 室内外两用 | 从厨房到露台，一盘接住刚好的烟火气 |
| 调味罐便携/密封 | 户外可带 | 把喜欢的味道，整齐带到周末的风里 |
| 收纳容量 | 大容量 | 把台面还给光，把小物安放好 |
| 防烫/隔热 | 隔热耐高温 | 怡到好处的温度，安心托付每一份热爱 |
| 易清洁 | 好擦洗 | 烘焙后的厨房，也能很快回到清爽 |
| 天然材质 | 环保自然 | 和浅木、面包、猫一起，留在家的温柔质感里 |
| 适合厨房 | 厨房专用 | 让每一次轻烘焙都有自己的位置 |

杉居的 C 层不要写成参数表，而要把功能放进“刚出炉、台面、猫、早餐、露台、周末、托付”这些场景里。

## 6. 可视化世界观四模块

### 6.1 空间模块

固定空间：

- 明亮家庭厨房。
- 白色台面、水槽、烤箱、开放木架。
- 露台、阳台、庭院、野餐桌、轻户外烧烤台。
- 可延展到餐桌、窗边、厨房岛台、早餐角、户外餐桌、草地野餐布。

空间质感：

- 真实生活，但被整理过。
- 清爽，不冰冷。
- 有烘焙、调味、煎烤、摆盘、分享动作，不是空房间或空桌面。
- 有猫活动的安全感。
- 户外场景要轻、净、明亮，不走粗犷露营、重油烟烧烤或硬核装备路线。

景别规则：

- 远景：建立“光与厨房/露台/庭院，有猫有你”的世界。
- 中景：展示人物动作、产品位置和猫的互动。
- 近景：解释面包、手套、收纳、温度、调味、烧烤盘、材质和细节。

光线：

- 自然窗光。
- 高亮暖白。
- 柔阴影。
- 户外可用清晨、午后或傍晚金色柔光。
- 不用戏剧性强光，不用夜晚暖黄灯，不用重烟雾炭火感。

### 6.2 人物模块

人物固定项：

- 年轻女性。
- 干净自然，不浓妆。
- 浅蓝宽松衬衫可作为标志性造型。
- 松弛、专注、温柔。

行为：

- 倒水、切水果、拿面包、开烤箱、整理台面、摆放调味罐、端起烧烤盘、给食物轻轻调味、轻抚猫、与猫对视。
- 动作要小，不要夸张摆拍。

禁忌：

- 网红摆拍感。
- 过度精致妆造。
- 职场白衬衫/西装感。
- 大幅度开心跳跃或强情绪表演。

### 6.3 元素模块

高记忆元素：

- 橘猫。
- 浅蓝衬衫。
- 白色厨房。
- 面包 / 烘焙食物。
- 烧烤盘。
- 调味罐。
- 鼠尾草绿厨房小物。
- 白陶瓷和玻璃杯。
- 鲜花 / 小绿植。
- 浅木开放架、托盘、户外浅木桌。

如果杉居未来要做 AI 生图，建议把这些元素分级：

| 层级 | 元素 | 作用 |
| --- | --- | --- |
| 核心记忆 | 橘猫、浅蓝衬衫、白厨房/浅木料理场 | 用户一眼认出杉居 |
| 场景支撑 | 面包、烤箱、水槽、木架、露台、庭院、野餐桌、鲜花 | 让“轻厨房生活”成立 |
| 产品承接 | 烧烤盘、调味罐、收纳篮、隔热手套、厨房容器、餐厨小物 | 让商品进入品牌世界 |
| 氛围补充 | 青葡萄、牛奶、白盘、玻璃杯、绿色布巾、轻烤蔬菜、面包片 | 增加轻烘焙、轻烧烤与清新感 |

禁用元素：

- 黑金奢华道具。
- 重油烟中式厨房。
- 高饱和红黄蓝。
- 大面积奶油米色网红家。
- 过多宠物用品导致变成宠物品牌。
- 过多食物堆叠导致变成烘焙品牌。
- 黑色重装备、帐篷、强户外机能风导致变成露营品牌。
- 烟雾很重、油脂飞溅、夜市烧烤摊感导致丢失杉居的清爽秩序。

### 6.4 版式模块

情绪板的版式很明确：

- 横向长图。
- 左侧留白文字区。
- 编号 `01/02/03/04`。
- 英文小字 `LIGHT HOME BAKING / CAT & KITCHEN`。
- 中文短句两到三行。
- 品牌名小而克制。
- 图像区域占大部分，文字不压画面。

这可以成为杉居的品牌版式资产。

建议保留：

- 左侧窄栏信息区。
- 小编号。
- 中英文混排。
- 小字号、字距拉开、低对比灰绿/暖灰文字。
- 文案像生活章节，而不是广告标题。

## 7. 从“场景”到“情景”的杉居表达

### 7.1 场景层

场景层回答：在哪里用，解决什么问题。

杉居的基础场景是：

- 在明亮厨房做轻烘焙。
- 从烤箱取出刚出炉的面包。
- 把厨房小物安放在干净台面。
- 在露台用烧烤盘做一顿轻松晚餐。
- 把调味罐带到庭院、野餐桌或户外烧烤场景。
- 和猫一起度过早餐/午后。

### 7.2 情景层

情景层回答：为什么这件事对用户有情绪意义。

杉居真正卖的不是“厨房用品”，而是：

- 我把普通日子照顾得有一点温柔。
- 我愿意把热爱分享给家里最重要的它。
- 我的厨房不只是做饭的地方，也是我和生活和解的地方。
- 我的厨房工具不只留在厨房，也陪我把周末搬到户外。
- 户外也可以不粗糙，烧烤也可以干净、有序、温柔。
- 简单的幸福，不需要很宏大。

### 7.3 四加场景表达

| 公式 | 杉居表达 |
| --- | --- |
| 痛点 + 场景 | 烘焙后的台面，也能回到清爽有序 |
| 情绪 + 场景 | 刚出炉的松软，分享给最重要的它 |
| 优势 + 场景 | 怡到好处的温度，安心托付每一份热爱 |
| 差异 + 场景 | 有光、有厨房、有猫，也有被安放好的生活 |
| 户外情绪 + 场景 | 把喜欢的味道，带到周末的风里 |
| 户外优势 + 场景 | 从厨房到露台，一只烧烤盘接住刚好的烟火气 |

## 8. 基于极简北欧风的差异点

| 维度 | 通用极简北欧风 | 杉居品牌变体 |
| --- | --- | --- |
| 空间 | 明亮家居、白木、自然光 | 明亮厨房、露台、庭院、野餐桌、轻户外料理场 |
| 人物 | 可有可无 | 必须有人，年轻女性是重要识别资产 |
| 动物 | 一般没有 | 橘猫是核心记忆资产 |

## 10. 后续入库前需要补充的问题

这份草稿目前只基于情绪板和已知极简北欧风母体推演。正式入库前，建议补充：

1. 杉居厨房工具系列的产品清单：烧烤盘、调味罐、收纳、烘焙工具、宠物厨房周边、餐厨用品分别占多大比重。
2. 室内/户外图位比例：产品主图、详情页、短视频封面中，室内厨房、露台庭院、户外烧烤分别承担什么角色。
3. 目标用户是谁：养猫年轻女性、精致厨房用户、轻烘焙爱好者、家居收纳用户、轻户外料理用户，还是多重人群。
4. 品牌主张是否确认：`live gently.` 是否是长期固定语句。
5. 橘猫是否必须固定：是真实品牌 IP、情绪资产，还是只是这张情绪板的辅助角色。
6. 浅蓝衬衫女性是否作为固定人物资产延续。
7. 产品图是否必须出现人物和猫，还是按 A/B/C 层级决定。
8. 是否需要把“猫”从陪伴情绪进一步发展成品牌符号或 IP。
9. 户外场景是否要限定在“轻户外/露台/庭院/野餐”，避免进入硬核露营与粗犷烧烤路线。
