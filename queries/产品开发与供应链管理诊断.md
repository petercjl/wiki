# 产品开发与供应链管理诊断

当问题涉及新品开发、产品同质化、供应商开发、供应商分级、OEM/ODM、自有品牌、打样、品质、真实净利、SKU 角色、库存周转、安全库存、补货、滞销、退货入库或部门协同时，先读取：

- [[domains/电商运营/01-通用电商方法/05-产品开发与供应链管理系统/06-Agent使用模板：产品开发与供应链诊断|Agent 使用模板：产品开发与供应链诊断]]

## 输出要求

回答时不要只给泛泛建议。必须判断：

1. 当前主要问题属于产品开发、供应商、成本利润、库存还是协同。
2. 产品是否满足“需求先于产品、利润先于销量、测试先于备货、标准先于合作、协同先于规模”。
3. 供应商是否有候选池、评分、分级和替代方案。
4. 成本是否按净利计算，而非只看毛利。
5. 库存是否按爆款、动销、滞销分层管理。
6. 补货是否结合 7/14/21 天销量、交货期和现金流。
