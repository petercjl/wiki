# 品牌策略诊断总入口

当用户要求诊断品牌、定位、品类心智、差异化、大单品、产品矩阵、品牌视觉资产、内容资产或案例类比时，先读：

1. [[domains/品牌策略/index|品牌策略知识域]]
2. [[domains/品牌策略/01-品牌基础与增长方法/index|品牌基础与增长方法]]
3. [[domains/品牌策略/02-品类心智与差异化/index|品类心智与差异化]]
4. [[domains/品牌策略/03-产品战略与大单品/index|产品战略与大单品]]
5. [[domains/品牌策略/04-品牌视觉资产/index|品牌视觉资产]]

## 分支读取

- 白牌到品牌、势能/动能/蓄能：读 [[domains/品牌策略/01-品牌基础与增长方法/01-电商品牌增长方法论/index|电商品牌增长方法论]]。
- 品类聚焦、人群认同、定位配称：读 [[domains/品牌策略/02-品类心智与差异化/01-电商品牌竞争战略：品类心智与品牌打造/index|电商品牌竞争战略：品类心智与品牌打造]]。
- 差异化感知、符号、Slogan、气质：读 [[domains/品牌策略/02-品类心智与差异化/02-品牌差异化感知系统/index|品牌差异化感知系统]]。
- 心智产品力、大单品、产品矩阵：读 [[domains/品牌策略/03-产品战略与大单品/index|产品战略与大单品]]。
- 品牌视觉资产、ABC 分层、可视化世界观：读 [[domains/品牌策略/04-品牌视觉资产/index|品牌视觉资产]]。
- 需要案例类比时，读 [[queries/品牌案例库调用入口|品牌案例库调用入口]]。

## 工作边界

- 不要把品牌诊断直接等同于 Logo、VI 或视觉升级。
- 不要只做传播建议；必须回到品类、人群、产品、渠道、视觉和内容是否配称。
- 不要凭空判断当前平台规则、投放成本或行业规模；涉及实时市场数据时要求用户提供或另行检索。

## 标准输出

```text
1. 品牌当前阶段
2. 核心问题：品类、人群、产品、视觉、内容、渠道或组织
3. 定位与配称判断
4. 大单品/产品矩阵判断
5. 品牌资产与视觉资产判断
6. 可调用案例
7. 30/90 天优先动作
```
