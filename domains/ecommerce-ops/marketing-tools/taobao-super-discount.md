# 淘宝营销工具：超级立减

## 一句话定位

超级立减是淘宝平台重要的日销营销场，核心是商品维度直降，让商家通过活动价和单品立减制造明确到手价与活动氛围。

## 适用目标

- 高频上新、爆品成交、日销转化。
- 需要商品详情链路展示“超级立减 xx 元 / xx%”利益点。
- 需要用商品维度直降，而不是店铺级券或多件优惠承接转化。
- 希望报名后长期有效，并按平台周期校验价格。

## 核心规则

| 规则项 | 当前规则 | 经营含义 |
|---|---|---|
| 招商范围 | 除部分投资贵金属等类目外，其余行业全类目可报名，具体以报名页为准 | 先以报名页校验为准。 |
| 招商模式 | 一次报入，长期有效 | 适合长期日销工具，但要持续关注活动状态。 |
| 基本玩法 | 直降 9 折；超级立减单品立减金额 >= 活动价 x 10% | 本质是商品级直接让利。 |
| 生效范围 | 活动商品 | 不是全店自动生效。 |
| 报名时间 | 2025-06-17 00:00 至 2028-07-31 23:59 | 来源白皮书中的长期报名窗口。 |
| 活动售卖 | 2025-06-21 00:00 至 2028-07-31 23:59 | 来源白皮书中的长期售卖窗口。 |

## 价格与门槛公式

```text
活动普惠券后价 = 活动价 - 超级立减单品立减 - 其他普惠优惠
活动普惠券后价 <= 最低普惠券后价
超级立减单品立减金额 >= 活动价 * 10%
自行设价时：超级立减金额 >= 活动价 * 10%，且 < 活动价 * 50%
```

超级立减建议价逻辑：

```text
活动价 = 最低标价
超级立减单品立减金额 = 最低标价 - 店铺/商品其他优惠 - 最低普惠券后价
若结果不满足默认折扣，则按最低标价 * (1 - 默认折扣) 生效，如最低标价 * (1 - 0.9)
```

## 活动周期与普惠优惠

报名超级立减的商品，店铺券等其他普惠优惠只有在有效期完整覆盖对应活动周期时，才会被纳入活动普惠券后价计算；如果没有完整覆盖周期，优惠可能仍会对消费者生效，但报名校验不会计入，容易造成资损或活动异常。

平台按月分周期进行价格校验，通常每 5 天为一个周期：1-5 日、6-10 日、11-15 日、16-20 日、21-25 日、26 日至月底。

## 可叠加 / 互斥关系

| 相关活动或工具 | 关系 | 经营含义 |
|---|---|---|
| 店铺券等其他普惠优惠 | 条件性计入 | 只有完整覆盖活动周期才计入活动普惠券后价；未完整覆盖仍可能生效，需防资损。 |
| 部分跨店优惠，如跨店满减、官方立减 | 不可叠加 | 白皮书列为不可叠加。 |
| 单品宝 | 不可叠加 | 白皮书列为不可叠加。 |
| 聚划算、百亿补贴、天天热卖、淘宝秒杀、行业大型、平台大型、平台特大型活动 | 不计入超级立减最低价 | 这些活动不计入超级立减活动的最低标价和最低普惠券后价。 |
| 主题营销、超级立减活动 | 计入超级立减最低价 | 会影响超级立减活动最低价。 |
| 行业大型、平台大型、平台特大型活动 | 超级立减计入这些活动最低价 | 超级立减可能影响后续大促最低价校验。 |
| 主题营销、聚划算、百亿补贴、天天热卖、淘宝秒杀 | 超级立减不计入这些活动最低价 | 来源白皮书中的最低价关系。 |

## 大促与双报规则

| 场景 | 规则 | 商家动作 |
|---|---|---|
| 2026 年 5-6 月平台大促含 618，2026-05-06 至 2026-06-21 | 无论商品是否报名大促，超级立减均不生效，含价格、优惠和氛围 | 不要把超级立减作为该售卖期主优惠。 |
| 618 抢先购预售 | 可以同时报名；同时报名成功后系统会将超级立减置为异常，预售结束后尝试恢复 | 关注商品活动状态。 |
| 双11预售 | 可同时报名，但在预售审核通过后，超级立减会被置为异常并不生效；若审核中未退出，可能影响预售审核 | 预售前确认状态，避免审核失败。 |
| 双11现货 | 可同时报名；现货预热至售卖结束期间超级立减不生效，结束后恢复 | 大促售卖期以大促活动为主。 |
| 平台大型活动现货，如双12、超级88 | 可同时报名现货；预热期可生效超级立减，售卖期生效大促活动，超级立减不生效 | 预热期要防多重优惠叠加。 |
| 平台大型活动预售 | 不可同时报名 | 需要二选一。 |
| 服饰新年焕新 | 超级立减长期活动商品不能同时报名；符合类目时可自动生效活动氛围 | 服饰类目需特别核对。 |
| 春节不打烊 | 可以同时报名；同时报名时春节不打烊生效，超级立减不生效 | 关注异常时超级立减是否回补生效。 |
| 超级品类日 | 可双报，但活动周期内仅生效超级品类日氛围和优惠 | 不需要退出超级立减。 |
| 主题营销 | 商品维度与超级立减报名互斥 | 已报名超级立减会导致主题营销无法发布或置为异常。 |

## 风险点

- 大促售卖期、春节不打烊、超级品类日等场景下，超级立减可能被系统置为不生效或异常，不能只看报名成功。
- 普惠优惠未完整覆盖活动周期时，报名校验与消费者实际使用优惠可能不一致。
- 超级立减可能计入部分大型活动最低价，影响后续活动报名。
- 活动价、单品立减、店铺其他优惠共同作用时，必须核算活动普惠券后价，避免资损。

## 适合使用的场景

- 日销阶段需要商品级强转化和官方活动氛围。
- 没有复杂大促、预售、主题营销冲突，且可以稳定设置活动价和普惠优惠周期。
- 希望用直接到手价推动单品成交，而不是通过领券或凑单完成优惠。

## 不适合使用的场景

- 商品正在参加大促售卖、主题营销、国补抢先购或其他明确互斥活动。
- 商家主要目标是人群/渠道定向运营，此时优惠券更合适。
- 商家主要目标是提升多件成交或客单价，此时多件优惠可能更合适。
