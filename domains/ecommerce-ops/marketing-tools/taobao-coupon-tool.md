# 淘宝营销工具：优惠券

## 一句话定位

优惠券是淘宝官方提供的商家定向优惠工具，适合按商品、店铺、人群、渠道或场景给消费者让利，以提高下单率和 GMV。

## 适用目标

- 对全店或指定商品提供明确优惠。
- 对直播间、会员、客服追单、私域链接等渠道或人群做定向转化。
- 用公开透出时间让消费者提前领券，在使用期内完成转化。
- 在优惠券与多件优惠互斥后，作为单一店铺级工具承担目标到手价。

## 核心规则

| 规则项 | 当前规则 | 经营含义 |
|---|---|---|
| 类型 | 店铺券适用于全店；商品券适用于指定商品 | 先判断优惠范围，再选券类型。 |
| 模板上限 | 店铺券同时生效上限 100，商品券同时生效上限 200 | 无效券要及时结束，避免占用额度。 |
| 创建提前期 | 最早提前 60 天创建 | 大促前可提前规划，但不要过早铺满。 |
| 有效时长 | 优惠券有效时长不能超过 60 天 | 平台希望保持下单紧迫感，避免长期无效券占用消费者卡包。 |
| 无门槛券 | 除全网自动推广券、领券链接外，均支持无门槛券；门槛默认等于面额 + 0.01 元 | 无门槛券更直接，但资损风险更高。 |
| 消费者领券限制 | 消费者在单店铺可领取优惠券张数为 15 张 | 过多券模板可能导致消费者无法继续领取。 |

## 渠道与玩法

| 玩法 | 适用场景 | 是否自动透出 |
|---|---|---|
| 全网自动推广 | 无特殊定向需求，希望在公域透出 | 是 |
| 自有渠道推广 | 私域、客服、社群、短信等领券链接或二维码 | 否 |
| 定向渠道推广 | 直播间等特定渠道 | 视配置 |
| 定向人群运营 | 店铺会员、特定人群 | 视配置 |
| 定向场景运营 | 客服追单等场景 | 视配置 |

## 核销规则

优惠券核销已从“一笔订单购买多件商品仅可核销一张优惠券”升级为“购买多件商品时，每件商品最多可核销一张优惠券”。实际计算以子订单为核心。

| 核销点 | 规则 | 经营含义 |
|---|---|---|
| 多件商品 | 订单购买多件商品/SKU 时，平台按子订单总金额和件数判断是否达到门槛 | 多件成交可核销多张券，但要控制限领和资损。 |
| 多券圈选同一商品 | 同一商品被不同优惠券圈选时，按子订单维度择优核销 | 避免为同一商品配置多个相互抢生效的券。 |
| 单件限制 | 一件商品/SKU 至多使用 1 张优惠券 | 不能把一件商品金额拆开多次凑券。 |
| 子订单合并 | 先核销已达门槛的子订单，剩余未达门槛子订单再合并判断 | 复杂订单下，优惠结果不等同于简单按主订单金额凑券。 |
| 物流拆分 | 拆分物流场景按拆分后的子订单金额和件数核销 | 多地址或帮亲友买可能改变核销结果。 |

## 资损与风控

| 风险 | 平台规则 | 商家动作 |
|---|---|---|
| 总让利过高 | 总让利金额 = 发放量 x 优惠券面额；非无门槛券上限 10w，无门槛券上限 50w，超出为强卡控 | 先用发放量和面额算总预算，再提交资损校验。 |
| 优惠叠加风险 | 若扣除商品叠加优惠后的预估最低到手价 < 红线价，会出现低价提醒；默认红线价 = 商品标价 x 0.5 | 低价提醒是弱卡控，可忽略但需要主动评估毛利和价保风险。 |
| 价保风险 | 核销规则升级前的订单若仍在价保期，规则升级后可能重新计算价差 | 规则变更前应调整券金额或限领，避免后续补差。 |
| 商品图误导 | 优惠规则变更后，商品主图中的优惠信息可能和实际生效不一致 | 规则切换前检查主图、详情页和活动文案。 |

## 可叠加 / 互斥关系

| 相关工具 | 关系 | 说明 |
|---|---|---|
| 跨店满减、品类券、消费券 | 可叠加 | 优惠券属于店铺优惠，可与跨店级优惠叠加。 |
| 赠品 | 可叠加 | 赠品不属于同类满减满折工具。 |
| 单品级活动，如单品宝、单品立减 | 可叠加 | 适合先做单品价，再叠加一种店铺级工具。 |
| 淘金币、红包 | 可叠加 | 属于其他消费者资产或平台权益。 |
| 多件优惠、店铺宝、店铺级优惠券 | 同一子订单互斥择优或不可叠加 | 2026-03-10 后，优惠券与多件优惠等店铺满减满折类优惠在同一子订单仅生效优惠力度较大的一个。 |

## 适合使用的场景

- 需要做会员券、直播间券、客服追单券、私域领券链接等定向运营。
- 商品不适合用多件优惠拉客单，但适合用清晰面额降低决策门槛。
- 希望通过一个店铺级工具稳定到手价，而不是同时叠加多件优惠。

## 不适合使用的场景

- 已经用多件优惠承接主要让利，且同一子订单下优惠券无法成为择优工具。
- SKU 需要非常精细的小额阶梯调价，此时应优先考虑单品立减。
- 毛利低、价保期长、叠加平台活动多时，未做红线价和资损校验不应直接发券。

## 操作入口

- 千牛后台 -> 营销 -> 营销工具 -> 优惠券。
- 规则冲突检查：商品价格管理，查询 2026-03-10 起可能发生互斥的优惠券和多件优惠活动。
