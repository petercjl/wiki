# 天猫超市与京东自营合作模式对比

天猫超市和京东自营都不是普通 POP 店铺逻辑。二者都把仓配、售后、部分销售资源上移到平台，品牌侧以供货、资质、库存、财务、运营协作为主。

## 对比矩阵

| 维度 | 天猫超市 | 京东自营 |
| --- | --- | --- |
| 合作本质 | 平台超市供应商模式，商家与猫超小二协作经营 | 京东以零售商身份采购，品牌/供应商进入京东自营供货体系 |
| 货权/销售控制 | 资料强调平台仓配售后，货权细节需看合同 | 资料明确京东买断货权，京东控制销售链路 |
| 品类重点 | 食品生鲜、日用清洁、个护美妆、家居用品等高频家庭消费 | 更广泛标准化零售品类，尤其适合强调正品、履约、服务的商品 |
| 流量来源 | 淘宝/天猫主站入口、88VIP、天猫超市频道 | 京东自营标识、搜索推荐、大促主会场、采销资源 |
| 履约 | 入中心仓，平台负责仓储、打包、发货和售后 | 京东仓储、物流、客服、售后统一处理 |
| 商家重点 | 供货、库存、账单、发票、运营配合、活动资源沟通 | 供货、质检、合同、发票、毛保、采销沟通、商品资料 |
| 费用口径 | PPT 口径为 20%-25% 毛保/费用、5 万保证金 | PDF 口径为 3 万-15 万保证金、20%-35% 毛保、无平台使用费 |
| 主要风险 | 毛利被费用和促销压缩，库存/补货/退货协作压力 | 毛保、价格管控、库存风险、服务商承诺不确定 |
| 更适合 | 高频刚需、补货型、家庭消费、供应稳定 SKU | 品牌需要强正品背书、快速履约、高信任转化的标准品 |

## 决策逻辑

优先考虑天猫超市的情形：

- 商品是家庭高频、补货型、快消或生活必需品。
- 品牌已经在淘宝/天猫体系有销售基础或流量承接能力。
- 需要阿里生态入口、88VIP 和天猫超市履约能力。
- 商家能承接商超库存、补货、退货和账单协作。

优先考虑京东自营的情形：

- 商品需要强正品信任、快递履约、售后服务和中高客群转化。
- 品牌愿意接受京东采销和平台链路强控制。
- 毛利足以覆盖毛保与大促分摊。
- 资质、授权、质检、发票和供应链稳定性强。

两者都不应进入的情形：

- 没有算清毛利、扣点、促销、损耗和资金周转。
- 仅凭服务商承诺做决策，缺少官方规则和合同条款复核。
- 产品供应不稳定或缺少合规资质。
- 期望通过平台自营替代自身产品力、供货能力和基础运营。

## Agent 输出建议

给品牌做平台入驻建议时，输出应包含：

- 适配结论：适合、暂缓、只适合某平台、先补资质/供应链。
- 核心理由：品类、毛利、供应、品牌阶段、客群、履约需求。
- 成本测算：保证金、毛保/扣点、促销分摊、仓配损耗、回款周期。
- 风险清单：费用时效、服务商承诺、库存压力、平台政策变化。
- 下一步资料：营业执照、一般纳税人、商标/授权、质检、销售数据、SKU 表、毛利表。
