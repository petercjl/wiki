# 电商企业财税股权与战略财务管理

本 learning path 基于“财税股权 28 期”4 份课程转写重构，不按 D1/D2 课堂顺序归档，而按电商企业可复用的知识依赖组织：**风险基因 → 四本账 → 税务合规 → 税负模型 → 主体架构 → 业财核算 → 股权控制 → 激励治理 → 股权资源整合 → 战略财务**。

> 风险边界：本知识库用于经营分析、方案设计和 Agent 诊断，不替代税务申报、法律意见或股权文件。涉及税率、政策阈值、公司架构、代持、分红、社保用工等执行动作，必须结合最新法规和专业财税/法律意见复核。

## 核心判断

电商企业的财税问题通常不是单点税务问题，而是增长方式、组织方式、数据方式、资金方式共同造成的系统问题。只补票、只补税、只调整公司主体，不能解决经营端利润失真、资金混同、股权失控和激励错配。
