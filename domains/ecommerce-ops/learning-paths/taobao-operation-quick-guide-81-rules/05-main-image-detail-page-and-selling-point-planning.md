# 主图、详情页与卖点策划

## 问题定义

主图解决“愿不愿意点”，详情页解决“为什么买”。书中把主图、详情页、卖点策划和美工绩效放在同一条转化链里，这一点对今天依然成立。

## 主图方法

主图不是审美作品，而是搜索和推荐入口的点击决策界面。判断主图优劣，应以同词、同渠道、同竞品环境下的点击率和后续转化为反馈，而不是老板个人审美。

## 详情页方法

详情页要回答用户真实购买疑虑：这个产品适合谁，解决什么问题，和竞品有什么不同，为什么现在买，售后风险如何。详情页不是信息堆叠，而是购买决策路径。

## 卖点策划

卖点必须从产品事实转译为消费者可感知价值。可以借用 [[domains/visual-production/concepts/marketing-theory-for-ecommerce-images|FABE、USP、AIDA 等营销框架]]：先讲功能事实，再讲优势，再讲利益，最后给证据。

## 美工绩效

书中的关键组织观点是：美工的正反馈应来自点击率、转化率和测试结果，而不是迎合老板审美。视觉岗位要通过猜想、执行、验证形成刻意练习。
