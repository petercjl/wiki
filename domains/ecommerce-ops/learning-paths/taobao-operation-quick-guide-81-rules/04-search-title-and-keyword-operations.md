# 搜索、标题与关键词运营

## 问题定义

书中关于标题、搜索和关键词的具体规则具有明显平台时间边界，但背后的方法仍然稳定：搜索运营不是优化一个宝贝的平均数据，而是围绕关键词、人群意图和竞品相对表现建立优势。

## 核心原则

1. 标题是搜索入口的需求映射，不是堆词。
2. 搜索权重应按词和人群理解，而不是只看全宝贝平均转化。
3. 数据价值来自对比：和竞品、同词、同渠道、同时间窗口比较。
4. 单品异常诊断要区分流量入口、点击、转化、评价、价格、库存和竞品动作。

## 标题优化流程

1. 提取核心品类词、属性词、场景词、人群词和问题词。
2. 判断每个词背后的购买意图和竞争强度。
3. 用低竞争但高匹配的词启动，而不是一开始抢最强词。
4. 观察词级点击、收藏、加购、转化和成交。
5. 对表现差的词做替换，对表现好的词集中资源强化。

## 当前复核点

2018 年书中对淘宝搜索和生意参谋的描述不能直接当作当前规则。执行前应复核当前搜索词工具、流量来源口径、搜索排序因素、标题长度限制和平台违规规则。
