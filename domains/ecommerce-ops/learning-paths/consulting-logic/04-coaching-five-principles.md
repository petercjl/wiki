# 带教五大原则

## 总目标

带教不是无限回答问题，而是把企业从“有路径”带到“有未来”。脑图将带教过程总结为五条原则。

## 第一原则：同类问题只回答一遍

- 同类问题只回答一遍，团队要形成方法
  - 没标准化流程化的带教最终都是无用功
    - 因为你不可能一直有时间帮着解决具体问题
      - 没有标准化流程的团队也无法放大

带教的目标是形成方法、流程和标准。没有标准化流程的带教会变成无休止答疑，团队也无法放大。

## 第二原则：要对学员有所要求

- 要对学员有所要求，敢提要求
  - 有明确要求是不被学员带着走的关键
    - 敢提要求是赢得学员内心尊重的核心关键
      - 你没有要求学员就会有无限的要求

明确要求是避免被学员带着走的关键。敢提要求，反而能赢得学员尊重。

## 第三原则：不打无把握之战役，首战即决战

- 不打无把握之战役，首战即决战
  - 不摸清资源优势能力模型不轻易给建议
    - 第一次的失误指导会产生深深的信任危机
      - 上中下难简易给全面给其利弊讲清

第一次指导失误会产生信任危机。因此在摸清资源、优势和能力模型之前，不应轻易给建议。

## 第四原则：勤于学习，超纲学习，不断学习

- 勤于学习，超纲学习，不断学习
  - 先有大量的案例才有参考的方案
    - 有了大量的案例才能分出策略等级高低
      - 有了高低等级才能有终极引领

咨询师必须积累大量案例，才能区分策略等级，进而形成更高层次的引领。

## 第五原则：按规模等级看路径未来

- 规模等级看路径未来
  - 两倍规模看执行，三倍规模看布局
    - 四倍规模看组织，五倍规模看效率
      - 十倍等级看资源，往上等级看心念

| 规模目标 | 重点 |
| --- | --- |
| 2 倍规模 | 执行 |
| 3 倍规模 | 布局 |
| 4 倍规模 | 组织 |
| 5 倍规模 | 效率 |
| 10 倍等级 | 资源 |
| 更高等级 | 心念 |

## 应用提示

- 每次答疑都要沉淀成流程或标准。
- 带教要敢于提出要求，不只满足客户要求。
- 首次建议必须谨慎，先诊断后出手。
- 规模目标不同，辅导抓手不同。
