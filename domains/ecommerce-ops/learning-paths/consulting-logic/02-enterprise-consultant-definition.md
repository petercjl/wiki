# 企业咨询师的定义与底层任务

## 定义

脑图给出的企业咨询师定义是：

> 通过与企业主多次询问沟通，并查考、谋议，形成一套认知共识、方法可循、引领未来、达成企业主事业企图的动态方案。

这一定义强调三件事：不是替客户做决定，而是通过询问建立共识；不是只解决眼前动作，而是给出可循的方法；不是只看当下问题，而是引向未来。

## 字义拆解

- 咨
  - 跟别人商议，询问
- 询
  - 问，征求意见
- 师
  - 传道授业解惑也
- 企业
  - 企图
    - 企划
      - 确定性
      - 规划性
  - 事业
    - 事情
      - 认清环境资源
      - 人情能力意愿
- 企业咨询师
  - 通过与企业主多次询问沟通，并查考，谋议，形成一套认知共识，方法可循，引领未来，达成企业主事业企图的动态方案
    - 从非共识到共识
      - 先攻其心
    - 从障碍到有路径
      - 再铺其路
    - 从看当下到有未来
      - 终燃其志

## 咨询师的三段任务

| 阶段 | 任务 | 脑图原句 |
| --- | --- | --- |
| 共识 | 从非共识到共识 | 先攻其心 |
| 路径 | 从障碍到有路径 | 再铺其路 |
| 未来 | 从看当下到有未来 | 终燃其志 |

## 对 Agent 的使用价值

当 Agent 被要求生成咨询方案时，不应直接进入“给建议”。更稳妥的顺序是：

1. 先识别客户当前的非共识点。
2. 再拆障碍，找到路径。
3. 最后把路径连接到未来目标和老板意愿。
