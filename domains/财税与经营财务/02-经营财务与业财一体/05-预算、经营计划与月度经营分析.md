# 预算、经营计划与月度经营分析

预算不是财务填表，而是战略落地、资源分配和绩效考核的连接器。没有预算，增长目标、费用上限、奖金包和现金安全线都只是口号。

## 经营计划要回答凭什么增长

从 5000 万到 1 亿、从 8000 万到翻倍，必须说明凭什么：新增平台、提高客单价、拓 SKU、提升转化率、降低退货、提高复购、涨价、降推广、优化供应链，还是进入新类目。财务把经营计划转成收入、毛利、费用、库存、现金流和利润。

## 月度经营分析会

月度经营分析至少看：

- 销售额、毛利率、净利率
- 推广费率、快递费率、平台扣点
- 平台/店铺/链接边际贡献
- 库存周转天数和滞销品
- 现金流与净利润比
- 人均能效和管理费用率
- 预算与实际差异
- 下月经营动作和责任人

课程中提醒，报表一个半月才出，老板看到时“黄花菜都凉了”。电商经营分析必须及时，最好月度固定节奏，重点业务可周度。

## 财务不是只指出问题

优秀财务给老板汇报时要带方案：A 方案、B 方案、C 方案，各自优缺点和风险。只说“又出问题了”会制造焦虑。财务要把问题转成选择题，让老板做经营决策。

## Agent 使用提示

为用户设计经营分析模板时，不要只给财务指标。要把“指标 -> 异常解释 -> 业务动作 -> 责任人 -> 下月预算”连起来，否则会议会变成报数而不是经营复盘。
