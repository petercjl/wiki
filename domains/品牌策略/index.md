# 品牌策略知识域

本知识域沉淀品牌定位、品类心智、人群认同、产品战略、大单品、差异化感知、品牌视觉资产和品牌表达方法。这里的“品牌视觉”关注的是视觉如何成为品牌资产，而不是具体图片如何制作。

## 阅读顺序

1. [[domains/品牌策略/01-品牌基础与增长方法/index|品牌基础与增长方法]]：品牌为什么成立、从白牌到溢价、品牌能力如何积累。
2. [[domains/品牌策略/02-品类心智与差异化/index|品类心智与差异化]]：品类聚焦、人群认同、差异化感知和细分品类防守。
3. [[domains/品牌策略/03-产品战略与大单品/index|产品战略与大单品]]：心智产品力、大单品、产品矩阵、拓品和场景营销。
4. [[domains/品牌策略/04-品牌视觉资产/index|品牌视觉资产]]：品牌视觉记忆、符号、ABC 分层、可视化世界观和视觉制作边界。
5. [[domains/品牌策略/90-样本/index|品牌案例库]]：课程案例、策略样本和按问题调用的案例索引。
