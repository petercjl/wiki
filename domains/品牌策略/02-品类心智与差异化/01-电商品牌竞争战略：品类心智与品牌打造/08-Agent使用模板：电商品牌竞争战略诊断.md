# Agent 使用模板：电商品牌竞争战略诊断

## 用途

本模板用于让 Agent 基于品牌资料、店铺页面、竞品信息和经营目标，诊断企业应优先解决定位、视觉、大单品还是营销传播问题。

## 输入资料

- 品牌名称、品类、价格带、核心 SKU。
- 近 12 个月销售结构、渠道结构、增长目标。
- 主要竞品及其定位表达。
- 店铺首页、主图、详情页、短视频、直播间截图。
- 已有品牌故事、创始人故事、检测认证、用户评价。

## 分析提示词

```text
请基于以下资料，按“品类路线—心智定位—视觉美学—大单品—营销传播—优先级”的顺序，诊断这个电商品牌的增长约束。

要求：
1. 判断它更适合品类聚焦、人群认同，还是先品类聚焦再人群延展。
2. 用三原则验证当前心智定位：自身优势、用户需求、竞品区隔。
3. 从产品、人群、场景、情感理念、风格五维度提出候选定位。
4. 评估视觉是否“看起来像品牌”，并指出电商、产品、空间、品牌四类美学断点。
5. 判断是否存在能代表品牌的大单品；若没有，提出候选大单品选择标准。
6. 检查传播是否仍停留在卖产品/卖功能，提出升级为情绪、调性、理念和背书的路径。
7. 对定位、视觉、大单品、营销四模块分别 0-10 分评分，并给出未来 90 天优先级。

输出格式：
- 一句话诊断
- 当前最大瓶颈
- 四模块评分表
- 推荐路线
- 候选定位
- 大单品建议
- 视觉升级建议
- 传播主线建议
- 90 天行动清单
```

## 输出判断标准

- 不只给口号，必须解释口号为什么成立。
- 不只给视觉建议，必须说明视觉如何支撑定位和价格带。
- 不只找爆款，必须判断产品是否能成为品牌心智入口。
- 不只列营销动作，必须说明每个动作沉淀什么品牌资产。
