# Agent 使用模板：细分品类品牌升级诊断

## 用途

用于让 Agent 诊断一个细分品类品牌是否已经具备从单点优势升级为系统品牌的条件，并输出 90 天行动建议。

## 输入资料

- 品牌名称、品类、成立时间、销售规模、价格带。
- 核心产品、SKU 结构、渠道结构。
- 主要竞品：大品牌横向产品线、低价工厂、同类细分品牌。
- 主图、详情页、短视频、直播间、包装、私域截图。
- 专利、认证、机构合作、用户评价、复购数据。
- 团队当前管理工具：日报、考核、AI 工具、私域运营机制。

## 分析提示词

```text
请基于以下资料，诊断这个细分品类品牌的升级路径。

要求：
1. 判断它当前是否面临“大品牌横向进入”和“低价工厂冲击”的上下夹击。
2. 按六维模型评分：视觉、技术、渠道、背书、心智、产品，每项 0-10 分。
3. 判断品牌应继续聚焦品类、拓品线、拓渠道，还是需要谨慎拓品类。
4. 识别一个最适合承接品牌心智的大单品或入口产品。
5. 检查视觉是否支撑目标人群和价格带。
6. 设计 3 个低成本执行力动作，要求简单、可操作、有效果。
7. 判断是否适合做私域共创，并设计获客、群规则、投票和新品反馈流程。
8. 给出 90 天行动清单，按优先级排序。

输出格式：
- 一句话诊断
- 上下夹击风险
- 六维评分表
- 当前最大短板
- 品类/品线/渠道选择建议
- 大单品建议
- 组织执行力动作
- 私域共创方案
- 90 天行动清单
```

## 判断原则

- 不把拓品类当默认增长方式。
- 先判断是否能在原品类中破品线、拓渠道、提心智。
- 执行动作必须低成本、当天可启动、能被基层理解。
- 私域不是广告群，而是产品共创和品牌日常关系系统。
