# 10｜Agent 使用模板：品牌视觉记忆与可视化体系诊断

## 适用任务

当 Agent 需要为电商品牌生成品牌策划、视觉规划、主图方向、详情页方向、AI 生图方向或品牌升级建议时，应先判断品牌是否已经具备可继承的视觉记忆资产，再判断是否有稳定的可视化世界观。

这个模板尤其适合以下输入：

1. 店铺商品 Excel、主图、详情页、包装图和竞品图。
2. 品牌已有 logo、包装、VI、主色、IP、店铺首页和达人内容。
3. 用户对品牌升级、视觉统一、AI 生图、详情页优化的需求。

## 输入清单

最小输入：

1. 品牌名称、品类、价格带和主要销售渠道。
2. 店铺商品数据或核心 SKU 列表。
3. 现有主图、详情页、包装或店铺视觉截图。
4. 3-5 个竞品或对标品牌。

增强输入：

1. 用户评论、客服反馈、搜索词、点击率、转化率、跳出率。
2. 老用户问卷或访谈。
3. 线下物料、直播间、达人内容、投放素材。
4. 品牌历史、创始人故事、产品研发证据。

## 诊断流程

第一步，识别记忆资产：

1. 列出所有可能被用户记住的视觉资产，包括颜色、图形、包装结构、人物、场景、符号、口号和功能图形。
2. 判断资产属于高记忆、可优化、低价值还是负资产。
3. 对高记忆资产提出保留和强化方式。

第二步，判断三要素：

1. 传承：是否保留用户已有记忆。
2. 特色：是否有不同于竞品的识别点。
3. 效率：是否能降低用户理解和购买成本。

第三步，建立 ABC 分层：

1. A 层：品牌态度、信任、层级、价格感。
2. B 层：使用场景、人物、生活状态、向往感。
3. C 层：功能卖点、证据、参数、购买行动。

第四步，搭建可视化世界观：

1. 空间：画面发生在哪里，空间带来什么联想。
2. 人物：谁代表目标用户，人物如何行动。
3. 元素：哪些道具、符号、材料和功能证据必须出现。
4. 版式：信息顺序、画面密度、字体、色块和阅读路径。

第五步，输出执行规则：

1. 固定项：必须长期保留的颜色、符号、人物气质、空间质感和版式规则。
2. 变化项：可以按新品、节日、渠道和内容主题变化的元素。
3. 禁用项：会误导品类、拉低价格感、稀释资产或造成合规风险的画面。

## 评分表

| 维度 | 高分表现 | 低分表现 |
| --- | --- | --- |
| 记忆资产 | 用户能快速识别品牌，旧资产被有效继承 | 每次升级都换风格，旧用户认不出 |
| 传承 | 保留强资产并做现代化表达 | 只追求新鲜感，推翻既有识别 |
| 特色 | 与竞品有明确视觉区隔 | 画面像行业通用模板 |
| 效率 | 用户一眼理解产品差异和购买理由 | 信息堆叠，功能和价格带关系不清 |
| ABC 分层 | 品牌、场景、功能各司其职 | 只有功能清单或只有氛围图 |
| 世界观 | 空间、人物、元素、版式可复用 | 每张图风格漂移 |
| AI 可执行性 | prompt 可稳定生成同一品牌气质 | AI 输出像不同品牌拼图 |

## 输出结构

Agent 生成品牌视觉规划时，建议至少包含：

1. 现有视觉资产盘点。
2. 可继承资产与需剔除资产。
3. 品牌视觉三要素判断：传承、特色、效率。
4. ABC 视觉分层建议。
5. 可视化世界观：空间、人物、元素、版式。
6. 主图、详情页、包装、内容和投放素材的执行规则。
7. AI 生图参考方向和 prompt 约束。
8. 风险与验证方式。

## Agent 提示词

```text
你是品牌视觉策略顾问。请基于输入的店铺数据、现有视觉材料和竞品材料，为该品牌建立一套品牌视觉记忆与可视化体系。

请按以下顺序输出：
1. 识别品牌已有视觉记忆资产，并判断哪些必须传承、哪些可以优化、哪些应该放弃。
2. 用“传承、特色、效率”三个维度评估现有视觉体系。
3. 用 A/B/C 分层拆解品牌当前主图、详情页和内容素材的信息结构。
4. 建立可视化世界观，包括空间、人物、元素、版式四个模块。
5. 给出主图、详情页、包装、短视频、AI 生图的执行规则。
6. 给出需要验证的数据指标，包括点击率、转化率、跳出率、老用户识别率、评论反馈和问卷结果。

要求：只输出可执行的品牌视觉策略，不输出分析过程说明；所有结论都要说明来自数据、现有资产、竞品对比还是策略推断。
```
