# 精益画布与三类压力测试

## 问题定义

创业点子在脑中通常是完整、漂亮、充满潜力的；但在验证时必须被拆成可测试的商业模式假设。精益画布的作用，是把“一个想法”拆成多个可以被客户行为和数据检验的部件。

来源锚点：`raw/books/jingyi-chuangye-shizhan-3rd-2023/chapters/ch026-1.md` 至 `ch030-5.md`。

## 核心模型

### 精益画布九栏

| 栏位 | 要回答的问题 | 典型风险 |
| --- | --- | --- |
| 客户群体 | 谁最早遇到这个问题并愿意尝试？ | 范围过宽，谁都像客户，谁都不行动。 |
| 问题 | 客户想完成什么任务，现有方案哪里不好？ | 把自己的功能想象成客户问题。 |
| 独特价值主张 | 为什么客户现在要关注你？ | 承诺泛泛，无法与现有方案形成对比。 |
| 解决方案 | 最小解决方案是什么？ | 过早做完整平台或复杂产品。 |
| 渠道 | 如何触达和获取早期客户？ | 渠道与客户群体不匹配。 |
| 收入来源 | 客户如何付出可变现价值？ | 只有流量，没有变现路径。 |
| 成本结构 | 交付、获客、运营和研发成本是什么？ | 低估现金流和交付压力。 |
| 关键指标 | 哪些行为说明客户工厂前进？ | 使用虚荣指标。 |
| 不公平优势 | 什么东西难以被复制？ | 把普通功能、热情或先发想象成护城河。 |

### 创新三位一体

任何点子都要通过三类压力测试：

- 需求性：客户需要吗？问题是否值得解决？
- 发展性：能否变现并形成足够大的客户工厂？
- 可行性：能否在当前资源和阶段创建并交付？

这三个问题的顺序很重要。不要因为技术可行，就跳过需求性和发展性。

## 操作流程

### 1. 一想法多画布

一个宏大想法通常包含多个客户群体、多个问题和多个解决方案。不要把它们挤进一张画布。每个客户-问题-解决方案组合单独一张，方便比较风险。

### 2. 先收窄早期采用者

早期采用者必须具备三点：

- 问题已经发生或即将发生；
- 正在用现有方案凑合解决；
- 有较强转换动机。

如果画布上的客户是“所有中小商家”“所有年轻女性”“所有用户”，就还没有进入验证状态。

### 3. 做需求性压力测试

需求性不靠问“你喜不喜欢我的产品”，而靠还原客户任务：

- 客户现在想完成什么？
- 现有方案是什么？
- 哪里出了问题？
- 是什么触发客户考虑新方案？
- 客户不换的惯性力和焦虑是什么？

详细访谈方法见 [[domains/品牌策略/03-产品战略与大单品/04-精益创业实战验证系统/03-客户受力分析与问题访谈|客户受力分析与问题访谈]]。

### 4. 做发展性压力测试

发展性测试从最低成功标准倒推客户工厂：

1. 定义 3 年或阶段性最低成功标准，例如收入、利润、活跃客户或订单。
2. 倒推出所需活跃客户数。
3. 估算获客、激活、留存、变现、推荐各环节的合理转化。
4. 检查定价、客单价、复购、留存和获客成本是否支持目标。
5. 如果不支持，调整目标、客户群体、价格、渠道或商业模式。

定价不要只按成本加成。客户会拿你和现有解决方案的时间、金钱和精力成本比较；如果你的价值确实更强，定价可以有溢价，但必须能被客户理解和接受。

### 5. 做可行性压力测试

可行性不是问“最终能不能做出来”，而是把路线拆成三个阶段：

| 阶段 | 目标 | 当前应避免 |
| --- | --- | --- |
| 问题与解决方案匹配 | 找到值得解决的问题和愿意行动的客户 | 过早做全功能产品 |
| 产品与市场契合 | 让客户满意循环稳定运转 | 过早大规模获客 |
| 扩张 | 找到可持续增长火箭 | 同时追很多增长技巧 |

### 6. 清晰阐述创业点子

对客户的推介应按客户世界观组织，不按内部功能表组织：

- 为什么是现在；
- 现在有什么利害攸关的问题；
- 原有方式哪里需要被打破；
- 新方案如何更好；
- 为什么你能做；
- 客户下一步要采取什么行动。

## 判断标准

画布可以进入验证阶段的最低质量：

- 每栏都有明确假设；
- 早期采用者不是泛人群；
- 问题来自客户任务和现有方案，而不是产品功能；
- 发展性有粗略数字，不只是“大市场”；
- 当前阶段成功标准清楚；
- 关键指标是行为指标，而不是声量指标。

## 案例与证据锚点

书中用星巴克、Facebook、爱彼迎等画布示例说明，同一个平台经常有多边角色：用户、客户、买家、卖家不能混写。对电商品牌而言，消费者、渠道商、达人、平台和供应链也可能分别是不同角色，验证时要拆开。

## 适用边界

精益画布是工作底稿，不是商业计划书。它适合不断修改；如果一张画布长期不变，通常说明团队没有把验证结果回写进去。

## Agent 使用提示

Agent 诊断新品时，先输出一张精益画布，再列出三类压力测试中风险最高的 3 个假设。不要在画布未收窄前直接生成品牌定位、详情页结构或投放方案。
