# 11｜Agent 使用模板：心智产品力与大单品战略诊断

## 适用任务

当 Agent 需要根据店铺数据、商品列表、价格带、销量结构和竞品情况判断品牌的大单品机会时，应使用本模板。

它适合用于品牌规划报告中的“产品力诊断”“大单品机会”“产品线重构”“新品企划方向”和“渠道起盘策略”。

## 输入清单

最小输入：

1. 店铺商品 Excel 或 SKU 销售数据。
2. 商品标题、价格、销量、销售额、主图、类目和规格。
3. 品牌当前渠道结构。
4. 主要竞品和对标品牌。

增强输入：

1. 搜索词、用户评论、客服反馈、复购数据。
2. 达人内容、投放素材、小红书笔记和直播话术。
3. 供应链成本、毛利、库存、交付周期。
4. 商标、系列名、包装和视觉资产。

## 诊断流程

第一步，判断经营阶段：

1. 货品型电商：SKU 多但用户心智弱。
2. 动销型店铺：能卖货但产品资产难沉淀。
3. 用户经营型品牌：产品围绕人群和心智持续迭代。

第二步，评估心智产品力：

1. 产品是否有明确人群。
2. 产品是否有明确场景。
3. 产品是否有可传播名称或系列名。
4. 产品功能是否能翻译成用户认知。
5. 产品是否有长期迭代空间。

第三步，识别大单品层级：

1. 爆品：销量高但品牌心智弱。
2. 大单品：产品和品牌能绑定。
3. 战略大单品：可长期经营并带动品牌资产。

第四步，检查后手：

1. 商标和关键词。
2. 供应链和产能。
3. 内容主题和视觉系统。
4. 系列化和新品迭代。
5. 渠道分工和达人机制。

第五步，提出策略：

1. 当前爆品是否值得升级为大单品。
2. 哪些 SKU 适合合并为英雄系列。
3. 哪个新品方向能承担年轻化、防守反击或第二增长曲线。
4. 是否适合高定价、高佣金、KOC 起盘模型。

## 输出结构

Agent 输出报告时建议包含：

1. 店铺当前经营阶段判断。
2. Top SKU 的销量爆品 vs 心智大单品分级。
3. 心智产品力评分表。
4. 候选大单品与英雄系列建议。
5. 大单品后手清单。
6. 新品企划机会。
7. 渠道起盘建议。
8. 风险与验证指标。

## Agent 提示词

```text
你是电商品牌产品战略顾问。请基于输入的店铺商品数据、价格带、销售结构、主图和竞品信息，诊断该品牌的心智产品力与大单品机会。

请按以下结构输出：
1. 判断店铺处于货品型、动销型还是用户经营型阶段。
2. 区分销量爆品、潜力大单品和战略大单品。
3. 对核心 SKU 评估心智产品力：人群、场景、利益、名称、符号、持续性。
4. 找出可以系列化的商品组，并提出英雄系列命名方向。
5. 为最有潜力的大单品设计后手：商标、关键词、供应链、内容、视觉、新品迭代。
6. 判断是否适合高定价、高佣金、KOC 起盘。
7. 给出 3 个需要用数据验证的假设。

要求：不要把销量最高的商品直接等同于大单品；每个建议都要说明依据来自销售数据、用户认知、竞品对比还是策略推断。
```
