# 品牌大单品打造与产品企划营销

这条路径用于把品牌大单品从战略判断推进到具体企划：如何选择产品战入口，如何设计差异化，如何把卖点翻译成买点，如何用可视化、昵称化和内容测试把核心心智打穿。

## 适用问题

1. 品牌想围绕 Top SKU 打造大单品，但缺少企划方法。
2. 商品同质化严重，需要找到可感知差异。
3. 卖点很多但用户记不住，需要聚焦核心心智。
4. 需要为主图、详情页、直播、短视频和小红书写大单品表达方案。
5. 老品牌需要用新品做防守反击或第二增长曲线。
