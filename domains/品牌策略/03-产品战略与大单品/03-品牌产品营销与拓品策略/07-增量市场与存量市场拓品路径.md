# 07｜增量市场 vs 存量市场的拓品路径

## 摘要

增量市场和存量市场的拓品逻辑完全相反。增量市场：先聚焦单品打穿一个细分心智，再横向延伸同场景/人群品类，最终形成品牌生态（以点穿面）。存量市场：拉开战线，多品类渗透同一批人，做强排他性，最终成为生活方式品牌（多点布局）。用错路径，要么把自己做窄，要么把自己做没。

## 核心问题

- 怎么判断自己所在的是增量还是存量市场？
- 两种市场的拓品顺序为什么相反？
- 海龟爸爸和欧莱克各自用的是哪条路径？

## 方法框架：两种路径对比

| | 增量市场 | 存量市场 |
| --- | --- | --- |
| 市场比喻 | 水位上涨的鱼塘，随便圈一块都能活 | 即将干涸的鱼塘，必须成为最大那条鱼 |
| 拓品顺序 | 单品突围 → 同类心智品类覆盖 → 横向拓展（人群/场景） → 品牌生态 |多品类并行布局 → 跨品类渗透同一人群 → 生活方式品牌 |
| 核心逻辑 | 以点穿面，聚焦细分，贪多嚼不烂 | 拉开战线，多点渗透，强排他性 |
| 风险 | 过早分散，稀释初期心智，变成大而全 | 窝在单一品类，被头部压死，原地踏步 |

## 推导逻辑：两个案例

### 海龟爸爸（增量市场·儿童防晒）

儿童护肤是增量市场（市场规模快速扩大）。

- 2021年：聚焦"物理防晒"，四个月做出100万支大单品（小雪山）
- 不做洗护，继续围绕防晒矩阵：丝光、喷雾、日间防护……防晒品类全覆盖
- 2024年：防晒市场做大之后，把"防晒"升级成"抗光损"，切护肤四大赛道里最大的修复赛道
- 同期布局祛痘——发现防晒人群的下一个生命周期需求（青春期痘痘焦虑/社交焦虑）

路径：单点（物理防晒）→ 细分矩阵（防晒全品类）→ 大赛道延伸（抗光损/修复）→ 人群生命周期延续（祛痘）

最终实现：从品带牌（用防晒的好口碑带动整个品牌）到牌带品（品牌力反过来推新品）。

### 欧莱克（存量市场·小家电）

小家电慈溪系是典型存量市场，美的等大牌掌握极致供应链成本 + 品牌力，随时可以抄款压价。

- 起点：陶瓷小奶锅（发现妈妈热奶容易糊、锅会翻倒，做插电小白锅，卖爆）
- 美的立刻抄——欧莱克不焦虑，直接开第二款：日式电饭煲，再次打爆
- 策略：美的学多了就不是美的，大而全是美的的定位，欧莱克只做速简美学生活电器
- 完整品类链（沿速简美学路径扩张）：
  - 主餐场景：小奶锅→电饭煲→高压锅→料理锅→加热菜板→电煮锅→电烤箱→电蒸锅→电磁炉
  - 配餐/早餐场景：蒸蛋器→豆浆机→华夫饼机
  - 下午茶场景：榨汁机→咖啡机→料理壶
  - 生活电器：风扇→加湿器→除螨仪→吹风机
  - 锅碗瓢盆 → 卫浴（花洒、水龙头）
- 原点人群：职场白领，租简装房，速简美学与简装家居天然匹配
- 多品类不是乱拓，是始终围绕同一批人（白领租房族）的全生活场景渗透

## 核心原则

- **增量市场**：科细分够了，贪多嚼不烂。先打穿单品，逐步过渡大品类，形成品类心智。
- **存量市场**：不拓就等于原地踏步。要用多品类方式加大心智渗透，最终强排他性才能活。
- 误用路径的后果：增量市场搞大而全 → 把自己做宽、做散；存量市场死守单品 → 把自己做没。

## 判断标准

判断增量 or 存量的快速方法：
- 这个品类的消费人群是在增加（年轻化趋势、新场景涌现）还是在萎缩（出生率下降、替代品挤压）？
- 头部品牌的份额是还在扩大、尚未集中，还是已经高度集中、市场份额固化？

儿童护肤2021年时是增量（人群教育成本在下降，市场快速扩大）；儿童服饰2024年已是存量（出生率持续下降）。

## 常见误区

- 存量市场里还用增量逻辑"聚焦单品等市场来"，等不来的。
- 增量市场里急着铺大而全，初期心智还没建立就分散。
- 欧莱克的可怕之处不是品类多，是每个品类都带"速简美学"标签，大牌学了就不是大牌。不理解这个护城河的商家，只看到"品类多"就去模仿，会直接死。

## 可用于 Agent 的提示词

> 给定一个品牌（主营品类、当前市场阶段：增量/存量、原点人群），请生成一份拓品路线图：按增量（以点穿面）或存量（多点渗透）路径，输出未来3步的拓品方向，并说明每一步的逻辑依据（人群延续 / 场景覆盖 / 心智延伸）。
