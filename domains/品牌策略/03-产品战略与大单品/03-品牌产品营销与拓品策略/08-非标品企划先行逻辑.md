# 08｜非标品的企划先行逻辑

## 摘要

标品可以先打单品、看数据再调方向。非标品（服饰、袜子、首饰等）不行——款式一旦定了就很难改，如果没有提前规划好风格体系，产品就无法形成系列感，盲开款是死路。正确方式是：先企划品线（人群分析→风格体系→研发语言→视觉意向），再依此开品，并围绕内核讲文化故事。

## 核心问题

- 为什么非标品必须"企划先行"而不是"走一步看一步"？
- 企划的完整框架是什么？
- 潮趣袜和可露首饰各自怎么做？

## 方法框架：非标品企划流程

```
人群分析 → 潮流趋势勘察 → 风格体系定义 → 研发/工艺语言 → 视觉意向 → 品线关键词 → 系列开品 → 文化内核叙事
```

每一步都有具体工作：

| 步骤 | 内容 |
| --- | --- |
| 人群分析 | 这批人"潮"在哪？把潮的维度拆开（趣味潮、国潮、独立个性……） |
| 趋势勘察 | 看同类产品（鞋、服饰）的广告、设计、配色；看为什么消费者喜欢这些元素 |
| 风格体系 | 给整个品线一个核心定位词，下面拆出2-3个风格分支 |
| 研发/工艺语言 | 具体到：哪些工艺能体现风格？（织带、结绳、刺绣、文字、渐变……） |
| 视觉意向 | 确定色彩语言、情绪感受（多巴胺潮流感/都市自由感/先锋艺术感） |
| 品线关键词 | 系列命名方向（童趣元素、少年感、校园感、自然教育、甜美酷飒……） |

## 案例一：潮趣袜（2021年）

**PM背景**：核心企划是留学生出身，对潮流感知极强；这类PM不可替代，非标品需要对产品有深度审美理解的人来主导。大成老师的产品课也强调：一个品牌成功，背后必有一个强产品部门和能主导项目的PM。

**企划结果**：
- 核心定位词：**潮趣**
- 风格分支：先锋运动 / 时尚通勤 / 治愈萌趣 / 甜美酷飒
- 工艺语言：织带/结绳/刺绣/文字/渐变色
- 设计元素：多巴胺字母、扑克牌元素、艺术化元素、小怪兽（两只眼睛藏在长筒袜的脚底）
- 品线关键词：童趣元素 + 少年感 + 校园感 + 自然教育
- 叛逆分支：黑暗萝莉（肋骨花边）、斑马系列、野马系列、黑嘴唇

**2021年的企划基本预判了2024年的潮流方向**——这是企划先行最大的价值：用对，可以预测趋势，让产品有前置的系列感护城河。

## 案例二：可露首饰（香港轻珠宝品牌）

可露是非标品里从品牌理念出发开品的正面案例。

**品牌核心**：女性要探索自己的多种可能性（反对"嫁给一个男人"、鼓励探索自我）

**三大心智轴**：探索 / 自我 / 快乐

**产品设计**：把"探索"的谐音梗做成产品——字母锁头（你姓刘就买L锁，你姓孙就买S锁），叫"人生关键词系列"。产品与消费者本人直接绑定，像景区文创的"找你自己名字"一样，自带购买动机。

**品线延伸**：
- 探索系列（原点）→ 探索原点、探索经典、探索轨迹……
- 自我系列 → 女主角系列（刻名字定制）、释放自我
- 快乐系列 → 与 Smiley 笑脸联名做大（Smiley = 快乐的视觉载体）
- 2024年拓展：从"探索自己"升级为"探索关系"→ 人宠、情侣、母女、闺蜜系列

**内容延伸**：freestyle内裤品牌的操作思路类似——找代言人时挖掘其最强人格标签。发现代言人的核心标签是"偏见"（晒黑身材、纹身、秀身材，都是主流偏见点），围绕"偏见"展开直播间内容共创，切中当代独立女性最大的情绪痛点。

**荣誉佐证**：2021年FA亚洲最具影响力设计奖。创始人兼设计师凯琳：比较少被鼓励去探索，但女性拥有无限潜能。品牌的理念一直是"为大家带来启发和思考"。

## 核心原则

非标品的护城河不在工艺，在**已建立的系列感 + 文化内核**。大牌可以抄单款，但抄不了一个有内核的完整品线体系。先企划、后开品的品牌，每一款新品都在强化系列认知；盲开款的品牌，每款都是孤立的，也可以被单独抄走。

## 常见误区

- 没有美学能力的人主导非标品开发，产品视野窄，只跟着已有爆款模仿。
- 企划做了但只停在"关键词 PPT"层面，没有落到工艺语言和视觉意向，设计师无法执行。
- 文化内核讲不清楚，或没有持续通过内容强化，消费者不知道这个品牌代表什么。

## 可用于 Agent 的提示词

> 给定一个非标品品类（如：袜子 / 首饰 / T恤）和目标人群描述，请输出完整的非标品企划框架：1）潮流/人群维度拆解；2）核心定位词 + 2-3个风格分支；3）推荐工艺/材质语言；4）视觉情绪意向描述；5）3-5个品线关键词；6）文化内核叙事方向（1-2句品牌主张）。
