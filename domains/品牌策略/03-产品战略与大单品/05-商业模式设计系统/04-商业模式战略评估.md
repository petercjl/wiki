# 商业模式战略评估

## 问题定义

画布画出来后，还需要判断它是否适应外部环境、是否经得起竞争、是否有改造空间、是否会和组织内其他模式冲突。战略评估不是单独做 SWOT，而是把环境、SWOT、蓝海四项行动和多模式管理都落到九块画布上。

## 环境分析

商业模式环境可从四组力量看：

| 环境力量 | 要看什么 | 对画布的影响 |
| --- | --- | --- |
| 市场力量 | 客户需求、细分、转换、议价能力、收入吸引力 | 客户细分、价值主张、渠道、收入来源 |
| 行业力量 | 竞争者、新进入者、替代品、供应商、利益相关者 | 价值主张、伙伴、成本结构 |
| 关键趋势 | 技术、监管、社会文化、经济趋势 | 资源、活动、渠道、未来情景 |
| 宏观经济 | 全球市场、资本、商品、基础设施 | 成本结构、收入模型、关键资源 |

## 画布式 SWOT

不要只写“优势、劣势、机会、威胁”四个大框。应逐块问：

- 这块现在是不是强项？证据是什么？
- 这块是不是弱项？它会拖累哪些块？
- 外部机会会强化哪一块？
- 外部威胁会打击哪一块？
- 如果某块被重构，其他块需要怎样同步调整？

例如，一个强价值主张如果缺少可承受的渠道，仍可能无法成立；一个低成本结构如果伤害客户体验，也会反噬收入来源。

## 蓝海四项行动与画布

蓝海四项行动可以直接嵌入画布：

| 进入视角 | 问题 |
| --- | --- |
| 成本侧 | 哪些高成本资源、活动、伙伴可以减少或剔除？剔除后损失的价值如何补偿？ |
| 价值主张侧 | 哪些低价值功能可减少，哪些体验应增加或创造？ |
| 客户侧 | 应减少服务哪些客户，创造哪些新客户细分？新客户需要怎样的渠道、关系和收入机制？ |

关键是看连锁反应。价值主张改变会牵动渠道、关系、资源、活动和成本；客户细分改变会牵动定价、交付和伙伴。

## 多个商业模式管理

老牌企业引入新模式时，要同时评估：

- 新旧模式冲突程度：是否蚕食现有收入、品牌、渠道或资源。
- 战略相似度：是否服务同一战略方向，是否共享能力。
- 风险：是否带来品牌、法律、财务或运营风险。
- 组织选择：整合、独立、渐进整合或渐进分离。

如果新模式需要完全不同的客户、渠道、文化、物流和收入机制，独立实体常常更安全；如果新模式只是强化现有模式，整合可能更有协同。

## 案例与证据锚点

- Nespresso 与雀巢咖啡：高端胶囊咖啡需要直销、会员关系和不同物流系统，独立组织帮助它避开传统业务阻力。
- Swatch：在既有瑞士表产业中用低价、设计和品牌重构细分市场。
- Car2go：先在创新部门内设计和道路测试，再决定与核心汽车业务的组织关系。
- Wii：用蓝海视角重构游戏机价值和客户侧，而不是继续性能竞争。

## Agent 使用提示

当用户问“这个模式有没有机会”时，输出不应只给主观判断。先做环境四力、画布 SWOT、蓝海四项行动和新旧模式冲突评估，再给出保留、重构、分拆、试点或放弃建议。
