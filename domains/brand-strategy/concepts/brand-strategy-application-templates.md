# 品牌战略应用模板

## 摘要

本页把品牌战略理论转化为可直接使用的诊断模板、Agent 提示词和检查清单。它的用途不是学习概念，而是帮助 Peter 或 Agent 快速分析品牌、店铺、产品线、视觉、内容和案例。

## 1. 品牌三能力诊断模板

### 输入问题

请先收集：

- 品牌/店铺名称。
- 品类和核心产品。
- 当前规模和价格带。
- 主要渠道。
- 主要竞品。
- 当前增长瓶颈。
- 用户是否主动搜索品牌词。
- 同品同价转化是否有优势。
- 是否有持续溢价。

### 诊断问题

#### 动能

- 运营能力是否稳定？
- 供应链是否有优势？
- 渠道是否能持续获客？
- 投放和转化是否基本健康？
- 产品交付是否稳定？

#### 蓄能

- 定位是否清晰？
- 用户为什么买你而不是别人？
- 视觉是否承接定位？
- 产品是否证明品牌承诺？
- 是否有战略大单品？
- 内容是否围绕原点用户展开？

#### 势能

- 品牌是否已有一定认知？
- 是否有自然搜索和复购？
- 是否需要更大范围传播？
- 是否有足够利润支撑品牌声量建设？

### 输出格式

| 能力 | 评分 | 证据 | 问题 | 建议 |
| --- | --- | --- | --- | --- |
| 势能 | 1-5 |  |  |  |
| 动能 | 1-5 |  |  |  |
| 蓄能 | 1-5 |  |  |  |

## 2. 品牌蓄能四模块检查表

### 竞争战略

- [ ] 品类策略是否清楚？
- [ ] 目标用户是否具体？
- [ ] 是否有明确心智定位？
- [ ] 是否能一句话说清“为什么买我”？
- [ ] 是否避开了强竞品的优势战场？
- [ ] 增长路径是买更多、买更贵，还是更多人买？

### 感知系统

- [ ] 视觉是否匹配定位？
- [ ] 视觉是否匹配价格带？
- [ ] 主图、详情页、包装、内容是否统一？
- [ ] 用户是否一眼能看出差异？
- [ ] 是否有品牌记忆符号？

### 产品策略

- [ ] 产品是否承接定位？
- [ ] 是否有战略大单品？
- [ ] 产品命名是否有传播性？
- [ ] 产品细节是否能变成内容？
- [ ] 品阶结构是否清楚？
- [ ] 是否避免低价款稀释品牌？

### 营销系统

- [ ] 原点用户是谁？
- [ ] 内容是否围绕原点用户情绪和表达欲？
- [ ] 渠道选择是否匹配内容？
- [ ] 是否能把产品特性转化为社会话题或场景话题？
- [ ] 是否持续强化同一个心智？

## 3. 品牌案例拆解提示词

> 请基于我的 wiki 中的品牌战略知识体系，拆解以下品牌案例。输出：原始处境、竞争环境、关键洞察、战略选择、感知系统、产品策略、营销系统、结果、可复用规律、不可照抄部分。请不要只复述案例动作，要解释为什么这些动作有效。

## 4. 店铺/品牌诊断提示词

> 请基于品牌三能力模型和品牌蓄能框架，诊断这个电商品牌当前最缺势能、动能还是蓄能。请按竞争战略、感知系统、产品策略、营销系统分别指出问题，并给出优先级改进建议。

## 5. 电商图片品牌化分析提示词

> 请基于我的 wiki，从品牌感知系统角度分析这张电商图片。重点判断：图片是否承接品牌定位，是否体现价格感，是否让用户看见差异，是否证明产品价值，是否适合当前渠道场景。请输出缺点和优化方向。

## 6. 战略大单品分析模板

### 判断问题

- 这个产品是否能代表品牌核心价值？
- 它是走量品、利润品、心智品，还是三者兼具？
- 用户能否通过它记住品牌？
- 它是否有清晰卖点？
- 它是否有内容传播点？
- 它是否能带动其他产品？

### 输出结构

1. 产品角色：走量 / 利润 / 心智 / 引流 / 高端锚点。
2. 核心卖点。
3. 用户利益。
4. 信任证据。
5. 内容传播点。
6. 与品牌定位的关系。
7. 风险和改进方向。

## 7. 原点用户分析模板

### 判断问题

- 谁最需要这个品牌价值？
- 谁最容易被品牌打动？
- 谁最愿意表达和传播？
- 谁能影响更大人群？
- 这个人群的情绪、身份、场景和语言是什么？

## 8. 品牌定位句模板

品牌定位不应该只写“高端、专业、年轻、品质”。可以使用：

> 面向【目标用户】，在【品类/场景】中，提供【核心利益/情绪价值】，通过【产品/视觉/证据】形成【差异化记忆点】。

示例结构：

- 面向精致中产父母，在 0-3 岁学步鞋场景中，提供兼具安全与高端时尚的育儿表达，通过高端设计、稳走产品和精致视觉形成“高端时尚学步鞋”心智。
- 面向高知精致女性，在人生重要时刻的鞋履场景中，提供闪耀、仪式和体面的情绪价值，通过婚鞋设计、爱心鞋底和内容传播形成“人生大事闪耀时刻”心智。

## 9. 常见诊断结论句式

- 这个品牌不是缺流量，而是缺清晰的购买理由。
- 这个品牌不是视觉不好看，而是视觉没有承接定位。
- 这个品牌不是产品不够多，而是缺少能树心智的战略大单品。
- 这个品牌不是不能做内容，而是没有找到原点用户和传播触发点。
- 这个品牌现在不适合直接做势能，应该先补蓄能。
- 这个品牌可以不急着拓品类，先判断是否能拓场景、拓人群、拓价格带。

## 10. 使用边界

- 本模板适合电商品牌、消费品品牌、内容驱动型品牌。
- 对工业品、强渠道品牌、纯 B2B 品牌，需要调整用户心智和渠道判断。
- 课程案例中的数字和结果如用于外部报告，需要二次核对。
- 大模型可以补充逻辑分析，但不能编造品牌事实。
