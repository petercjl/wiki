# 05｜功能如何被看见：爱眼饮品的视觉直达案例

## 核心判断

爱眼饮品用视力检测表把抽象功效转成消费者一眼看懂的视觉表达。

## 方法含义

本节用于把课程中的口头经验重构成稳定的方法框架。重点不是复述课堂顺序，而是把判断标准、行动顺序、案例证据与适用边界整理成可复用知识。

## 康源案例的证据价值

康源爱眼饮品的价值，不在于包装做得多复杂，而在于它直接借用“视力检测表”这种用户天然认得的视觉符号，让功能利益不需要解释就能被看见。课程中还把它放进一个更强目标里——三个月冲到 1 亿体量——强调这种设计不是装饰，而是为了提升货架环境里的识别效率和传播效率。
