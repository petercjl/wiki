# 全站推广打款放量流程

全站推广的核心不是一上来追求漂亮投产，而是按阶段回答三个问题：这个款能不能成交、成交人群是谁、能不能在更高预算下继续稳定成交。前期用成交打标，中期用目标投产校正人群，后期用预算和内容补充放大。

## 一、启动前先确认能不能打

不要因为手里有一个产品就直接放大。启动前至少看四件事：

1. 产品是否有确定性：例如已有货源、专利、去年同类卖得好、类目有延续性。
2. 竞品是否证明需求存在：看竞品流量价格、访客量、推广计划、销售量和链接矩阵。
3. 自己是否具备基础承接：评价、主图、详情页、价格、货源和库存不能明显拖后腿。
4. 市场体量是否足够：如果整个细分市场访客很少，预算再大也很难强行放量。

如果产品本身转化率太低，或者宝贝基础、评价、内功、图片、人群明显不匹配，优先优化基础，而不是继续研究要不要加预算。

## 二、第一阶段：最大化放量拿成交标签

前期目标是让系统知道谁会买，而不是马上把投产打到最终盈利线。这个阶段可以接受一定人群偏差，因为成交标签比理论精准更重要。

适合进入第一阶段的信号包括：

- 链接已经出现成交属性。
- 成交人群和预期买家基本一致。
- 平台出现推荐或趋势信号。
- 关键词或其他入口已经能打动，证明商品有基础需求。

第一阶段常见动作：

1. 用最大化或智能计划先把量放出来。
2. 配合关键词或手动计划补充触达，纠正人群。
3. 观察哪个链接、哪个人群、哪个素材先跑出来。
4. 不急着只看 ROI，先看成交量、加购、转化和人群是否成形。

课程中强调，搜索流量在很多类目里不是无限的，后期搜索占比可能只有约 20%，有些情况下做到 30% 已经不容易。因此，不能把所有增长希望都押在搜索上。

## 三、第二阶段：从最大化切到目标投产

当成交标签已经有基础，就可以逐步转向目标投产。切换时不要一开始就设最终目标，而是从低于保本的目标开始，让系统继续跑量，同时慢慢把人群拉正。

示例逻辑：

- 如果保本投产是 3，第一次目标可以先设 1.5 或 2。
- 如果宝贝基础好、老店人群清楚，可以稍微高一些。
- 如果跑量慢、效果一般，先低一点，让系统有学习空间。
- 每次目标上调可以按约 0.2-0.4 递增，具体看类目、客单价和跑量速度。
- 调整后观察 8-24 小时，不要刚加完就连续大幅加码。

这个阶段的重点不是一次到位，而是让跑量速度、投产目标和人群质量三者同时变好。

## 四、预算不是越大越好

预算决定的不只是花多少钱，也会影响系统拿到的人群范围和跑量速度。同一个产品、同一个目标，不同预算可能跑出完全不同的人群质量。

课程里出现过多种预算锚点：50-100、100、200、300、500、1000、1500、2000 等。这些数字不是固定公式，而是提醒运营要分层测试预算，不要从极低预算直接跳到失控预算。

预算调整规则：

1. 先看计划是否跑得动。
2. 再看成交量、转化率、投产、加购成本是否变好。
3. 如果花费增加但转化和投产差，缩小计划限额，把预算挪给效率更好的计划。
4. 如果投产打正且市场仍有空间，可以继续扩大预算。
5. 如果预算拉高后人群变差，先调目标或素材，不要只靠继续加钱。

全站放量时，最怕计划已经学习出稳定匹配，又被频繁来回切换。大幅度切换会让系统重新匹配，成本由商家承担。

## 五、计划效率复盘

每天或每个观察周期要把计划拉出来看：

| 检查项 | 判断 |
| --- | --- |
| 花费 | 哪个计划花得最多 |
| 转化率 | 高花费计划是否带来有效成交 |
| 投产 | 高花费计划是否值得继续放大 |
| 加购/收藏成本 | 是否在种草但没有收割 |
| 人群质量 | 成交人群是否符合预期 |
| 素材点击 | 素材是否足以支撑后续放量 |

如果某个计划花费高、转化差、投产差，不要因为它能消耗就保留预算。更合理的做法是缩限额，把预算给下方更有效率的计划。

## 六、常见失败原因

第一，转化率太低。根因通常不是投放，而是产品、价格、评价、主图、详情页、素材或人群不匹配。

第二，成交量太少，没有标签。没有足够成交证据时，系统很难判断该找谁，急着切投产目标会让效果变差。

第三，产品体量太小。细分市场本身访客少、竞品流量也少，就不能指望全站无限放大。

第四，只做拉新不做收割。前端花钱引入的人，如果没有后端智能计划、关键词、老客或重定向承接，可能被竞品收割。

第五，频繁切换计划。计划刚学到人群又切换目标或入口，系统需要重新匹配，成本和波动都会变大。

## 七、案例型判断

课程中的手提包案例强调：当产品转化率、价格、评价和页面基础已经能承接，投产目标可以从较低目标逐步往上拉，花费也可以逐步放大。过程中可能经历前期亏损，但目标是用短期亏损换取成交标签、系统学习和后期稳定放量。

这类打法不适合所有店铺。老店、人群基础强、宝贝基础好时，更可能一边放量一边打正；多数普通链接前期需要承受一定试错成本。

## Agent 判断模板

当用户问“我的全站为什么跑不动/投产差/要不要加预算”时，先按顺序判断：

1. 产品和市场空间是否足够。
2. 链接基础是否能承接转化。
3. 是否已经有成交标签。
4. 当前阶段是拿成交、拉新、转投产，还是稳定放量。
5. 预算、目标和人群是否匹配。
6. 是否缺少收割计划。
7. 是否需要短视频或内容补充拉新。
