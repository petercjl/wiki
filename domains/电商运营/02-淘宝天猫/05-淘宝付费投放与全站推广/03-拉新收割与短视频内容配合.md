# 拉新收割与短视频内容配合

投放不是只把新人拉进来。课程里最重要的风险提醒是：如果前端持续拉新，但后端没有收割，商家可能是在用自己的预算替竞争对手种草。

## 一、拉新和收割要成对设计

拉新的任务是让更多符合产品的人看到、点击、加购或产生购买意向。收割的任务是把已经被触达、已经加购、已经看过或已经有需求的人重新带回成交。

如果只有拉新：

- 前端成本会不断上升。
- ROI 可能持续变差。
- 用户第一次看到不买，第二次再想买时可能被竞品拿走。
- 商家实际承担了市场教育成本，却没有拿到后端转化。

因此，一个完整投放结构至少要问：

1. 哪些计划负责拉新。
2. 哪些计划负责成交和收割。
3. 哪些人群已经被触达但未成交。
4. 哪些人群需要排除，避免重复浪费。
5. 哪些素材负责种草，哪些素材负责转化。

## 二、短视频适合在全站稳定后补充上层流量

短视频内容不是一开始就替代全站。课程中更推荐的路径是：当全站计划已经稳定、有目标投产、有明确商品和人群后，再用短视频做上层种草和拉新。

适合加短视频的信号：

- 全站计划到达稳定期，继续加预算上不去。
- 商品 ROI 已经验证过，宝贝和人群比较正。
- 预算相对稳定，能承接更多前端种草。
- 主推商品已经明确，不是随便拿一堆商品试。

短视频的目标可以不是立刻高投产，而是以较低成本带来新用户、点击、加购和可收割人群。

## 三、短视频素材怎么判断

课程特别强调视频的考核不只看播放，至少要看：

- 前三秒是否能抓住人。
- 点击进入率是否高。
- 点击后是否有加购。
- 是否有转化。
- 加购成本和成交成本是否能接受。
- 素材带来的人群是否和商品匹配。

如果某个短视频创意在点击、加购或转化上表现好，可以再用控成本点击、控成本成交等方式复制出来，支持主推商品持续获得新流量。

## 四、人群选择与排除

短视频拉新可以选择与商品相关的人群，而不是只追求大范围曝光。课程中提到的可用思路包括：

| 人群方向 | 用法 |
| --- | --- |
| 相似店铺 | 用于拉新，找到和目标店铺相近但尚未转化的人群 |
| 自己正在推的宝贝 | 让短视频围绕主推商品，而不是泛内容 |
| 关键词/类目行为 | 找到搜索过、浏览过相关类目但未购买的人 |
| 平台推荐人群 | 如高消费、会员或平台判断转化较好的人群，需后台复核 |
| 已购买人群排除 | 拉新时排除近期已买用户，避免把老客当新客重复触达 |
| 地域与偏远地区 | 根据商品履约、利润和流量质量决定是否开放 |

核心原则：拉新计划要尽量给新人看，而不是把预算花在已经购买或已经能自然成交的人身上。

## 五、不要频繁切换

当计划已经学习到稳定人群后，不要在全站、短视频、目标投产之间频繁来回切。大幅度操作会打断系统学习，导致重新匹配。

可以调整，但要遵守两个原则：

1. 调整有明确目的：拉新、收割、提投产、放预算、测素材。
2. 调整后给系统观察周期，不要连续干预。

## 六、内容和投放的协同

投放负责找人，内容负责让人有理由停留、点击、加购。短视频内容要和产品卖点、主图、详情页一致，而不是另讲一套故事。

如果素材在短视频里跑出低成本点击和加购，后续应同步回到：

- 主图表达。
- 详情页首屏。
- 问大家和评价引导。
- 直播或客服话术。
- 其他投放素材。

这样短视频不是孤立内容，而是主推商品增长系统的一部分。
