# 链接发散与人群素材测试

一个确定性产品跑不起来，不一定说明产品不行，可能只是给它的链接、场景、人群和素材机会太少。课程中反复强调：同一产品可以通过不同场景、不同人群、不同主图和不同链接来寻找成交入口。

## 一、不要只用一个链接判断产品

当产品本身有确定性，例如有专利、有货源、历史同类卖得好、类目有延续性，就不要只上一个链接等结果。更好的做法是给产品更多测试机会。

可以发散的维度：

| 维度 | 例子 | 目的 |
| --- | --- | --- |
| 场景 | 拖鞋可按秋冬、居家、浴室、不同材质场景拆 | 找出触发购买的具体使用环境 |
| 人群 | 女士、学生、家庭、特定消费层级 | 找出更容易成交的人 |
| 主图 | 不同利益点、不同视觉风格、不同场景图 | 找出点击率和成交人群更好的入口 |
| 颜色/规格 | 同款不同颜色、不同规格单独链接 | 用低成本测试额外成交支撑 |
| 价格/套餐 | 单件、组合、赠品、不同价格带 | 找出转化效率和利润平衡点 |

课程里提到的 `5 x 2 x 3` 和十个链接式发散，不应理解为固定公式，而是提醒运营：在第一步就用较高效率筛选多个链接机会。

## 二、先宽后细

前期为了更快拿到信号，可以先选择和产品相关的较宽主题或较宽人群，不必一开始就拆到极细。只要方向与产品匹配，就可以先跑出初步成交。

当数据出现后，再把主题、人群和素材细分出来精测。这样的好处是：

1. 前期拿量更快。
2. 能避免过早陷入小样本判断。
3. 先找出大方向，再优化精细人群。
4. 后续可以把好人群、好素材复制到其他计划。

## 三、人群和素材要一起测

只测人群、不测素材，会导致后期放量周期变长。因为即使人群找对，如果主图或创意点击没有解决，后续扩量仍会浪费。

前期应同时记录：

- 哪些人群成交更好。
- 哪些素材点击率更高。
- 哪些素材带来的加购、成交、转化更好。
- 哪些链接先跑出成交量。
- 哪些素材适合后续成为大量创意模板。

判断一个素材不是只看好不好看，而是看它是否能在目标人群里带来低成本点击、加购和成交。

## 四、同款多链接的使用边界

同款多链接适合这些情况：

- 产品基础确定，但单链接没有给出足够机会。
- 颜色、规格、款式较多，且不同组合可能面对不同买家。
- 标品中存在多个颜色或规格能单独承接需求。
- 需要用小预算测试额外营业额支撑。
- 需要训练系统识别更好的成交人群。

不适合的情况：

- 产品基础差，转化率低。
- 主图、评价、详情页、价格都没有做好。
- 市场体量太小，访客本身不够。
- 只是为了堆链接，而没有明确人群、场景或素材假设。

## 五、从测试变成资产

测试的最终结果不是得到一个“能不能跑”的结论，而是沉淀资产：

1. 可继续放量的链接。
2. 可复制的人群包。
3. 可复用的高点击素材。
4. 可迁移的主图表达。
5. 可进入下一轮全站或短视频投放的创意。

后续做新款时，这些资产可以缩短从市场到产品再到推广的周期。
