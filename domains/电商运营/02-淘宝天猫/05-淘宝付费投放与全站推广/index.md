# 淘宝付费投放与全站推广

本目录沉淀淘宝/天猫平台内付费投放、全站推广、打款放量、人群标签、短视频拉新和收割配合的方法。当前主来源是 **山峰组253期6-14课程**，课程重点不是泛泛讲流量，而是把一个产品从测链接、拿成交、打标签、转投产、加预算，到短视频补充拉新的放量过程拆成可执行步骤。

## 使用边界

来源文本存在明显 ASR 错字，具体工具名、后台按钮和平台规则需要以当前淘宝/天猫后台为准。本目录保留课程中的操盘逻辑：阶段目标、判断标准、预算节奏、人群和素材测试、拉新与收割配合。

## 阅读顺序

1. [[01-全站推广打款放量流程|全站推广打款放量流程]]：从候选产品到最大化、目标投产、预算递增和失败诊断。
2. [[02-链接发散与人群素材测试|链接发散与人群素材测试]]：同款多链接、场景、人群、主图和素材测试方法。
3. [[03-拉新收割与短视频内容配合|拉新收割与短视频内容配合]]：前端种草、后端收割、短视频内容投放和人群排除规则。
4. [[04-Agent使用模板：淘宝全站推广诊断|Agent 使用模板：淘宝全站推广诊断]]：未来 Agent 诊断店铺投放、放量或全站推广问题时的输入和输出结构。

## 核心模型

这套方法的主线是：

1. 先从市场和竞品判断产品是否值得打。
2. 用同款多链接、多场景、多素材给产品足够测试机会。
3. 前期先拿成交和人群标签，不急着追求完美投产。
4. 用全站/智能/关键词/手动计划分工完成放量、纠偏和触达。
5. 当成交标签和买家画像足够清楚后，再切到目标投产。
6. 通过小幅提高投产目标和预算，把跑量与盈利慢慢拉平。
7. 当全站进入稳定期后，用短视频内容补充拉新和种草。
8. 把成功款沉淀为 A/B/C 款梯队和可复制的市场-产品-推广流程。
