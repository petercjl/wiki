# 标准计划智能计划与出价模型

淘宝推广计划不是越智能越好，也不是越手动越精准越好。标准计划和智能计划解决的是不同问题：标准计划负责可控、纠偏、打标；智能计划负责效率、宽度、收割和规模。

## 标准计划的价值

标准计划的优势是可控：

- 可以锁定核心关键词。
- 可以锁定目标人群。
- 可以控制时间、地域、出价和溢价。
- 可以用来提升核心关键词和目标人群权重。
- 可以纠正智能计划或推荐流量带来的人群偏差。

适用场景：

- 精准测图。
- 人群纠偏。
- 中高客单价产品打标拉新。
- 新品需要先建立精准标签。
- 目标人群非常明确，不能完全交给系统泛化。

标准计划的弱点是转化效率和投产可能低于智能计划，因为它从更外层、更冷的人群开始打，需要更长链路让 O 人群进入 A/I/P。

## 智能计划的价值和风险

智能计划的优势是效率高、流量宽：

- 能探索更多关键词和类目优选流量。
- 能覆盖更宽的人群。
- 更容易优先触达已有点击、收藏、加购、购买等行为的人群。
- 收割属性强，短期转化效率通常高于标准计划。

但智能计划也有风险：

- 最大化成交可能压制免费搜索或已付费带来的二次回流。
- 预算放大时 PPC 和成交成本可能同步上涨，点击量未必增加。
- 控成本或目标优化可能导致用户画像、消费力和产品标签跑偏。
- 新品如果还没有精准人群标签，智能计划可能拿量但数据差。

课程的建议是：新品或中高客单产品，最好先用标准计划把人群标签拉出来，再配合智能计划做效率和扩量。

## 测图要匹配后期主推计划

课程特别提醒：如果后期主要用智能计划打，单纯用标准计划测图，可能测不出真实效果。因为标准计划和智能计划的流量、人群和关键词不同。

更稳妥的做法：

- 后期用什么计划类型最多，就用什么计划类型测图。
- 标准计划和智能计划测出来都好的图，可信度更高。
- 如果二者结果冲突，以后期主推计划的结果为准。
- 如果全站或智能计划主要跑搜索流量，标准计划测图仍有参考价值。

## 控成交成本

控成交成本是双向约束：

```text
预算约束 + 成交成本约束
```

系统以设定的成交成本为核心，优先匹配预计能在目标成本内转化的流量。如果某个流量很适合宝贝，但抢它的成本超过目标成交成本，系统可能放弃。

适用：

- 有稳定成交人群的产品。
- 中高客单或有一定成交基础的产品。
- 希望控制单笔成交成本的动销计划。

风险：

- 目标成本设得过低，流速慢。
- 新品没有稳定成交人群时，容易跑不动或拿错人群。

## 控投产

控投产同样是双向约束，但核心目标是投产比。

```text
预算约束 + 目标投产约束
```

目标投产越低，拿量能力越强；目标投产越高，拿量能力越弱。

它适合追求利润稳定的阶段，但不适合一边喊打爆款、一边又把投产目标设得很高。课程中的砂锅案例说明：如果想冲到行业前排，又不愿意承受前期花费，计划就很难放大。

## 最大化投放

最大化投放是单向预算约束：

```text
预算约束
```

系统会在预算范围内尽量抢流量，不严格受单次点击成本限制。预算越高，可能越激进；预算放大后，PPC 也可能上涨，导致花费增加但点击量没有同步增加。

适用：

- 大促抢流量。
- 高客单或竞争激烈市场抢高意向流量。
- 清库存或冷启动需要快速拿量。

注意：

- 不适合完全没有标签、没有转化基础的新品盲目大预算跑。
- 放量后要看点击量是否真的增加，而不是只看预算是否花出去了。

## 手动出价

手动出价更依赖关键词出价、质量得分和预算。它的价值是可控，适合对关键词和人群有明确判断的场景。

课程提到的常见结构：

| 结构 | 适用 |
| --- | --- |
| 低出高溢 | 吃人群、中高客单、目标人群明确 |
| 中心出价 | 关键词和人群都重要，较均衡 |
| 高出低溢 | 不太吃人群，关键词需求明确 |

## 低出高溢

低出高溢的核心是：降低关键词基础出价，提高目标人群溢价，让点击更多落在想要的人群上。

判断重点：

- 人群点击占比是否足够高。
- 人群点击率、加购率、转化率是否优于计划平均。
- 高价换来的是否是高点击、高加购、高转化，而不是纯粹高 PPC。

课程给出一个锚点：如果目标人群点击占比低于 60%，计划可能有问题，需要调整。

常见调整：

1. 降低关键词出价。
2. 提高目标人群溢价。
3. 减少或调整关键词数量。
4. 压低分时折扣。
5. 若仍无法落在人群上，可测试流量自选低价策略。

## 关键词包和计划阶段

不同关键词包服务不同阶段：

| 关键词/词包类型 | 适用 |
| --- | --- |
| 店铺专有词、品牌词 | 新品快速打标、代理品牌防守、点击率提升 |
| 精准类目词 | 新品搜索入库、核心需求验证 |
| 行业大词 | 爆款冲量、抢市场、强拉新 |
| 趋势主题词包 | 非标批量测款、趋势拉新、容量扩展 |

新品不一定一开始就打行业大词。先用更容易产生点击和标签的词，让计划稳定，再逐步扩词。

## 选择原则

```text
标准负责精准，智能负责效率。
标准负责纠偏，智能负责扩量。
短转化看成交，长转化看加购。
新品先打标，爆款再抢量。
```
