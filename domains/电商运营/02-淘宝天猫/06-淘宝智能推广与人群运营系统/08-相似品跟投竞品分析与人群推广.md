# 相似品跟投竞品分析与人群推广

相似品跟投和人群推广的目标，是让商品借助竞品人群和平台人群包找到更高概率成交的人。它们不是盲目抢竞品，而是先判断自己对哪个竞品有优势，再决定跟投。

## 相似品跟投的阶段选择

| 产品阶段 | 可用策略 |
| --- | --- |
| 新品期 | 新品跟爆 + 相似跟投，用于建立爆品人群画像、打标、积累权重 |
| 成长期 | 乘胜追击、流失防守 |
| 爆品期 | 相似跟投、乘胜追击、流失防守、竞争拉新都可测试 |
| 平销期 | 重点测试乘胜追击和流失防守 |

新店默认开启智能跟投时，可能是黑盒模式；消费达到一定规模后，可考虑关闭智能跟投并启用自定义策略。课程提到 5 万元消费作为一个可参考门槛，但实际要结合后台权限和店铺数据。

## 竞品筛选

跟投不是找最大竞品，而是找“打得过”的竞品。

课程提到几条路径：

| 路径 | 判断 |
| --- | --- |
| 竞争解析流失榜单 | 流失度 ≥ 6 的竞品具备对抗价值 |
| 流入/流出关系 | 流入度 > 流失度，说明自己对竞品有一定优势 |
| 竞争态势分析 | 购买人数控比 > 50%，说明对竞品有相对优势 |
| 竞品流失分析 | 流入资产 > 流出资产，说明该竞品可打 |
| AI选品 | 系统匹配点击人群重合度高、价格高出 10%-20% 的相似商品 |

这里的逻辑是：优先打与你人群重合、但你在价格、评价、款式、服务或转化上有优势的竞品。

## 设置建议

相似品跟投建议：

- 关闭无效流量自选。
- 手动添加竞品成交词。
- 使用广泛匹配。
- 预算控制在 200 元以内。
- 小预算稳定后再放大。

预算太大容易导致 PPC 飙升，尤其是在系统还没有稳定识别竞品人群时。

## 人群推广资源位

课程将人群推广资源位理解为不同任务：

| 资源位 | 任务 |
| --- | --- |
| 猜你喜欢 | 拉新 |
| 全屏微详情 | 种草 |
| 购中/购后猜你喜欢 | 收割 |
| 信息流追投 | 搜索再营销 |

同一个商品不要把拉新和收割混成一个模糊计划。拉新和收割应分计划运行，便于看数据和控预算。

## 人群定向精度

课程给出一个人群定向精度排序：

```text
达摩盘圈人 > 基础属性 > 平台精选 > 关键词人群 > 宝贝/店铺人群 > 智能定向
```

智能定向不是不能用，但它对宝贝标签要求高。宝贝标签不准时，智能定向容易导致画像跑偏。

达摩盘圈人适合更精细的人群筛选，尤其当店铺已经知道高消费力、年龄、性别、兴趣标签和购买行为特征时。

## 人群推广计划类型

| 类型 | 用法 |
| --- | --- |
| 自定义推广 | 单品投放，分计划做“喜欢相似宝贝”和“喜欢我的宝贝” |
| 高效拉新 | 单一类目清晰的大店，可用首购拉新 + 同叶子类目拉新 |
| 常客转化 | 长周期转化产品补充收割，预算 50-100 元 |
| 人群超市 | 类似人群包，可尝试核心人群增长包、行业必争人群包、高购买力包 |

老客复购不一定要靠推广，很多时候更适合私域运营。推广更适合拉新、潜客回访和高价值人群触达。

## 与达摩盘 AI 的关系

这堂课中的人群推广和竞品筛选，可以和 [[domains/电商运营/02-淘宝天猫/04-达摩盘AI/index|达摩盘 AI]] 记忆结合使用：达摩盘 AI 负责挖掘竞品、人群、关键词和数据；本页负责把这些人群和竞品选择转为推广计划。
