# 淘宝智能推广与人群运营系统

本目录基于 **山峰组259期6-15/6-16课程**，把淘宝智能推广、人群资产、计划类型、归因口径、搜索卡位、人群推广和报表优化重构成一套可诊断、可执行的知识系统。

这套系统的核心不是“多开计划”，而是：

```text
平台趋势 -> 店铺模型 -> 产品转化周期 -> 人群资产 -> 流量渠道 -> 计划类型 -> 指标归因 -> 放量/纠偏
```

如果不知道店铺靠新客、潜客回访还是老客复购增长，就会把拉新计划开成收割计划；如果不知道产品转化周期，就会把长决策品当天无成交误判为计划失败；如果不知道计划人群浓度，就会在错误人群上继续放量。

## 阅读顺序

1. [[01-淘宝运营趋势与增长底层逻辑|淘宝运营趋势与增长底层逻辑]]：理解存量市场、AI、合规、品牌化和曝光价值公式。
2. [[02-店铺模型与新老客增长结构|店铺模型与新老客增长结构]]：用 721、172、333、127 判断销售额来自哪里。
3. [[03-人群资产与计划健康诊断|人群资产与计划健康诊断]]：用流转效率、人群资产规模和 TA 人群浓度判断计划能不能放大。
4. [[04-转化周期归因与流量渠道选择|转化周期归因与流量渠道选择]]：按搜索、推荐、直播、站外和转化周期选择指标。
5. [[05-标准计划智能计划与出价模型|标准计划智能计划与出价模型]]：区分标准、智能、控成本、控投产、最大化和手动出价。
6. [[06-搜索卡位流量金卡与活动资源|搜索卡位流量金卡与活动资源]]：使用搜索卡位、好货快投、流量金卡和活动场域。
7. [[07-测图测款动销与低价引流计划|测图测款动销与低价引流计划]]：搭建测图、测款、动销和低价引流计划。
8. [[08-相似品跟投竞品分析与人群推广|相似品跟投竞品分析与人群推广]]：选择竞品、做人群推广和相似品跟投。
9. [[09-推广空烧互抢与报表优化|推广空烧互抢与报表优化]]：诊断空烧、互抢和预算结构。
10. [[10-Agent使用模板：淘宝智能推广诊断|Agent 使用模板：淘宝智能推广诊断]]：给未来 Agent 的诊断输入和输出结构。
