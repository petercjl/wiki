# 推广空烧互抢与报表优化

计划空烧通常不是单一原因，而是人群、计划类型、预算、词路和渠道之间错配。诊断时要先判断计划承担的是拉新、收割、测款、动销还是再营销。

## 五大空烧原因

| 原因 | 表现 |
| --- | --- |
| 人群混乱或标签偏移 | 点击有了，但不是目标人群，转化差 |
| 计划类型错配 | 用促点击做收割，或用收割计划期待拉新 |
| 智能计划未满足打标条件 | 成交不足、频繁改价、数据不稳定 |
| 计划间互抢 | 同类型、同人群、同词路计划争同一批流量 |
| 预算过大 | 系统为花完预算而泛化流量 |

一个计划投产差，不一定该马上关。先看它是不是还在转化周期内，是否有加购、询单或潜客贡献。

## 计划互抢

不是所有计划都会互抢。

课程判断：

- 关键词推广和人群推广通常不互抢，因为渠道不同：搜索 vs 推荐。
- 关键词推广和超级短视频、线索推广、全站推广可能互抢，因为共享部分资源位或搜索链路。
- 同类型、同词路、同人群包、同时间段的计划更容易互抢。

## 互抢规避策略

| 维度 | 做法 |
| --- | --- |
| 类型错开 | 拉新 + 收割搭配，例如手动拉新 + 智能收割 |
| 时间错开 | 按转化高峰时段分布投放 |
| 地域错开 | 高转化区域单独建计划 |
| 词路错开 | 大词、长尾词、趋势词、品牌词分计划 |
| 人群错开 | 不同人群包分计划 |
| 溢价错开 | 同一人群不同计划设置不同溢价，例如 1.1 vs 1.3 |

互抢的本质是：多个计划在争同一个人、同一个词、同一个时段、同一个资源位，却承担相同任务。

## 推广效能分析表

课程建议下载 30 天计划报表和商品报表，合并分析。

分析目标：

- 各渠道花费占比。
- 各渠道投产比。
- 商品维度的花费和成交。
- 哪些计划花费高但投产低。
- 哪些计划花费低但投产好，值得加预算。

课程举例：某店关键词推广花费占比高但投产差，人群推广只占 5.1% 却有 1.9 投产，说明预算结构应调整。

## 人群效果分析表

人群报表可按 30 天汇总，统计：

- 人群包花费。
- 点击率。
- 转化率。
- 投产比。
- 引导潜客占比。
- 对哪个宝贝有效。

这个表的价值是把“哪个人群有效”从感觉变成数据。后续投放不再笼统地说“人群推广没用”，而是判断哪个人群包、哪个宝贝、哪个转化目标有效。

## 诊断流程

1. 判断产品转化周期。
2. 判断计划任务：拉新、收割、测款、动销、再营销。
3. 看计划人群浓度。
4. 看加购、咨询或成交是否符合任务。
5. 看是否仍在转化周期内。
6. 看是否与其他计划词路、人群、时间、资源位重复。
7. 看 30 天花费占比与投产结构。
8. 决定调预算、拆计划、错开人群、暂停或重建。

## 不要只看投产

投产是结果指标，不是唯一诊断指标。对长周期产品和高客单产品，计划可能先贡献加购、收藏、咨询和潜客。对短周期产品，直接成交和引导新客更重要。

报表优化要和产品类型、转化周期、计划目标一起看。
