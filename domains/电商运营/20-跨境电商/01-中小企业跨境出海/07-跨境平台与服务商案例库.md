# 07｜跨境平台与服务商案例库

## 问题定义

案例的价值不在于“谁成功了”，而在于抽取可迁移的经营机制：定位、数据、履约、支付、合规、本地化和组织能力。

## 案例迁移表

| 案例 | 书中角色 | 可迁移机制 | Agent 使用方式 |
| --- | --- | --- | --- |
| PatPat | 母婴/童装 DTC | 种子用户、超级用户、产品共创、社群反馈 | 用于设计核心用户验证和本地化选品 |
| Lululemon | 社区/口碑营销 | 品牌大使、社区活动、用户信任传播 | 用于设计 UGC 和品牌社群 |
| SHEIN | 实时零售 | 数据驱动、快速上新、供应链响应 | 用于讨论快反和数据闭环，但需注意组织门槛 |
| Temu | 平台与全球营销 | 本地化营销、合规战略、低价心智 | 用于平台化增长与合规压力分析 |
| 传音控股 | 区域市场适配 | 搜索与社交布局、本地需求理解 | 用于新兴市场定位参考 |
| 安克创新 | 品牌出海 | 产品力、数字营销和品牌影响 | 用于从卖货转品牌的路径讨论 |
| 菜鸟/纵腾/京东物流/递四方 | 物流服务商 | 海外仓、专线、本地配送、可视化履约 | 用于物流模式选择 |
| Lazada/阿里国际/速卖通/小红书 | 平台/社区 | 本地运营、退货、社区触达 | 用于本地化运营设计 |
| Ozon/亚马逊/杭州互联网法院 | IP 与平台治理 | 平台保护机制、司法/平台联动 | 用于知识产权保护策略 |
| PayPal/连连支付/PingPong | 支付服务 | 信任、安全、资金效率、多币种管理、生态协同 | 用于支付方案和现金流方案 |
| 敦煌网/Woot/TikTok/赛维时代/北仑/有棵树 | 合规与风险 | 隐私保护、平台合规、风险预警、韧性经营 | 用于风险管理和合规体系设计 |

## 使用原则

1. 不把大企业案例直接复制给中小企业。
2. 先抽取机制，再判断适用条件。
3. 对时间敏感事实保留来源日期和复核要求。
4. 案例只能作为启发，不能替代当前平台数据和企业自身约束。
