# 产品数据精细化诊断SOP

产品维度诊断不需要平均用力。课程建议重点分析爆品、实力优品和已经沉淀较多数据的产品，因为这些产品承接了店铺主要流量和权重；数据极少、长期零效果的产品，应优先判断是否下架、重发或放弃，而不是投入大量诊断时间。

## 先看产品结构是否健康

店铺需要动态产品结构，而不是只依赖少数老爆品。

| 产品类型 | 作用 | 诊断问题 |
| --- | --- | --- |
| 爆品 | 承接核心流量和询盘。 | 是否稳定，是否存在下滑风险。 |
| 实力优品 | 提升平台分层和搜索/推荐机会。 | 数量是否足够，是否接近爆品。 |
| 新品 | 持续挖掘潜力品。 | 是否持续发布和测试。 |
| 半托管/RTS 产品 | 打开交易型流量入口。 | 是否适合店铺赛道和目标买家。 |
| 引流款/跑量款/利润款 | 承接不同买家和经营目标。 | 资源倾斜是否清晰。 |

爆品和实力优品有生命周期，不能假设它们永远稳定。健康店铺应持续让新品进入测试池，让潜力品成长为实力优品和爆品。

## 诊断产品转化效果

第一步看产品是否还在正确类目下。平台类目拆分、调整或迁移后，原本表现好的产品可能因为类目不匹配或新类目权重低而短期下滑。

检查顺序：

```text
产品所属类目
-> 产品类型和状态
-> 产品 360 单品数据
-> 关键词和访客精准度
-> 价格、MOQ、详情页和竞品差距
-> 流量来源和推广计划
```

如果爆品突然推广花不出去、曝光减少或转化下降，先查类目和产品状态，再查标题、关键词、国家和推广计划。

## 产品 360 单品分析

对重点单品进入产品 360，至少看六个问题：

| 问题 | 判断方向 |
| --- | --- |
| 访客是否准确 | 访客国家、买家类型和类目偏好是否匹配目标买家。 |
| 曝光词和点击词是否准确 | 高曝光、高点击词是否真的对应产品款式和目标国家。 |
| 价格和 MOQ 是否合理 | 是否符合目标市场采购预期。 |
| 详情页是否能承接 | 卖点、参数、信任证据和询盘入口是否清楚。 |
| 竞品是什么水平 | 同类产品的价格、MOQ、款式、详情页和询盘数据如何。 |
| 流量来源是什么 | 自然、直通车、推荐或活动中哪类渠道贡献效果。 |

如果该产品正在投直通车，还要拆到对应计划，对比产品 360 和直通车报告，不要只看单边数据。

## 关键词商机转化率

单个关键词不仅要看点击，还要看转化。点击多但商机转化率低，对排名促进有限，还可能拉低推广效率。

判断规则：

1. 高点击高转化词：优先保留和放大，推动排名和数据沉淀。
2. 高点击低转化词：检查词品匹配、国家、价格、MOQ 和页面承接。
3. 低点击高转化词：可能是潜力词，可考虑提升排名、加预算或补标题覆盖。
4. 低点击低转化词：谨慎投入，必要时屏蔽或降权处理。

## 全店访客有效性

访客不是越多越好，要看是否有效。访客详情中可重点看：

| 维度 | 问题 |
| --- | --- |
| 停留时长 | 大量 1-3 秒跳出，说明页面或流量可能无效。 |
| 访问类目偏好 | 买家关注的主类目是否与产品一致。 |
| 搜索词偏好 | 买家全网搜索偏好是否与产品相关。 |
| 国家和人群 | 是否属于目标国家和目标采购人群。 |

例如某批发商主营 A/B/C 类目，偶尔做 D 类目，而店铺产品属于 D 类目，那么这个访客可能不是最核心客户。诊断访客质量时要看买家的主营偏好，而不只是“他来过我的产品页”。

## 低价值产品处理

长期没有曝光、点击、访客、询盘或订单的数据极少产品，通常不适合精细诊断。可以按产品规则和店铺策略选择：

1. 下架或删除长期零效果产品。
2. 按新关键词、新类目或新款式重发。
3. 如果产品战略价值高，重新包装为新品进入测试池。
4. 把资源转给爆品、实力优品和新增潜力品。
