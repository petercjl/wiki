# 平台规则与搜索排序机制

CH10-CH12 进入平台规则和搜索排序模块。课程强调：不理解平台规则，运营只能凭感觉做流量；理解规则以后，才知道哪些动作能获得搜索机会，哪些动作会造成低效、违规或付费推广浪费。

这一页把三章内容压成一个判断框架：先看产品是否符合平台规则，再看搜索词能否匹配产品，最后看产品和店铺综合数据能否在排序中胜出。

## 搜索排名的三条原则

| 原则 | 课程含义 | 运营判断 |
| --- | --- | --- |
| 买家导向 | 搜索主要围绕买家需求展开，产品和店铺信息越能匹配买家搜索和偏好，越容易获得曝光。 | 标题、关键词、类目、属性、详情页、价格、物流和服务能力都要服务买家真实需求。 |
| 严惩作弊 | 侵权、禁限售、错放类目、虚假信息、虚假交易、重复铺货等会破坏买家体验和市场秩序。 | 不要用违规动作换短期曝光。违规会影响搜索机会、店铺权重和后续推广效率。 |
| 模糊匹配 | 系统会对大小写、单复数、同义词、词序等做一定匹配。 | 不能只靠精准堆词，要让产品文本和类目语义整体一致。 |

课程里的核心提醒是：只有符合平台规则的店铺和产品，才有更多机会获得曝光；否则运营动作会像盲人摸象。

## 搜索排序链路

```text
买家搜索词
-> 平台过滤违规/无效/低价值商品
-> 判断关键词、标题、类目、属性、详情页等文本相关性
-> 结合产品信息质量、供应商能力和买家偏好
-> 按综合数据排序
-> 展示给买家
```

这个链路解释了为什么“有关键词”不等于“有排名”。产品先要进入正确匹配池，再通过综合数据争取更靠前的位置。

## 影响搜索排序的核心维度

| 维度 | 课程要点 | 诊断问题 |
| --- | --- | --- |
| 类目相关性 | 产品要放在正确类目。卖鞋应进入鞋类目，卖袜子应进入袜子类目。 | 产品类目是否和买家搜索词、产品实物一致。 |
| 文本相关性 | 标题、关键词、属性、详情页要描述同一款产品。 | 标题是否含核心产品词，关键词是否与标题和属性一致。 |
| 产品信息质量 | 图片规范、文字专业、价格和物流真实，避免重复、冲突和虚假信息。 | 信息质量分、图片清晰度、属性完整度、价格/物流可信度是否达标。 |
| 供应商信息 | 店铺类型、星级、服务年限、回复能力、交易能力、评价等会影响排序。 | 店铺基础能力是否拖累产品排名。 |
| 买家偏好 | 买家的历史搜索、点击、询盘和订单行为会影响推荐和排序。 | 产品是否真的被目标买家点击、询盘和成交。 |
| 产品核心数据 | 点击、询盘、订单和转化等数据越好，权重和排序机会越强。 | 是否只看曝光，而没有看点击率、询盘率和订单转化。 |

## 搜索排序的运营含义

1. 先做正确性，再做排名。类目、文本和产品信息错了，后续推广会更贵、更不精准。
2. 先获得匹配机会，再争取靠前排序。被搜索系统识别和进入正确流量池，是排名的前置条件。
3. 排名不是单一关键词动作，而是产品层、店铺层、买家行为和平台资源共同作用。
4. 付费推广不是绕开规则，而是在规则理解清楚后放大有效产品和精准流量。

## Agent 使用提示

诊断国际站搜索排名时，先按以下顺序提问：

1. 产品是否有违规、重复铺货、错类目或虚假信息风险。
2. 买家搜索词和产品类目是否一致。
3. 标题、关键词、属性、详情页是否围绕同一个产品。
4. 产品信息质量、价格、物流和图片是否符合平台基础要求。
5. 店铺星级、回复率、交易能力和评价是否拖累排序。
6. 点击率、询盘率、订单转化是否证明买家真的喜欢这个产品。
