# 关键词词根与覆盖率

关键词是国际站流量的源头，也是产品和买家匹配的桥梁。买家通过关键词表达需求，平台通过关键词、标题、属性、详情页和行为数据判断产品是否匹配买家。词根则是拓展关键词覆盖率的支点：词根找得对、找得全，店铺才能覆盖更多买家搜索入口。

## 关键词是什么

关键词是产品或服务的具体名称，通常是买家用来表达需求的词。例如 T-shirt、phone、TV、air shipping、sea shipping 等，都可以是产品或服务关键词。

关键词承担三件事：

1. 让买家找到产品。
2. 让平台理解产品是什么。
3. 让标题、直通车和搜索匹配更精准。

如果关键词判断错误，就容易出现询盘不精准、直通车预算浪费、标题覆盖不全和产品匹配错误。

## 词根为什么重要

同一种产品可能有多个英文表达形式，它们互相独立，但都描述同类产品。课程举例：T-shirt、tshirt、t shirt 可能都表达 T 恤，但在关键词指数里拓展出来的相关词并不完全相同。

词根的价值在于：

- 一个词根可以拓展出一批相关关键词。
- 多个同义或不同写法的词根能覆盖更多关键词。
- 不同词根只有少部分重复，大部分相关词可能不同。
- 词根越准确，标题组合和产品覆盖越完整。

## 如何确定产品词根

```text
中文产品名
-> 翻译出多个英文形式
-> 到国际站前台搜索
-> 判断搜索结果是否与自己的产品一致
-> 到关键词搜索指数验证
-> 看能否拓展出相关关键词
-> 导出相关词，用于标题和产品覆盖
```

词根通常较短，可能是一个单词，也可能是两个单词。越短的准确词根，越容易拓展出更多相关词。但短不等于一定正确，必须用前台搜索和关键词指数验证。

## 前台搜索校验

一个词能不能作为词根，必须先看它在国际站前台搜索出来的产品是否与自己产品一致。

例如课程中提到“纸盒”一类词时，如果英文词搜出来的产品与自己要卖的纸盒不同，就不能直接作为词根。可以继续用组合词、属性词或更准确的产品词验证，直到搜索结果与产品高度匹配。

## 关键词指数拓展

在关键词搜索指数里输入词根，选择近 30 天，不限定终端时，可以看到大量相关关键词。操作时要：

- 判断相关词是否真的描述自己的产品。
- 区分单复数、空格、连字符、大小写等不同表达。
- 把不同词根拓展出的相关词分别导出。
- 去重后用于标题、关键词库、产品发布和推广。

## 关键词覆盖率检查表

| 检查项 | 判断 |
| --- | --- |
| 产品是否有多个英文表达 | 有多个表达就要分别验证。 |
| 每个词根是否经过前台搜索 | 未验证的词根不能直接用。 |
| 每个词根是否能拓展相关词 | 不能拓展或拓展不相关，应放弃。 |
| 相关词是否与主图、属性、详情页一致 | 不一致会导致匹配错误和低转化。 |
| 标题是否覆盖主要词根和长尾词 | 不要只用一个词根组所有标题。 |
| 词库是否能覆盖整个品类或主推产品 | 覆盖不足会限制店铺流量入口。 |

## 与后续模块的关系

CH39 是“询盘高转化系统”的入口，后续关键词、标题、产品、详情页、旺铺和选词都会依赖词根判断。词根工作做错，后续标题和直通车会越做越偏；词根工作做对，后续选词、标题组合和产品覆盖才有基础。
