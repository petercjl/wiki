# 营销活动与优惠券免费流量

营销活动不是可有可无的装饰，而是提高点击、停留、询盘转化和活动场景流量的重要入口。课程把活动中心、官方报名、折扣营销、优惠券和客户营销串成一个日常运营模块。

## 活动入口

课程中的主要入口包括：

| 入口 | 作用 |
| --- | --- |
| 活动中心/官方报名 | 报名平台活动、行业活动、直播活动和大促场景。 |
| 折扣营销 | 对 RTS 产品设置折扣，适合现货、引流款和活动款。 |
| 优惠券 | RTS 和定制产品都可设置，可在详情页、旺铺、直播和客户营销中使用。 |
| 客户营销/智能圈客 | 将优惠券放入营销卡片，提高客户回访和停留。 |

实操前应以当前后台入口和规则为准，课程中的入口名称和限制属于课程时点信息。

## 官方活动报名原则

只要符合条件的官方活动，都应定期查看和报名，但不是随便拿差产品报名。

报名前检查：

1. 活动要求和报名条件。
2. 报名成功后展示位置和流量权益。
3. 是店铺报名还是产品报名。
4. 报名产品是否为主推、有优势、数据较好、能承接买家。
5. 报名后是否通过审核，不要报完就不看。

数据差、承接弱、非主推产品即使报名成功，也很难获得有效流量。

## 折扣和优惠券的差异

| 工具 | 适用 | 重点 |
| --- | --- | --- |
| 折扣营销 | 主要适用于 RTS 产品。 | 常用于现货、引流款、活动款；通常需要提前设置，课程中提到可能次日生效。 |
| 优惠券 | RTS 和定制产品都可用。 | 可设置使用门槛、优惠上限、券数量、有效期和指定商品。 |
| 直播券 | 直播场景单独创建。 | 用于直播转化和互动承接。 |
| 全店券 | 覆盖全店商品。 | 要测算毛利，避免过度让利。 |

优惠信息会在产品、详情页、旺铺或营销卡片中展示，提升买家点击和停留概率。

## 优惠设置的毛利边界

活动和优惠必须先算毛利。正式执行时应通过合规方式控制亏损：

1. 调整优惠门槛。
2. 设置优惠上限。
3. 限定商品范围。
4. 选择毛利足够的引流款或活动款。
5. 用样品、组合购、阶梯报价或报价备注承接定制需求。

不得为了覆盖优惠成本而误填产品重量、尺寸、物流信息或其他会误导买家的参数。课程中涉及“用物流费用抵消优惠”的口播，应按合规毛利测算理解，不作为正式操作指令。

## 日常运营节奏

建议建立每周动作：

1. 查看活动中心新活动。
2. 检查已报名活动是否成功。
3. 更新折扣和优惠券有效期。
4. 给主推款、引流款、活动款设置合适优惠。
5. 在客户营销和访客营销中使用券。
6. 观察产品点击、停留、询盘、TM 和订单变化。
7. 对无效果优惠调整门槛、商品或投放场景。

## RTS 和定制店铺的应用

RTS 店铺要更重视折扣、优惠券和官方活动，因为现货产品更容易进入活动和榜单。

定制店铺也可保留少量常规 RTS 链接，专门用于活动、优惠、引流和场景流量承接；定制产品本身也可设置优惠券，用于详情页、旺铺和客户营销展示。
