# 店铺流量增长逻辑与战略

CH92 讲的是店铺流量为什么增长、为什么下跌，以及运营该从哪些因子入手。核心观点：店铺总流量不是一个孤立数字，它由店铺赛道和一组可优化因子共同决定。

## 总流量模型

课程把店铺总流量拆成两层：

1. 店铺层级的赛道选择：定制、RTS、半托管等入口决定流量天花板。
2. 运营层级的增长因子：品类、国家、关键词覆盖、关键词排名、排名位置和场景流量决定天花板内能跑到哪里。

可以简化为：

```text
店铺总流量 = 赛道入口天花板 x 运营因子完成度
```

当赛道固定时，增长主要来自运营因子提升；当运营因子已经接近上限时，可能需要新增赛道或品类入口。

## 六个核心增长因子

| 因子 | 增长含义 | 诊断问题 |
| --- | --- | --- |
| 品类数量和质量 | 主推品类越合理，流量入口越多。 | 当前品类是否太窄，还是铺得太散？ |
| 国家覆盖 | 目标国家越清晰、越多有效国家，流量池越大。 | 是否只做少数国家，或国家标签已经偏移？ |
| 关键词覆盖量 | 覆盖更多有效关键词，产品才有更多参与匹配的机会。 | 主推品类是否覆盖核心词、精准词和长尾词？ |
| 有排名关键词数量 | 有排名词越多，能获得曝光的入口越多。 | 覆盖词是否只是收集了，还是已有排名？ |
| 排名位置 | 同样有排名，第一页比第二、三页更容易拿点击。 | 重点词排名是否前移？ |
| 场景流量 | 爆品、榜单、新品、活动、推荐等带来额外免费流量。 | 是否有实力优品、爆品、活动和场景入场产品？ |

这些因子不是孤立提升，而是连锁反应。新增一个合适品类，可能同时增加国家、关键词池、有排名关键词和场景流量机会。

## 产品是源头

课程强调，分析流量增长或暴跌时，产品是源头。因为产品决定：

1. 进入哪个类目和品类。
2. 覆盖哪些关键词。
3. 匹配哪些国家和买家。
4. 能否形成点击、询盘、TM 和订单。
5. 是否成为实力优品、爆品、榜单品或活动品。

所以店铺流量下跌时，不要只看“流量来源少了”，还要从产品层往外剥：

```text
产品是否减少或失效
-> 覆盖关键词是否减少
-> 有排名关键词是否减少
-> 排名位置是否后移
-> 优品/爆品/场景资格是否减少
-> 店铺总流量是否下降
```

## 增长分析方法

当店铺 3-4 月效果好，5-6 月变差时，按以下顺序查：

1. 品类是否减少，主推品类是否失效。
2. 有排名关键词总量是否下降。
3. 关键词排名位置是否从第一页后移到第二、三页。
4. 准爆品、爆品、实力优品是否减少。
5. 榜单、新品、活动、猜你喜欢等场景流量是否下降。
6. 是否新增了不匹配产品，导致流量池变杂。

这套方法也适用于直通车流量和单品流量分析，只是诊断对象从店铺缩小到计划或产品。

## 赛道层增长

如果运营因子没有明显变化，但平台新增了半托管、RTS 或其他入口，店铺可以通过新增赛道打开新的流量天花板。

赛道层增长要看：

1. 公司是否能真实承接该赛道。
2. 产品是否符合赛道规则。
3. 价格、库存、物流、交付是否匹配。
4. 新赛道是否会影响原有定制或 RTS 标签。

新增赛道不是为了追热点，而是为了增加真实、可承接的流量入口。

## 做最赚钱的店铺战略

“最赚钱”不是单纯流量最大，而是流量、询盘质量、订单和毛利可持续。

判断顺序：

1. 先看哪些品类能带来有效流量。
2. 再看哪些国家和买家质量更好。
3. 再看哪些关键词能带来高质量询盘。
4. 最后看业务是否能成交、利润是否成立。

如果某个品类流量很大，但询盘质量差、业务不愿承接、订单利润低，它不是好战略。
