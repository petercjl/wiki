# 店铺流量结构与渠道增长分析

店铺流量增长不能只问“怎么多拿流量”，而要先知道平台从哪些端口把买家导入店铺，买家进入后去了哪里，为什么没有留下。流量结构分析的目标是找到店铺还有增长空间的渠道，并把前面学过的产品、关键词、橱窗、直通车、活动和内容方法接到对应渠道上。

## 平台流量结构

课程把平台流量大致拆成三类：自然流量、付费流量和推荐流量。比例会随平台和行业变化，不应当作固定规则；真正要学的是“流量入口地图”。

| 流量类型 | 常见入口 | 运营问题 |
| --- | --- | --- |
| 自然流量 | 搜索、榜单、导购会场、新品、汇采购、直播、短视频等。 | 是否有足够优品、爆品、关键词排名和场景入场能力。 |
| 付费流量 | 直通车搜索、直通车推荐、顶展、问鼎、年框、超级名片、明星展位等。 | 预算是否投给精准产品、精准词和精准国家。 |
| 推荐流量 | 平台首页、猜你喜欢、行业首页、产品详情页推荐、买家发布询盘列表推荐等。 | 店铺和产品标签是否精准，产品是否能承接推荐人群。 |

如果付费流量占比很高、自然流量占比很低，店铺需要想办法提高自然流量占比，降低获客成本。如果推荐流量转化高但量少，可以尝试通过产品标签、优品、内容和推荐推广放大。

## 流量参谋三大板块

后台流量分析可重点看三块：

| 板块 | 作用 | 关键问题 |
| --- | --- | --- |
| 流量来源 | 看店铺访客来自哪些渠道。 | 哪些渠道流量多，哪些没有流量，哪些转化高。 |
| 流量助手 | 看场景热品、入场规则和场景效果。 | 哪些产品可打造进入场景，哪些热品可参考选品。 |
| 流量承接及去向 | 看买家进入后由哪些产品承接，以及流失到哪里。 | 为什么留不住买家，同行哪些产品吸走了流量。 |

不要只看总访客。总访客是各渠道相加的结果，只有拆到渠道，才能知道该优化搜索、推荐、活动、直通车、内容还是产品承接。

## 流量来源分析

流量来源用于判断店铺当前流量构成和渠道短板。建议按周或按月看，日数据只用于细致排查短期波动。

分析顺序：

```text
查看全部渠道
-> 不隐藏低流量渠道
-> 对比访客量和商机转化率
-> 找高转化低流量渠道
-> 找高流量低转化渠道
-> 拆到产品、关键词、国家和推广计划
```

判断规则：

1. 高转化低流量渠道：优先放大。例如推荐渠道转化高但访客少，可以尝试推荐流量推广、产品标签优化和优品/爆品建设。
2. 高流量低转化渠道：优先诊断精准度。例如搜索流量访客多但询盘/TM 少，可能是关键词不精准、产品与买家不匹配、国家偏移、价格/MOQ 不匹配。
3. 无流量渠道：判断是否有操作空间。如果渠道本身可做，就查入场规则、产品条件和资源配置。
4. PC 端和无线端差异：如果访客主要来自某一端，产品详情页、旺铺装修和推广素材要适配该端买家体验。

## 从渠道拆到产品

渠道问题最终常常要拆到产品层。

```text
渠道数据异常
-> 定位贡献或下滑的产品
-> 查看产品关键词、排名、国家、价格、MOQ 和详情页
-> 查看是否来自直通车或自然搜索
-> 调整产品、词、国家和推广计划
-> 回看渠道数据是否恢复
```

例如搜索流量从 1 月到 3 月暴跌，应先找出哪些原本贡献搜索流量的爆款产品下滑，再看这些产品的关键词排名、曝光点击词、访客国家和转化是否变差。渠道不是独立存在的，渠道流量由产品数据和推广计划共同构成。

## 流量助手分析

流量助手用于发现平台场景、场景热品和入场规则。它的价值有两类：

1. 场景增长：知道某个场景需要什么产品条件，反向打造产品进入场景。
2. 选品参考：观察场景热品、行业优品和买家画像，挖掘可发布、可推广、可打造的产品。

使用时不要只截图看热闹，要问：

1. 这个场景现在有哪些热品。
2. 热品有什么共同款式、属性、价格、国家和关键词。
3. 入场规则是什么。
4. 我店铺哪些产品离规则最近。
5. 进入场景后如何承接询盘和订单。

## 流量承接及去向

流量承接及去向回答两个问题：买家进入店铺后由哪些产品承接，买家离开后去了哪些同行店铺或产品。

可分析的维度：

| 维度 | 用法 |
| --- | --- |
| 我的产品 | 找到承接流量的产品，判断是否需要优化主图、标题、价格、详情页和关联产品。 |
| 行业优品 | 参考平台认为优质的产品，用于选品和优化表达。 |
| 买家去向 | 查看买家离开后去了哪些同行和产品，判断差距。 |
| 地域偏好 | 看访客国家是否与目标国家一致。 |
| 类目偏好 | 看买家原本关注的类目是否与主营产品高度相关。 |
| 产品偏好 | 观察买家更喜欢的款式、属性和价格带。 |

如果买家进入后大量流向同行，要把同行产品搜索出来，对比主图、价格、MOQ、详情页、评价、认证、物流和产品款式。流失不是抽象问题，通常能在具体产品差距里找到线索。

## 渠道增长决策表

| 数据现象 | 诊断方向 | 动作 |
| --- | --- | --- |
| 付费占比过高 | 自然流量弱，产品和关键词沉淀不足。 | 提升优品、爆品、关键词排名、橱窗和内容。 |
| 搜索流量高但转化低 | 词品、国家、价格或详情页不匹配。 | 拆产品和关键词，修正标题、推广词和国家。 |
| 推荐流量转化高但量少 | 标签和推荐入口可放大。 | 优化产品标签，测试推荐推广和场景入场。 |
| 活动/场景无流量 | 未满足入场规则或产品不适配。 | 根据规则打造产品，增加优品和爆品基础。 |
| 无线端流量高但转化低 | APP 页面和详情页承接弱。 | 检查移动端首图、价格、详情页节奏和询盘按钮。 |
| 买家流向同行 | 产品、信任或价格竞争力不足。 | 对比同行产品，优化承接页和关联产品。 |
