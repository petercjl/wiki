# 商品力与营销力提升方法

商品力和营销力是店铺星等级里的两个关键能力分。商品力偏产品池质量，营销力偏推广产品带来的流量、商机和回复效率。课程的原则很直接：商品力要把优品和橱窗优品做足，营销力要舍弃差产品，让好产品获得更多流量。

## 商品力指标

商品力主要看：

1. 全店优品数。
2. 橱窗优品占比。
3. RTS 店铺还要关注可交易商品数量。

全店优品数的提升方法，承接前面的优品、实力优品和爆品方法。橱窗优品占比则更直接：尽量把橱窗绑定到优品、实力优品、准爆品或爆品上，而不是普通低质量产品。

## 商品力提升方法

商品力诊断要和行业对比一起看：

```text
查看全店优品数
-> 对比行业平均和行业优秀
-> 查看近 30 天趋势
-> 找优品数量和橱窗优品占比短板
-> 提升产品质量、发品质量、优品和爆品梯队
-> 用橱窗绑定优品提高占比
```

课程给出一个底线：关键指标不要低于行业平均。若低于行业平均，店铺整体效果通常较差；目标应是高于行业平均，并逐步接近行业优秀。

对 RTS 店铺，可交易商品数量要结合店铺定位和真实产品能力。如果主做定制，可交易商品数可以少一些；如果主做快速交易，则可交易产品数量应接近或超过同行水平。

## 营销力指标

营销力中的指标包括：

| 指标 | 含义 |
| --- | --- |
| 营销流量指数 | 由访客/点击量和买家质量折算，不等同于真实访客数。 |
| 商机指数 | 由询盘、TM、订单等商机动作折算。 |
| 商机转化率 | 可粗略理解为询盘 + TM + 订单除以访客，但后台指数还会考虑买家质量。 |
| 平均回复时间 | 买家咨询后的响应速度。 |
| RFQ 营销能力或极速回复率 | 随定制/RTS 赛道不同而变化的加分指标。 |

因为很多指标是指数，不要简单用“商机指数 / 营销流量指数”当作真实转化率。后台会折算买家质量和其他维度。

## 营销力统计范围

课程特别提醒：营销力统计的是营销产品和推广产品的数据，主要来自直通车、顶展、问鼎、访客营销、EDM、RFQ 等资源或营销动作。自然流量中的普通产品表现，不一定全部纳入这个板块。

这意味着提升营销力的关键是“推什么产品”。如果把高消耗、无询盘、无 TM、无订单的产品持续放在推广里，会拉低营销力。

## 营销力提升方法

核心原则：差产品果断舍弃，让好产品更好。

具体动作：

1. 推优品，尤其优先推广爆品、准爆品和实力优品。
2. 每周优化直通车、顶展、问鼎等资源产品，提高点击率和商机转化率。
3. 每日做访客营销、EDM、RFQ 等主动触达。
4. 上班期间尽量 5 分钟内回复买家。
5. 夜间设置智能接待，降低平均回复时间。
6. 对高曝光、高消耗、无商机产品及时换品或优化计划。

营销力提升最快的切入口通常是直通车，因为它能较快改变推广产品的流量结构。但直通车只能放大产品承接能力，不能拯救词品错配和页面转化差的产品。

## 高消耗无商机避坑

如果某产品持续高点击、高消耗，但没有询盘、TM 或订单，应立即诊断：

- 关键词是否不匹配产品。
- 目标国家买家是否偏好该款式。
- 主图和价格是否吸引错误买家。
- 详情页是否不能解释卖点和信任。
- MOQ、样品、物流或交期是否卡住转化。
- 业务回复是否慢或话术承接差。

无法短期修复时，应换产品。否则访客增加而商机不增加，商机转化率会被持续拉低。

## 回复时间管理

平均回复时间是营销力和买家体验的重要指标。执行上可拆成：

```text
工作时间：5 分钟内响应
夜间：智能接待先承接
重点买家：业务及时二次跟进
RFQ 和访客营销：固定每日处理
```

回复速度不是只为了指标，更是为了减少买家流失。尤其 RTS 小客户和快速交易买家，对响应速度更敏感。
