# 四象限产品优化 SOP

四象限优化法用曝光和点击先判断“买家是否愿意进来”，再用点击/访客和询盘/TM 判断“买家进来后是否愿意沟通”。它的价值不是画图，而是避免所有产品都用同一种优化动作。

## 第一张图：曝光 x 点击

横轴看点击量，纵轴看曝光量。

| 象限 | 数据现象 | 主要问题 | 优先动作 |
| --- | --- | --- | --- |
| 高曝光低点击 | 看得到但不点 | 词品、位置、主图、价格、MOQ、评价、运费可能有问题。 | 重点诊断点击率。 |
| 高曝光高点击 | 看得到也愿意点 | 前端吸引力基本成立。 | 不乱改，转向询盘/TM 转化。 |
| 低曝光低点击 | 看不到也没人点 | 产品缺流量或产品本身弱。 | 新品可改标题，老品多半放弃或大改。 |
| 低曝光高点击 | 量少但买家喜欢 | 产品有承接潜力，但流量不足。 | 加资源、加推广、加橱窗或活动。 |

阈值不能固定套用。课程举例可用 10 次点击作为基础门槛，但冷门行业可降低到 8-10，热行业可提高到 15、20、30。关键是让待优化产品数量可控。

## 高曝光低点击 SOP

高曝光低点击是点击率优化的重点。不要一上来换主图，应先排除流量来源和排名位置问题。

```text
高曝光低点击
-> 查曝光来源
-> 验证关键词是否准确
-> 查关键词排名位置
-> 查价格、MOQ、评价、运费
-> 判断是否来自非关键词曝光
-> 最后测试主图
```

检查要点：

1. 如果曝光主要来自关键词，前台搜索关键词，并用 Google Images 等外部结果辅助判断买家想看的款式。
2. 如果关键词准确，再查排名位置。排名在底部或第二/第三页，低点击率可能是位置问题。
3. 如果排名和词品都正常，再查价格、MOQ、物流费和评价分是否明显劣于同屏竞品。
4. 如果产品总曝光远高于关键词曝光，剩余曝光可能来自类目、图片搜索或推荐等非关键词来源，短期波动不一定值得大改。
5. 只有前面都排除后，才测试主图。

## 高曝光高点击 SOP

这类产品已经证明前端吸引力较好，不要为了优化而优化。重点转向询盘和订单。

动作：

1. 保持主图、标题和价格等关键资产稳定。
2. 检查询盘/TM 转化率和订单转化率。
3. 如果转化正常，优先加资源放大。
4. 如果转化低，进入第二张图的高点击低询盘诊断。

课程中提到点击率 0.8%-4% 可作为参考范围，但行业差异很大，应以周数据和行业对比为准。

## 低曝光低点击 SOP

低曝光低点击通常不值得精细优化，除非它是刚发布的新产品或公司战略产品。

判断边界：

| 产品状态 | 建议动作 |
| --- | --- |
| 发布 30 天内 | 可用近期高增长关键词重组标题，观察数据。 |
| 发布 60 天以上且无数据 | 多数情况下不继续优化，改为大改、重发或放弃。 |
| 公司重点产品 | 可重新做标题、主图、详情页和推广测试。 |
| 普通长尾产品 | 保留为覆盖或下架替换。 |

如果改标题，要注意后续推广和资源承接，否则标题变化可能带来短期流量波动。

## 低曝光高点击 SOP

低曝光高点击说明产品被看到的人少，但看到的人愿意点。它的核心是加流量，而不是先改产品。

可能原因：

1. 标题覆盖关键词少。
2. 关键词搜索指数低。
3. 关键词排名低。
4. 没有推广、橱窗、活动或其他资源支持。

动作：

1. 如果未推广且访客国家、关键词和商机质量都不错，可启动直通车。
2. 如果已推广，把预算向该产品倾斜，增加相关关键词并提高精准词出价。
3. 减少高消耗低转化产品对预算的占用。
4. 加橱窗、主营认证、问鼎、顶展等资源。
5. 用 EDM、访客营销、直播绑定等方式补充流量。

## 第二张图：点击/访客 x 询盘/TM

当产品已经有一定点击和访客后，重点看询盘转化率或商机转化率。

| 象限 | 数据现象 | 主要动作 |
| --- | --- | --- |
| 低点击低询盘 | 无流量也无转化 | 新品可重做标题，老品多半放弃或大改。 |
| 低点击高询盘 | 量少但转化好 | 加推广、橱窗、EDM、访客营销和 RFQ。 |
| 高点击高询盘 | 流量和转化都好 | 不乱改，观察并放大。 |
| 高点击低询盘 | 有流量无商机 | 重点诊断国家、关键词、价格、MOQ、详情页和人群。 |

商机转化率大于等于 6% 可作为课程参考值，但必须结合行业和店铺阶段判断。

## 高点击低询盘 SOP

高点击低询盘是最需要优先处理的产品，因为它可能持续消耗推广预算。

```text
高点击低询盘
-> 定义高点击门槛
-> 查访客国家精准度
-> 查点击关键词精准度
-> 区分推广词和自然词
-> 查价格/MOQ/详情页/信任
-> 调整直通车人群、国家和关键词
-> 必要时把产品作为导流入口
```

具体检查：

1. 先设门槛，例如周点击大于 30 或日点击大于 10；冷门行业可下调。
2. 查访客国家是否匹配目标市场。如果目标是美国但访客来自非目标国家，应从推广国家和产品标签修正。
3. 用产品 360 区分自然词和推广词。如果推广词不精准，优先改直通车；如果自然词不精准，再谨慎动标题。
4. 查价格、MOQ、物流费和详情页是否劝退买家。
5. 增加优惠、折扣、公司实力、认证和信任模块。
6. 如果产品本身不适合继续优化，但流量仍有价值，可在详情页关联爆品、导流到店铺首页或其他产品。

## 三步实操流程

课程实战案例可整理成三步：

1. 筛选产品：在产品分析/产品参谋中按周或按月查看数据，用四象限筛出可优化产品。周优化可放在周二，月度优化可在 3-4 日后看完整上月数据。
2. 勾选核心指标：至少关注曝光、点击、访客、询盘、TM、点击率和商机转化率。字段太多时先保留诊断链路核心项。
3. 进入产品 360：逐个查看关键词、推广来源、访客国家、询盘/TM、价格、竞品、MOQ 和页面，再决定优化动作。

产品优化要和直通车、橱窗、活动和内容资源联动。只筛产品不调资源，或只投推广不看产品承接，都会让优化失效。
