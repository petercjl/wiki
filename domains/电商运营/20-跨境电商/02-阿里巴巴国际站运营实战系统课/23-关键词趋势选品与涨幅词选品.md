# 关键词趋势选品与涨幅词选品

关键词趋势选品的核心是：用买家搜索变化反推产品机会。课程给出两条路径，一条从核心产品词进入关键词指数，寻找近期增长的属性/款式/风格词；另一条从选词参谋的热词榜、趋势榜直接发现涨幅词，再匹配已验证的热销款式。

## 路径一：核心产品词挖掘趋势款式

```text
确定核心产品词
-> 选词参谋/关键词搜索指数
-> 近 30 天查看搜索指数和波动
-> 找近 12 个月稳定且近期明显上涨的属性/款式/风格词
-> 到国际站前台搜索匹配款式
-> 到 Amazon/Google 复核外部需求
-> 选 2-5 款，组 3-5 个标题
-> 新品加速计划测试
-> 按数据调整标题和款式
```

核心产品词通常来自三级类目或明确产品名称，例如卫衣、T 恤等。课程里 ASR 多次把“类目”识别为“内幕”，正式操作应按“类目”理解。

## 筛选词的标准

| 标准 | 说明 |
| --- | --- |
| 近期上涨 | 最近几个月搜索指数明显上升。 |
| 长周期稳定 | 近 12 个月不是纯偶发尖峰，而是有需求基础。 |
| 竞争较小 | 卖家使用较少，或属于蓝海词。 |
| 具备产品含义 | 词里有属性、款式、风格、材质或功能指向。 |
| 能找到对应款式 | 国际站前台、Amazon、Google 能看到匹配产品。 |

如果搜索指数下降到很低，课程建议可停止继续挖。冷门行业阈值可以放低，但必须保持“词能指向真实产品需求”。

## 款式测试规则

不要只选一个款式。课程建议一次拿 2-3 款或 3-5 款测试，并为这些款式组合标题。每个款可以一个标题，也可以多个标题，但要进入新品加速或推广计划，用数据判断：

- 哪个词带来曝光和点击。
- 哪个款式有询盘或 TM。
- 标题是否需要替换增长词。
- 款式是否需要更换主图、详情页或直接放弃。

## 路径二：涨幅关键词直接挖掘热销品类和款式

涨幅词选品不一定从一个产品词搜索开始，也可以直接看选词参谋里的热词榜和趋势榜。课程建议每周查看一次，尤其关注最近上涨的新热搜词和细分类目词。

操作逻辑是：

```text
选词参谋
-> 热词榜/趋势榜
-> 按类目或细分类目查看周度上涨词
-> 判断词匹配的产品、款式和场景
-> 找已被市场验证的热销款式
-> 用上涨蓝海词重新组合标题
-> 形成与同行差异化的推广入口
```

这条路径的重点不是发没有验证过的新款，而是借用已知热销款式，再用新上涨词覆盖不同买家人群。课程强调，这种方式成功率高，是因为款式已被市场验证，差异化来自关键词和人群覆盖。

## 标题与推广注意点

1. 标题要围绕真实产品和增长词，不要堆不相关词。
2. 上涨词要能和主图、属性、详情页匹配。
3. 新品加速或直通车只是测试工具，不能替代选品判断。
4. 有数据后要快速调整标题和款式，不要让无效计划长期消耗预算。
5. 平台后台榜单、热词、趋势词口径会更新，实操前应以当前后台为准。
