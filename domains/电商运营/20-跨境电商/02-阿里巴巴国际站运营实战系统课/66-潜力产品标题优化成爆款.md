# 潜力产品标题优化成爆款

潜力产品不是没数据的新品，而是已经有一定点击、询盘或转化信号，但因为标题覆盖词少或词流量太小，导致曝光和点击不足的产品。

## 适用前提

适合优化的潜力产品通常具备：

1. 款式不错。
2. 有一定数据沉淀。
3. 标题覆盖关键词少。
4. 曝光和点击偏少。
5. 已有转化、询盘、TM 或订单信号。
6. 当前标题部分词流量指数太小。

不要把这个方法直接套到已经稳定出单的爆品上。效果很好的产品标题不要轻易大改。

## 诊断入口

先进入产品分析 360 或同类产品诊断入口，看关键词分析。

时间维度：

1. 按周看，至少看连续 1-3 周。
2. 按月看，至少看连续 1-3 个月。
3. 不只看单一周期，避免被短期波动误导。

重点找：

1. 有询盘的词。
2. 有 TM 的词。
3. 有订单的词。
4. 有点击且精准的词。
5. 曝光量较好且与产品匹配的词。

这些是有效词，应尽量保留。

## 找无效词

把标题拆成 A/B/C/D/E/F/G 等词组，逐个判断。

无效词特征：

1. 无曝光、无点击、无询盘、无订单。
2. 搜索指数很低或为 0。
3. 与产品款式不匹配。
4. 只占字符但不能带来有效流量。
5. 多周期都没有贡献。

无效词不是凭感觉删，而是用周、月数据反复确认。

## 替换词标准

替换和增补的新词要满足五个条件：

1. 有搜索指数。
2. 有点击率或点击数据。
3. 最近搜索指数上涨。
4. 是精准产品词或精准属性词。
5. 与当前产品款式高度匹配。

只替换无效词，尽量保留原始有效词及其顺序。

## 修改后的动作

改标题会影响产品链接数据和系统匹配，因此优化后要马上推广或给资源，避免权重断档。

建议流程：

```text
保留有效词 -> 替换无效词 -> 维持原词序 -> 发布修改 -> 立即推广稳权重 -> 7-14天复盘
```

观察指标：

1. 曝光是否提升。
2. 点击是否提升。
3. 新词是否开始有排名或点击。
4. 原有效词排名是否稳定。
5. 询盘/TM/订单是否受影响。

## 三个关键注意

1. 优质爆品不要轻易改标题。
2. 原始有效词顺序尽量保持，尤其是已验证的词组顺序。
3. 非改不可时，改完立刻推广，让产品权重和有效词排名尽量稳定。
