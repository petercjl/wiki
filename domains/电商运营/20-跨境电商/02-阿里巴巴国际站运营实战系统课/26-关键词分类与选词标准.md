# 关键词分类与选词标准

关键词工作不是把所有相关词都塞进标题和直通车，而是先识别词的层级、流量、竞争和买家意图，再决定它该用于标题覆盖、产品发布、直通车测试还是屏蔽排除。

## 四类关键词

| 类型 | 典型含义 | 运营价值 | 风险 |
| --- | --- | --- | --- |
| 行业词 | 很宽泛的一二级类目词，如 clothes、phone、bag。 | 可观察行业大盘和上层流量入口。 | 点击贵、转化低，早期直接推广容易浪费预算。 |
| 核心词 | 具体产品词根，如 T-shirt、hoodie、Android phone。 | 用来拓展词库、组织标题、建立产品覆盖。 | 如果只停留在核心词，买家意图仍不够清晰。 |
| 精准词 | 属性词加核心词，能表达较明确需求，如 black cotton men T-shirt。 | 直通车和标题优先使用，通常询盘转化更好。 | 需要与产品主图、属性、详情页真实匹配。 |
| 长尾词 | 多个属性、场景、人群或国家需求叠加的长关键词。 | 意图最细，转化可能高，适合补充标题和低成本测试。 | 单个词流量小，不能只依赖长尾词撑起规模。 |

课程强调，四类词的边界会因行业而变化。不要把时间耗在“这个词到底算精准词还是长尾词”的争论上，重点是判断它在当前行业里承担什么作用。

## 推广优先级

直通车投放应从精准词开始，再逐步加入核心词和长尾词。行业词和过宽泛词要谨慎，尤其在店铺和产品权重还没有形成时，宽词容易带来大量不转化点击。

一个简单判断：

```text
精准词可承接
-> 核心词可测试
-> 长尾词补覆盖
-> 行业词先观察，必要时屏蔽或低预算验证
```

如果店铺出现“点击不少、询盘很少”，先看直通车是否投了过宽行业词，或精准词与产品、国家、人群不匹配。

## 五个选词维度

### 搜索指数

搜索指数越大，代表买家搜索需求越明显。选词时先按搜索指数降序查看，但不能只选指数最高的词，因为高搜索词也可能竞争大、意图泛。

### 点击率

点击率代表买家看到后是否愿意点。课程提醒，点击率为 0 的词要特别谨慎，尤其不适合直接放进推广计划。高点击无询盘的词，也要拿回关键词指数里检查近 7 天或近 30 天数据。

### 卖家规模指数

卖家规模指数越小，竞争相对越轻。理想词通常是搜索指数相对高、点击率不差、卖家规模指数相对低。

### 时间窗口

关键词指数建议优先看近 30 天，至少也要看近 7 天。近 30 天更适合判断稳定需求，近 7 天适合发现近期变化。

### 国家偏好

如果店铺目标是美国，就要检查这个词在美国是否有偏好和搜索表现。目标国家和关键词偏好不一致时，容易出现“词看起来好，但目标国家买家不买单”的情况。

## 冷门行业扩词

冷门行业或小类目里，一个词根可能拓展不出足够词量。处理方法是把相关词再作为新的词根继续搜索，逐层扩展可用词库。

```text
核心词 A
-> 导出相关词
-> 选择仍然相关的词 B/C/D
-> 分别作为新词根再搜索
-> 汇总、去重、按类型和国家筛选
```

这一步不要只靠感觉判断，要回到关键词指数、前台搜索和产品匹配结果。

## 选词检查表

| 检查项 | 判断 |
| --- | --- |
| 词属于行业词、核心词、精准词还是长尾词 | 决定标题覆盖和投放优先级。 |
| 搜索指数是否足够 | 太低可能没有规模，太高要看竞争。 |
| 点击率是否为 0 | 0 点击率词不宜直接推广。 |
| 卖家规模指数是否过高 | 竞争过强时要找更细分词。 |
| 是否符合目标国家偏好 | 国家不匹配会降低询盘和订单转化。 |
| 是否与产品主图、属性、详情页一致 | 不一致会造成点击浪费和低转化。 |
| 是否已放入词库并去重分组 | 词库混乱会影响标题、发品和推广。 |
