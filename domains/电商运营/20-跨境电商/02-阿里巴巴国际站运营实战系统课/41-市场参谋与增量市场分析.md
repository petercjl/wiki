# 市场参谋与增量市场分析

市场分析的目标不是做很复杂的表，而是用足够的数据看清趋势、机会和动作。对阿里巴巴国际站店铺来说，课程建议优先使用国际站后台和市场参谋数据，因为不同平台的买家群体、交易方式和询盘行为并不完全相同；Google、Amazon 等站外数据可以补充趋势，但不能替代站内买家行为。

市场分析最终要落到运营决策：选什么类目、推什么产品、主攻哪些国家、用哪些关键词、怎样配置直通车和产品资源。

## 两条分析入口

市场参谋可按两条主线使用：

| 入口 | 适用问题 | 输出 |
| --- | --- | --- |
| 查行业 | 判断行业、类目和品类机会。 | 行业流量大小、发展趋势、热门品类、供需竞争、买家国家、产品和关键词机会。 |
| 查国家/地区 | 判断某个国家是否值得重点投入。 | 国家买家规模、采购金额层级、搜索偏好、产品偏好、类目偏好和终端偏好。 |

查行业解决“这个行业/类目有没有机会”；查国家解决“这个国家对这个类目有没有机会”。不要把两个问题混在一起，否则容易只看到一个国家热，实际品类不匹配，或只看到类目热，却没有明确目标国家。

## 判断市场潜力

判断行业或类目是否有潜力，至少看四类指标：

| 指标 | 课程含义 | 诊断问题 |
| --- | --- | --- |
| 买家规模指数 | 市场容量和买家规模。 | 这个类目是否有足够买家。 |
| 买家数年同比 | 市场增长趋势。 | 市场是在增长、停滞还是下滑。 |
| 供需比 | 买家需求和卖家供给的相对关系。课程语境中，供需比越高通常表示竞争相对更低。 | 这个类目是否过度拥挤。 |
| 转化率 | 买家是否真的产生询盘、TM 或订单。 | 流量是否有真实需求，而不是只有浏览。 |

实操时不要只看一个指标。买家规模大但年同比下降，可能是成熟或下滑市场；年同比高但买家规模小，可能只是小盘波动；流量高但转化低，可能只是泛需求或产品不匹配。建议把一级、二级、三级类目指标导出或录入表格，按转化率、买家规模、供需比和增长趋势多维排序。

## 类目综合分析

选定行业或类目后，需要进一步拆成产品、关键词、买家和卖家四个维度。

### 产品和关键词

商机分析里的产品榜单可用于发现产品款式和属性方向：

1. 人气榜：看当前关注度高的产品。
2. 飙升榜：看短期增长快的产品。
3. 蓝海榜：看供给相对不足、竞争可能较低的机会。
4. 效果榜：看更接近询盘、交易效果的产品。

关键词榜单回答买家用什么词搜索。关键词不要只用于标题，还要回写到直通车词包、产品属性、详情页表达和国家偏好判断。

### 买家分析

买家分析重点看三类交集：

| 数据 | 用法 |
| --- | --- |
| 询盘买家国家 | 判断哪些国家有主动咨询需求。 |
| 交易买家国家 | 判断哪些国家更容易成交。 |
| TM 买家国家 | 判断哪些国家有即时沟通意愿。 |

目标国家优先从询盘买家、交易买家和 TM 买家的交集中选择。课程也提到 MOQ 偏好、样品偏好、定制偏好和店铺偏好等字段，但这些偏好数据可能比较粗，应作为辅助，不要替代国家、类目、产品和关键词的主判断。

### 卖家分析

卖家分析用于判断竞争强度和目标现实性：

| 观察项 | 运营含义 |
| --- | --- |
| 星级分布 | 如果少数 3-5 星卖家拿走大部分流量，新店需要先评估星级和商品力差距。 |
| 年询盘量分层 | 年询盘量除以 12 可估算月度对标空间，用来判断目标是否合理。 |
| 年线上 GMV 分层 | 判断线上交易天花板。若行业里极少店铺做到某交易额，目标要结合线下交易和品类现实校准。 |
| 同行发品类目 | 看优秀卖家集中布局哪些三级类目和产品线。 |

## 国家市场机会分析

查国家时要固定类目、国家和时间窗口。否则看到的是泛国家数据，不能直接指导某个产品线。

国家分析建议按以下顺序：

```text
固定类目和国家
-> 看买家规模和国家排名
-> 看采购金额层级
-> 看搜索偏好词
-> 看产品偏好
-> 看类目偏好
-> 回写产品、关键词和推广国家
```

判断国家是否值得重点投入，可看：

1. 买家占比和排名：占比高说明该国家对该类目有更强采购容量。
2. 采购金额分布：如果绝大多数买家采购额很低，要警惕询盘质量和客单价不匹配。
3. 搜索偏好：热度词和涨幅词可用于标题、关键词和直通车。
4. 产品偏好：国家偏好的款式、属性、价格和 MOQ 是否与店铺产品匹配。
5. 类目偏好：访问偏好代表流量，询盘/TM/订单偏好代表需求强度，应优先找两者交集。

## 从数据到动作

| 数据发现 | 动作 |
| --- | --- |
| 某三级类目买家规模大、转化率高、供需比好 | 作为产品线或推广重点，补产品、补关键词、补橱窗和直通车资源。 |
| 某国家买家占比高且采购金额层级更优 | 加入目标国家池，单独看关键词偏好和产品偏好。 |
| 某国家搜索词上涨但产品偏好不匹配 | 不直接投放，先找匹配款式或新发产品测试。 |
| 访问偏好类目高但询盘偏好低 | 说明有浏览但转化弱，需要检查产品、价格、MOQ 和承接页。 |
| 同行集中布局某类目且年询盘分层高 | 评估是否进入同类目，或寻找差异化款式和国家切口。 |
