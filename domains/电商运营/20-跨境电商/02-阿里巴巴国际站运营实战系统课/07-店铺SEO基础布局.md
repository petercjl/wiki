# 店铺 SEO 基础布局

CH09 把国际站店铺 SEO 定义为：利用搜索引擎规则提升网站或店铺页面的自然排名，让产品有更多展示机会，吸引更多买家进店，进而促成询盘和订单。

这页的重点是“基础但容易被忽略的内部设置”。国际站店铺不是单个网页，而是由店铺首页、产品分组、栏目、详情页、产品链接、公司信息和认证标签共同构成的页面集合。搜索引擎和平台系统会抓取这些内容，用来判断店铺做什么、偏好哪些品类、匹配哪些买家和需求。

## SEO 布局的底层逻辑

```text
内容表达清楚 -> 系统识别店铺和产品 -> 买家搜索时精准匹配 -> 自然排名和流量机会提升 -> 询盘质量改善
```

如果标题、关键词、属性、详情页、类目、公司信息和实力标签彼此不一致，系统就很难判断你到底卖什么，买家搜索时也更难精准匹配。

## 产品层 SEO

| 模块 | 要求 | 错误表现 | 目的 |
| --- | --- | --- | --- |
| 主题明确 | 标题、属性、详情页都围绕同一类产品。 | 同一个详情页里 A 信息描述 C 产品，又混入 B 产品。 | 让系统和买家知道这条链接卖什么。 |
| 标题核心词 | 标题必须包含产品核心词。 | 卖围裙但标题没有围裙类核心词。 | 确保系统能识别产品主类。 |
| 长尾词 | 设置竞争度更小、更精准的长尾词。 | 只堆大词，无法获得初始精准流量。 | 用长尾词带动链接流量和匹配质量。 |
| 关键词相关性 | K1/K2/K3 等关键词要和标题、产品、详情页描述同一款产品。 | 关键词和标题、类目、属性不相关。 | 提高系统相关性判断。 |
| 属性与类目 | 产品属性、类目和标题匹配，属性完整准确。 | 属性缺失、类目错放、标题与类目不一致。 | 进入正确搜索、类目、榜单和场景。 |
| 详情页文字 | 详情页最好有文字描述，不只是图片。 | 只有图片，搜索引擎难抓取内容。 | 让搜索引擎抓取更多可识别关键词和语义。 |

## 店铺层 SEO

店铺层 SEO 关注的是整个店铺页面集合，而不是单条产品链接。

| 模块 | 课程要点 | 操作含义 |
| --- | --- | --- |
| 店铺层级结构 | 产品分组、首页栏目、详情页栏目要清晰。 | 买家阅读便利，搜索和平台抓取也更顺。 |
| 页面布局 | 店铺中几百到上千个产品链接共同构成店铺网站。 | 每个产品页面都要尽量清楚、完整、相关，而不是只优化少数爆款。 |
| 店铺更新 | 经常发布新品或优化内容，保证店铺活跃。 | 搜索引擎/平台会定期抓取店铺信息，长期不更新可能影响活跃判断和流量分配。 |
| 公司信息 | 管理公司信息、核心信息展示等要完善。 | 让系统识别商家主营方向、品类偏好、人群和需求。 |
| 认证与实力标签 | 证书、实力标签、行业竞争力等要上传和展示。 | 提升店铺信任信号，并在旺铺首页或详情页展示优势。 |
| 产品管理标签 | 产品管理、管理标签、公司标签等要设置。 | 帮助系统更准确理解产品和店铺结构。 |
| 商机沟通/订阅等基础资料 | 课程提到相关后台资料也要完善。 | 提升基础运营完整度，具体入口以当前后台为准。 |

## SEO 基础布局检查表

1. 每条产品链接是否只描述同一类产品。
2. 标题是否包含核心产品词。
3. 是否有长尾词来承接低竞争精准流量。
4. 关键词是否与标题、属性、类目和详情页一致。
5. 产品分组和店铺栏目是否清晰。
6. 详情页是否有可被搜索引擎抓取的文字信息。
7. 店铺是否每周保持新品发布或内容优化。
8. 公司信息、认证中心、实力标签、行业竞争力是否完善。
9. 店铺表达是否让系统知道你做什么、偏好哪些品类、服务哪些需求。
10. SEO 动作是否最终改善精准询盘，而不是只追求泛流量。

## 和首页流量入口的关系

SEO 不是孤立动作。CH08 讲的首页、类目、榜单、新品、汇采购和制造商入口，都依赖系统对产品和店铺的识别。SEO 基础越清晰，平台越容易把产品放进正确搜索、类目、推荐、榜单或场景中。

因此，店铺 SEO 的目的不是堆关键词，而是让平台和买家形成同一个判断：这个店铺的供给清楚、内容完整、产品相关、能力可信。

## Agent 使用提示

诊断国际站店铺 SEO 时，先要求用户提供：

- 目标产品和核心关键词。
- 产品标题、关键词、类目、属性和详情页。
- 店铺产品分组、首页栏目和公司信息截图。
- 最近发布新品和优化频率。
- 询盘质量反馈：买家是否精准，是否经常问不相关需求。

然后按“产品层相关性 -> 店铺层结构 -> 更新活跃度 -> 信任标签 -> 询盘质量”顺序诊断。

## 使用边界

课程中的 SEO 原则可作为基础判断框架，但国际站搜索排序、抓取机制、标签入口和后台名称会变化。正式执行前应以当前阿里巴巴国际站后台、前台搜索结果和官方规则为准。
