# 询盘量低于行业均值突破与店铺盘活

询盘量长期低于行业均值时，不适合全店平均发力。课程给出的思路是单点突破：先选 1-3 个最有机会的品类，集中产品、标题、推广和订单反馈，把店铺流量层级拉起来，再扩展其他品类。

适用场景：

1. 新店上线 1-3 个月内想快速起量。
2. 老店长期半死不活，需要重新盘活，也适用于“老店铺盘活”类问题。
3. 店铺品类很多但资源有限，不知道先做什么。
4. 直通车预算分散，推广后没有明显增长。

## 第一阶段：选品类和款式

不要平均推所有品类。优先选择符合更多条件的品类。

| 选择维度 | 判断方法 |
| --- | --- |
| 近 12 个月流量稳定 | 行业和店铺历史数据中有持续搜索和点击。 |
| 关键词池足够大 | 可覆盖的核心词、精准词、长尾词数量多。 |
| 款式被市场验证 | 接近行业热销款、榜单款或同类偏好人群款。 |
| 价格/MOQ 有优势 | 公司实际可承接，目标买家能接受。 |
| 老店可重新发布 | 老链接数据差时，不强行改旧标题，优先新发产品。 |

交集越多，越适合作为突破品类。老店盘活时，课程建议把部分老产品当作历史包袱处理：数据差、标签乱、长期无效的链接，不要无限改标题，可重新选品、重发、重推。

## 第二阶段：精细化标题和链接规模

单一品类要一次性做够一组链接，而不是零散发布。

操作原则：

1. 先按关键词课程方法挑词，选择对前期流量获取更有用的词。
2. 标题按爆款标题结构组合，保证核心词、精准词、属性词和长尾词覆盖。
3. 单一品类一次性准备 20 个链接，资源允许可扩展到 50 个以内。
4. 不优先选择老产品改标题，老店盘活更适合重新发布。
5. 让一批同品类产品同时进入新品加速，便于系统识别方向和筛选潜力链接。

链接数量不是为了铺货，而是为了让同一品类形成足够测试样本。

## 第三阶段：直通车推广和真实转化信号

新发产品应尽早进入新品加速计划。课程建议新店上线 7 天内就开始推广，预算不一定大，但不能完全不推。

参考节奏：

| 动作 | 建议 |
| --- | --- |
| 新品加速 | 同品类 20-50 个链接集中测试。 |
| 日预算 | 课程示例为 100-200；预算不足时，少量预算也比完全不测强。 |
| 两周筛选 | 从测试链接中找到约 10 个潜力链接。 |
| 后续计划 | 潜力链接可继续新品加速，也可进入爆品助推、关键词推广等计划。 |
| 转化信号 | 用真实询盘、TM、订单和合规线上交易反馈强化产品价值。 |

正式执行时，订单和信保绑定必须来自真实客户、真实交易和合规线上流程，不能为了数据制造虚假交易。

## 时间轴

课程给出的节奏可整理为：

```text
3 天：选品类、选款式
3-5 天：精细化标题和链接准备
7 天内：新店开始推广，新品进入测试
2 周内：筛选潜力链接
1-3 个月：围绕潜力链接持续补流量、补询盘、补交易反馈
```

新店要抓住平台扶持期；老店要敢于用新发产品重新启动，而不是把所有资源继续压在历史低效链接上。

## 常见误区

1. 预算少却推广多个品类，导致每个品类都没有有效样本。
2. 不清楚公司主推品类和优势品类，推广后业务也无法承接。
3. 老店只改旧链接，不重新发布新产品。
4. 标题、产品、国家和推广计划没有统一方向。
5. 只追询盘，不补真实订单和业务反馈。
