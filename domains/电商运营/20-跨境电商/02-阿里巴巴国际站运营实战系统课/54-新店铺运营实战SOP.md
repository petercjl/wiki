# 新店铺运营实战 SOP

CH90 把前一节的“三阶段框架”落成具体动作。新店铺不是一上来就全行业、全品类、全国家铺开，而是先选少数可验证品类，把定位、发品、推广、筛品、转化信号和复盘循环跑通。

适用对象：

1. 6 个月以内的新店。
2. 接手后缺少有效数据的新店。
3. 老店重新启动时，需要用新品类重新做流量和标签。
4. 运营不知道每天该按什么顺序做动作的店铺。

## 主线流程

```text
店铺定位 -> 选热品 -> 组合爆款标题 -> 快速发品 -> 新品加速/直通车推广 -> 精挑产品 -> 真实交易信号沉淀 -> 复制循环
```

这条流程的关键不是“动作很多”，而是每一步都服务于同一个目标：用少量品类跑出有效产品、有效关键词、有效国家和有效买家标签，再复制到更多产品线上。

## 第一步：先做店铺定位

新店开始前先确认四件事：

1. 选择什么赛道：定制、RTS、半托管或组合赛道。
2. 主推哪些产品线：优先 1-3 个品类，不要一开始铺得太宽。
3. 目标国家和目标人群是谁：避免低质量询盘先把店铺标签带偏。
4. 公司能承接什么产品和订单：运营动作必须匹配供应、价格、MOQ、交期和业务能力。

如果运营刚进入新行业，还要补产品知识：产品材料、功能、应用场景、优势、价格带、目标客户和公司资源。只懂平台后台，不懂产品和公司承接，后续推广容易把错误买家引进来。

## 第二步：选热品切入

课程建议优先从市场已经验证过的产品切入，而不是凭感觉发品。

选品来源：

1. 市场参谋、榜单、搜索结果中的热销款式。
2. 优秀同行的主推产品、爆品、橱窗产品和推广产品。
3. 自己公司最有优势、最容易交付、最容易获得询盘的产品。

选择标准：

1. 流量大且相对稳定。
2. 关键词池足够大，可以组合多个标题。
3. 款式、价格、MOQ、供应能力与目标国家匹配。
4. 能在短期内发足够数量的高质量链接做测试。

## 第三步：组合爆款标题并快速发品

课程给出的操作强度是：一个品类可以先准备约 20 个标题，3 个月内发 200-500 个产品，越快完成初始铺设越好，尤其第一周要尽快启动。

这里的“多发”不是重复铺货，而是围绕同一产品线做差异化标题、属性、款式、图片和场景承接。

发品要求：

1. 标题覆盖核心词、精准词和长尾词。
2. 产品词、类目、属性、图片和详情页保持一致。
3. 每个链接都要有明确的目标关键词和目标买家。
4. 不用低相关词堆满标题。
5. 不把多个品类塞进同一个标题。

## 第四步：发完马上推广

新店发完产品后要尽快用新品加速计划或直通车获取初始反馈。课程强调，第一周内就要让新品进入测试，不要等产品自然沉寂。

推广配置原则：

1. 同一品类放在同一个计划里测试，不要把 A、B、C 多个品类混在一个计划中。
2. 计划要锁定目标国家和目标人群。
3. L0 阶段可以先提高目标国家溢价，尽快获得目标国家数据。
4. 数据积累到更高阶段后，再逐步提高目标人群溢价，降低泛国家溢价。
5. 前期出价不能过低，否则拿不到足够数据，也容易吸引低质量流量。

直通车的目标不是单纯花钱买流量，而是验证哪些标题、关键词、国家和产品可以形成有效询盘。

## 第五步：精挑产品

推广一段时间后，要从产品池里筛出值得持续优化的链接。

优先保留：

1. 有询盘、TM、点击、收藏或访客沉淀的产品。
2. 目标国家买家占比高的产品。
3. 询盘质量被业务认可的产品。
4. 有机会成为实力优品、准爆品或爆品的产品。

谨慎处理：

1. 只有低质量询盘的产品。
2. 点击多但访客国家严重偏移的产品。
3. 高消耗但无有效商机的产品。
4. 会持续拉偏店铺人群标签的产品。

不要只看询盘数量。新店铺前期如果把低质量询盘当成成功，后面会越来越难纠正国家、人群和产品标签。

## 第六步：用真实信号扶持有效产品

当某些产品已经证明能带来目标买家和有效询盘，就要把真实业务信号沉淀到平台。

合规动作：

1. 将真实客户订单尽量引导到信保或平台认可的线上交易链路。
2. 让业务和运营共同确认哪些产品适合重点扶持。
3. 真实履约、真实评价、真实复购都要回写到有效产品上。
4. 用真实订单和服务能力提升单品权重、店铺权重和推荐机会。

不应做：

1. 虚假交易。
2. 虚假评价。
3. 与真实业务无关的补数据。
4. 为了短期指标牺牲账号安全和店铺标签。

## 第七步：循环复制

新店的关键是先在少数品类跑通闭环，再把方法复制到更多产品线。

复制前要确认：

1. 已经知道哪个品类最有询盘潜力。
2. 已经找到有效标题结构和关键词组合。
3. 已经明确目标国家和目标人群。
4. 已经筛出可持续推广的产品。
5. 业务可以承接这些询盘和订单。

如果一个品类都没跑通，就继续扩大品类，运营会变成低效铺货。正确节奏是：先小范围跑通，再放大。

## 执行检查表

| 检查项 | 判断问题 |
| --- | --- |
| 店铺定位 | 赛道、品类、国家、人群是否明确？ |
| 发品数量 | 3 个月内是否围绕主品类形成足够测试链接？ |
| 标题覆盖 | 是否有约 20 个差异化标题组合？ |
| 推广计划 | 是否按品类建计划，并锁定国家/人群？ |
| 数据筛选 | 是否每周筛出潜力链接和拉偏链接？ |
| 询盘质量 | 业务是否反馈询盘有效？ |
| 真实交易 | 有效产品是否沉淀真实信保/线上交易信号？ |
| 复制节奏 | 是否先跑通一个品类再扩品类？ |
