# 店铺权重与单品权重提升

CH18-CH19 把搜索排名落到“店铺权重”和“单品权重”。课程的核心结论是：权重提升会让关键词排名、曝光、点击和询盘机会变多，同时降低整体运营成本和单次点击成本。

权重不是一个单独按钮，而是店铺基础、交易能力、转化效率、人群标签、资源绑定和买家反馈长期叠加的结果。

## 店铺权重提升

| 影响因素 | 课程要点 | 提升方向 |
| --- | --- | --- |
| 店铺综合分 | 有效产品、零效果产品占比、重复铺货等基础因子会影响店铺综合表现。 | 提升有效产品占比，清理或优化长期无效产品，避免重复铺货。 |
| 星级 | 星级是店铺权重最直观的体现之一，星级下降通常会带来流量下降。 | 持续优化星级相关转化、交易和服务指标。 |
| 信保交易额 | 交易额、支付转化率、复购等体现店铺成交能力。 | 提升真实成交、复购和支付转化，和行业平均/优秀同行对标。 |
| 店铺类型 | 金品店铺、出口通、行业优商等带来不同权益。 | 结合预算和阶段判断是否升级店铺类型。 |
| 人群标签 | 询盘多但成交差，会让人群标签变差，进而影响推荐流量质量。 | 获取精准人群，提升询盘到订单转化，形成正向闭环。 |
| 流量端口 | PC、APP、WAP 等端口表现会共同影响流量结构。 | 分端口看访客质量、点击和转化，不用单一总量判断。 |

店铺权重的正向闭环：

```text
精准人群 -> 高质量点击和询盘 -> 订单转化 -> 店铺数据提升 -> 权重提升 -> 更多精准流量
```

反向闭环也成立：如果长期引入不精准流量，点击和询盘看似有量，但订单差、复购差，店铺标签和权重都会被拖累。

## 单品权重提升

| 影响因素 | 课程要点 | 提升方向 |
| --- | --- | --- |
| 产品信息质量 | 类目匹配、文本相关、标题、关键词、属性、详情页决定初始权重。 | 先把基础信息做正确、完整、相关。 |
| 资源绑定 | 顶展、P4P、橱窗、主营认证等资源会增强单品权重。 | 资源集中给核心产品，并用数据判断是否继续投入。 |
| 交易数据 | 信保交易额、买家数、转化率越高，产品对平台价值越高。 | 通过推广、私域、老客和 RFQ 增加真实交易积累。 |
| 基础转化 | 点击率和询盘率决定流量获取效率。 | 主图、标题、价格和详情页要持续测试。 |
| 产品类型 | 爆品、实力优品、潜力品等对应不同流量场景和权益。 | 按产品类型制定不同目标，不把所有产品当爆品运营。 |
| 买家喜好度 | 点击率是重要信号，课程提到常见合理点击率区间可参考 0.8%-4%。 | 曝光足够后，重点看点击率是否证明买家喜欢。 |
| 访客标签 | 国家、行业/人群、PC/APP/WAP、类目/产品偏好、收藏、分享/分析、对比等行为会影响匹配。 | 让产品承接更精准访客，减少低质量流量。 |

## 店铺权重和单品权重的关系

店铺权重给产品提供整体流量地基，单品权重决定某条产品链接能否把机会转成排名和转化。

```text
店铺权重高 + 单品权重高 -> 更容易获得稳定搜索排名和转化
店铺权重高 + 单品权重低 -> 有机会但产品承接差
店铺权重低 + 单品权重高 -> 单品可突破，但规模受限
店铺权重低 + 单品权重低 -> 先补基础和有效产品池
```

## 提升权重的操作顺序

1. 清理基础问题：重复铺货、错类目、低质量信息、长期无效产品。
2. 优化核心产品：标题、关键词、属性、详情页、图片、视频、价格和物流。
3. 聚焦资源：橱窗、P4P、顶展、认证等给核心品和潜力品。
4. 提升转化：点击率、询盘率、支付转化率、订单和复购。
5. 校正人群：国家、行业、访客类型、端口、搜索词和产品标签要匹配。
6. 对标同行：所有核心转化指标至少高于行业平均，尽量逼近优秀同行。

## Agent 使用提示

诊断权重问题时，把问题拆成两张表：

- 店铺表：星级、店铺类型、有效产品数、零效果占比、交易额、复购、支付转化、回复率、人群标签。
- 单品表：类目、标题、关键词、属性、详情页、资源绑定、曝光、点击率、询盘率、订单、买家数、访客标签。

如果用户只说“排名差”，先判断是店铺权重不足、单品权重不足、搜索匹配错误，还是买家喜好度不足。
