# 产品数据分析与诊断指标

产品优化不是看到数据差就改标题、换主图、重做详情页。正确顺序是：先读懂产品数据链路，再判断是哪一环出现问题，最后只改真正影响问题的要素。

课程把产品优化放在“单品优化、爆品流量分析、全店商品提升流量”的框架里。它的目标不是让每个产品都变好，而是识别值得优化的产品，并避免把好爆品乱改成废品。

## 产品数据主线

产品诊断的基础链路是：

```text
曝光量 -> 点击量 -> 访客 -> 询盘/TM -> 订单量
```

对应的关键转化指标是：

| 链路 | 指标 | 主要诊断问题 |
| --- | --- | --- |
| 曝光 -> 点击 | 点击率 | 买家是否愿意进入产品。 |
| 点击/访客 -> 询盘/TM | 询盘转化率、商机转化率 | 买家进入后是否愿意咨询。 |
| 询盘/TM -> 订单 | 订单转化率 | 商机是否能被产品、价格、信任和业务承接。 |

数据分析要沿链路向下找问题。曝光不足时，不要先改详情页；点击很多但无询盘时，不要只盯排名；询盘很多但订单少时，问题可能已经进入业务承接和客户质量环节。

## 产品优化原则

课程给出的底层逻辑可以整理为：

```text
积累产品数据
-> 判断数据反馈的问题
-> 结合平台规则和买家需求修复
-> 提升点击率、询盘转化率和订单转化率
-> 让产品获得更多精准流量和权重
```

常见误区：

1. 新品发布后不看数据，只等平台自然给流量。
2. 爆品一有波动就乱改标题和主图。
3. 把时间花在长期无数据、无潜力的产品上。
4. 只做直通车推广，不诊断词品匹配和页面承接。
5. 只追求询盘数量，不分析询盘质量和订单结果。

产品优化应该提高“有效产品数量”和“有效关键词数量”，最终让自然流量更精准，询盘、TM 和订单更稳定。

## 八个诊断指标

从买家前台视角看，一个产品能否被点击和询盘，至少受八类因素影响。

| 诊断项 | 影响链路 | 判断方式 |
| --- | --- | --- |
| 主图 | 曝光 -> 点击 | 是否清楚表达产品、款式、卖点和目标市场偏好。 |
| 关键词 + 标题 | 曝光、点击、访客精准度 | 词是否准确，标题是否覆盖目标词，词品是否匹配。 |
| MOQ/起订量 + 价格 | 点击、询盘、订单 | 是否符合目标国家和买家层级承受区间。 |
| 评价分 | 点击、询盘、信任 | 分数是否影响买家信任和同类产品比较。 |
| 详情页内容与排序 | 询盘、订单 | 是否先回答买家最关心的问题。 |
| 优惠券、折扣、关联营销 | 询盘、订单 | 是否降低首次沟通门槛，并带动相关产品。 |
| 物流费 | 点击、询盘、下单 | 体积、重量和模板是否导致运费异常偏高。 |
| 访客国家精准度 | 询盘、订单 | 访客国家是否匹配价格、款式、MOQ 和页面表达。 |

这些指标不是同时大改的清单，而是诊断顺序。要先确定数据坏在哪一环，再选择对应指标。

## 访客国家精准度

访客国家是很多产品“看起来有流量但没有询盘”的核心原因。比如产品定位美国市场，价格、MOQ、款式和详情页都按美国买家设计，但进来的访客主要来自非目标国家，就会出现点击、访客不少，询盘和订单很差。

国家不精准时，应检查：

1. 标题和关键词是否吸引了错误国家买家。
2. 直通车国家投放是否过宽。
3. 产品主图、款式和价格是否更像另一个市场偏好。
4. 详情页是否没有表达目标国家关心的认证、规格、物流或使用场景。

如果是自然流量造成的国家偏移，不要急着大改爆品；如果是推广带来的错误国家流量，应优先从推广计划、关键词和国家定向修正。

## 主图与排名的误判

低点击率不一定等于主图差。课程特别提醒，如果产品在某关键词下排名太靠后，例如页面底部、广告后置位置或第二/第三页，低点击率可能只是位置问题。

判断顺序：

```text
低点击率
-> 查曝光来源和关键词
-> 前台搜索验证词品匹配
-> 查排名位置
-> 查价格/MOQ/评价/运费
-> 最后才考虑换主图
```

如果排名靠后，优先提升关键词排名、产品权重和资源支持；如果排名靠前但同类产品中点击弱，再测试主图和价格表达。

## 标题优化边界

标题是高风险优化项。课程的边界很明确：高转化产品、爆品和正在出询盘的产品，不要轻易改标题。

可以考虑改标题的场景：

1. 产品持续下滑，询盘和点击明显减少。
2. 新品在 30 天内没有获得有效曝光和点击。
3. 低曝光低点击产品本身没有历史转化资产。
4. 关键词覆盖明显错误，且现有标题无法承接目标市场。

不建议改标题的场景：

1. 爆品正在稳定出询盘或订单。
2. 产品关键词排名和商机仍处在增长期。
3. 只是觉得词不够多，但当前词能带来高质量商机。
4. 没有准备好推广、橱窗或其他资源承接标题变化后的流量波动。

## 诊断输出模板

做产品诊断时，可按下面结构输出：

```text
产品：
时间窗口：
数据现象：曝光 / 点击 / 访客 / 询盘-TM / 订单
问题环节：曝光不足 / 点击弱 / 询盘弱 / 订单弱
核心判断：
- 关键词和标题：
- 主图和价格：
- 详情页和信任：
- 访客国家：
- 推广来源：
建议动作：
观察周期：
不要动的资产：
```

最后一项“不要动的资产”很重要。很多产品优化失败，不是因为没改，而是把已有的排名、标题、主图或人群标签改坏了。
