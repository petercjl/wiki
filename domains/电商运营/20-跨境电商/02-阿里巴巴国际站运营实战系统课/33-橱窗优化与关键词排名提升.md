# 橱窗优化与关键词排名提升

橱窗优化的核心不是频繁换产品，而是用周数据判断哪些橱窗有效、哪些橱窗浪费资源，再把位置让给更有潜力的产品。课程建议以周为单位优化，重点看点击、询盘、TM、订单和关键词排名变化。

## 优化时间

课程建议每周二优化橱窗，因为上一周数据基本完整，方便判断产品表现。节奏可以固定为：

```text
周二查看橱窗效果
-> 保留好产品
-> 标记低效产品
-> 从准爆品、优品、新品和高转化低曝光产品里找替换品
-> 更新橱窗卖点和旺铺排序
-> 下周二复盘效果
```

不要一次性把全部橱窗都换掉。橱窗优化应针对低效位置，保留已有好效果、资源推广中或主推战略产品。

## 判断是否替换

判断一个橱窗是否要替换，先看最近一周，再看连续 2-3 周趋势。

| 表现 | 建议动作 |
| --- | --- |
| 点击低但有询盘/TM | 观察一周，可能是小流量高转化产品。 |
| 点击、询盘、TM、订单都低 | 优先替换。 |
| 连续 2-3 周无有效数据 | 替换或先回炉优化标题、主图、详情页。 |
| 有订单但无询盘 | 结合业务绑定、老客成交和产品数据判断，可继续观察。 |
| 质量分低且无潜力 | 不建议占用橱窗。 |

替换时要记录“换出产品、换入产品、原因、日期、下周效果”，否则很难复盘哪类产品适合橱窗。

## 替换产品来源

替换产品可从以下池子找：

1. 准爆品：离爆品门槛最近，适合冲刺。
2. 爆品和实力优品：已有数据，更容易放大。
3. 平台新品：适合短期测试。
4. 产品参谋中询盘/TM 排名前列但未加入橱窗的产品。
5. 高转化低曝光产品。

课程中特别提到，可以在产品分析里按周视图和询盘/TM 降序筛选，找到没有橱窗标识但数据不错的产品。

## 旺铺展示排序和橱窗卖点

橱窗不只是搜索加权，也会影响旺铺前台展示。橱窗产品很多时，至少要手动整理前 15-20 个位置，把最重要、最有优势、最想成交的产品放在前面。

橱窗卖点也要认真设置：

- 卖点要准确对应产品。
- 尽量设置 3 个以上核心卖点。
- 避免空泛词，如“高质量”“热卖”，要表达规格、材料、应用、服务或交付优势。
- 搜索、旺铺和产品展示中出现的卖点要与详情页一致。

## 关键词排名检查

课程给出一个简单验证法：在产品加入橱窗前后，查同一批关键词的排名变化。

操作步骤：

1. 整理橱窗产品标题、K1/K2/K3 和核心关键词。
2. 加入橱窗前查询关键词排名。
3. 加入橱窗后等待至少 1 小时，再查同一批关键词。
4. 选择目标国家 IP 或国家条件，查询前 3 页排名。
5. 记录无排名、页码、自然位次和变化。

不需要对所有橱窗都做全量统计，课程建议只对重点产品做验证。目标是判断橱窗是否真的改善关键词排名，而不是把运营变成纯统计工作。

## 提升关键词排名

橱窗只能提供加权机会，关键词排名真正提升还需要行为数据。

排名提升链路：

```text
关键词曝光
-> 点击
-> 询盘/TM
-> 订单
-> 关键词和产品权重提升
-> 自然排名更靠前
```

因此，提升关键词排名要优先提升该关键词下的点击量和商机转化。如果关键词有点击但无询盘，要检查词品匹配、目标国家偏好、主图、价格、MOQ、详情页和业务承接。

## 智能橱窗、直通车和顶展

智能橱窗适合在不知道怎么选产品、整体效果较差时小范围测试。可以先开 2-3 个，如果转化好就保留；如果效果差，应回到手动选择产品。

橱窗和直通车不冲突。橱窗影响产品标题覆盖关键词的自然加权，直通车影响广告位和付费点击。付费点击如果带来真实商机，也能反向支持自然排名。

橱窗和顶展也不冲突。顶展通常集中在某个关键词或广告位，橱窗可以帮助同一产品的其他关键词和自然排名，形成多入口曝光。
