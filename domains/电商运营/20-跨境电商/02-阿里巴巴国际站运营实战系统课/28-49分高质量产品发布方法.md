# 4.9 分高质量产品发布方法

高质量产品发布不是只把链接发出来，而是让平台和买家同时看懂：这是什么产品、适合谁、能不能买、怎么买、是否可信。课程用 4.9 分产品发布作为操作目标，同时提醒：如果暂时达不到 4.9 分，至少应把多数产品做到 4.5 分以上，因为后续优品权益常与产品质量分相关。

## 四个基础标准

| 标准 | 含义 | 影响 |
| --- | --- | --- |
| 内容准确 | 类目、标题、关键词、属性、图片和详情页真实表达产品。 | 决定搜索匹配、买家理解和转化。 |
| 属性完整 | 材质、功能、工艺、规格、使用场景等尽量填全。 | 增加被平台召回和买家筛选到的概率。 |
| 价格阶梯合理 | MOQ、价格、样品价、定制价和 RTS 价格符合真实交易逻辑。 | 影响买家询盘意愿和直接下单意愿。 |
| 物流合理 | 箱规、重量、尺寸、路线、样品物流和目标国家匹配。 | 物流过贵或路线太少会压低订单转化。 |

## 发布流程

```text
选择类目
-> 判断 RTS / 定制产品
-> 填标题与 K1/K2/K3
-> 分配产品分组
-> 填产地、品牌、型号
-> 填系统属性和自定义属性
-> 上传主图、颜色/款式图和视频
-> 编辑详情页模板
-> 设置价格、MOQ、样品、箱规和物流
-> 检测质量分并按提示优化
```

### 类目和产品类型

新发产品时，先通过类目搜索或关键词搜索选择正确类目。类目错误会影响搜索相关性。RTS 产品和定制产品要分开判断：RTS 链接需要更准确的价格、库存、规格和物流，定制链接则更强调能力、样品、MOQ 和可定制范围。

### 标题和 K1/K2/K3

标题与 K1/K2/K3 要来自前面整理好的关键词表。K1、K2、K3 应该有区分，不要三个位置重复同一个词。标题不必机械追求 128 个字符，核心是覆盖准确词并与产品真实匹配。

### 属性和自定义属性

系统属性尽量完整填写，包括类型、材质、工艺、季节、风格、性别、规格等。定制产品尤其要覆盖可生产的工艺、重量、规格和服务能力，以便匹配“工艺 + 产品”等搜索。

自定义属性建议用于放买家最关心的信息，因为这些内容会在详情页可见。比如可定制范围、打样周期、包装方式、证书、使用场景、核心卖点等。

### 图片和视频

产品主图、颜色图、款式图和细节图要清楚表达产品。课程强调，质量较好的视频可以增加内容竞争力，也可能获得平台流量倾斜。能配产品视频时，应优先为主推链接配置。

## 价格、样品和物流

价格要接近真实交易。定制链接可以略低来提升询盘意愿，但不能失真；RTS 链接因为买家可以直接下单，价格、MOQ、库存和规格必须按真实可履约能力填写。

箱规、毛重、体积重、包装长宽高会影响物流费用。课程提醒，物流计费通常会在实际重量和体积重中取较高者，填写过低可能造成物流亏损，填写过高或线路过少又会影响下单转化。

样品设置也会影响转化和质量分。定制产品即使主链接不直接售卖，也可以通过样品承接下单，因此样品价格、重量、尺寸和物流要准确。

## 4.9 分诊断

如果产品只能到 4.5-4.7 分，按后台提示逐项检查。课程中重点指出四个高频问题：

1. 主图不完整或质量不足。
2. 详情页过于简单，缺少图片、表格、文字和模块化表达。
3. 样品没有设置或设置不完整。
4. 物流模板、国家、线路或费用不合理。

详情页模板是提分快速入口。课程示例中，替换为有图片、表格和完整模块的模板后，产品质量分从 4.6 提升到 4.9。

## 物流模板注意事项

物流模板要覆盖目标市场国家。课程提到，模板中只要对应物流线路覆盖目标国家池里的国家即可满足平台规则要求，不一定每条线路都覆盖全部国家。溢价要遵守平台限制，课程示例提到最高 120 的规则，实操前应以当前后台为准。

如果物流费看起来明显高于市场水平，买家即使点击也可能不下单。出现“点击、访客、询盘有，但订单少”的产品，要回看价格、MOQ、样品、物流线路和物流费用。

## 营销模块

详情页里可以加入折扣、优惠券等营销模块。优惠券不只是促销工具，也能让买家看到更多店铺产品，从而增加二次浏览和关联转化。后台活动中心设置生效后，详情页模块可自动抓取展示。

## 发布后复盘

发布不是结束。每条主推链接应定期复盘：

- 质量分是否达 4.5 或 4.9。
- 关键词是否带来曝光和点击。
- 点击后是否有询盘、TM 或订单。
- 价格、MOQ、样品和物流是否压制转化。
- 详情页是否能回答买家的主要问题。
