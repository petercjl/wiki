# 新贸节采购节大促布局

3 月新贸节和 9 月采购节是平台大活动，活动期有大量流量，但报名成功不等于一定获得流量。真正能吃到流量的店铺，需要提前理解会场规则、选择合适产品，并在活动前把产品数据做起来。

## 大促结构

课程把大促入口分成：

1. 行业活动。
2. 主会场。
3. 核心会场。
4. 其他细分会场。

会场不是每年都一样，会根据平台方向、行业趋势和活动策略变化。不同会场、不同类目、不同年份的报名条件和排序规则都可能不同。

## 必须读懂两个规则

每个会场都要看两类规则。

| 规则 | 作用 |
| --- | --- |
| 报名条件 | 决定产品或店铺有没有资格进入会场。 |
| 排名/圈品条件 | 决定报名成功后是否有机会排到前面并获得流量。 |

很多店铺误以为报名成功就能获得流量，但如果产品数据差、排序靠后，流量仍然可能很少。

## 活动前准备表

活动文档发布后，应按行业把可执行事项列成表格。

| 字段 | 示例 |
| --- | --- |
| 会场名称 | 主会场、核心会场、行业会场、细分会场。 |
| 报名时间 | 开始和截止时间。 |
| 报名条件 | 店铺等级、产品类型、价格、折扣、转化率等。 |
| 排序规则 | 询盘、订单、评价、支付转化率、活动价、商品力等。 |
| 可报名产品 | 已有数据好、主推、有优势的产品。 |
| 缺口数据 | 需要补的点击、询盘、订单、评价或转化。 |
| 执行动作 | 推广、优惠、访客营销、直播、页面优化、真实订单线上化。 |
| 负责人 | 运营、业务、设计、老板或供应链。 |
| 验证指标 | 报名通过、会场排名、活动流量、询盘、订单。 |

先列清楚，再按时间节点执行，活动才有节奏。

## 产品选择原则

大促产品要选已经有数据基础、市场需求强、公司能承接的产品。

优先选择：

1. 主推品类和优势品类。
2. 已有询盘、TM、订单或评价的产品。
3. 目标国家和目标人群清晰的产品。
4. 价格、MOQ、折扣和交付能力适合活动的产品。
5. 接近会场排序门槛的潜力产品。

不要把长期无数据、承接弱、非主推的产品拿去报名核心会场。活动期同行好产品也会集中进入会场，弱产品可能比平时更难拿流量。

## 活动前中后节奏

```text
活动前：读规则 -> 选会场 -> 选产品 -> 补数据 -> 报名 -> 优化页面和优惠
活动中：看会场排名 -> 看流量来源 -> 看询盘和订单 -> 调整推广、优惠和承接
活动后：复盘产品、会场、流量、询盘、订单 -> 沉淀下一次活动清单
```

活动后不要只看总询盘。要看哪些会场带来了有效流量，哪些产品获得了沉淀，哪些产品虽然报名但没有排名和流量。

## 关键判断

1. 报名成功只是入场券，产品数据和排序位置才决定流量。
2. 会场规则每年会变，不能套用上一年的经验。
3. 不同行业的规则不同，必须看自己行业的官方文档。
4. 活动产品要提前打造，不能等活动开始才补数据。
5. 大促后要把有效产品继续用于自然、直通车、活动和场景流量。
