# 买家喜好度与千人千面标签匹配

CH14-CH15 解释买家喜好度和千人千面。课程要点是：搜索结果不是所有买家看到同一套固定排名，系统会根据买家历史行为、产品标签和店铺标签做个性化匹配。

运营不能只问“我这个关键词排第几”，还要问“目标买家是谁、他们喜欢什么、我的产品标签是否和这些买家标签重合”。

## 买家喜好度的行为链路

```text
曝光 -> 点击 -> 询盘/TM -> 订单
```

行为越靠后，代表买家喜好和需求越明确。

| 行为 | 信号强度 | 运营解读 |
| --- | --- | --- |
| 曝光 | 弱 | 系统给了展示机会，但不能证明买家喜欢。 |
| 点击 | 中 | 买家对主图、标题、价格或产品表达产生兴趣。 |
| 询盘/TM | 强 | 买家有明确沟通意图，产品和需求开始匹配。 |
| 订单 | 最强 | 买家用交易证明匹配成立。 |

课程用例子说明：同样看到产品，A 只点击，B 发询盘，C 点击、询盘并下单，则 C 与该产品的匹配度最高。

## 为什么点击率重要

买家喜好度主要通过点击率体现，并结合询盘转化和订单转化综合判断。1000 次曝光只有 1 次点击，和 1000 次曝光有 10 次点击，对系统来说是完全不同的市场反馈。

因此，诊断自然搜索时不能只盯曝光量。如果曝光不少但点击率差，问题可能在主图、标题、价格、款式、目标人群或搜索词匹配，而不是单纯“没有流量”。

## 标签如何沉淀

```text
买家搜索/点击/询盘/订单
-> 产品标签沉淀
-> 店铺标签聚合
-> 与后续买家标签匹配
-> 影响推荐、搜索展示和排序
```

产品标签来自产品文本、类目、属性、图片，也来自真实买家行为。店铺标签由多个产品、买家行为和成交结果累积形成。

如果店铺持续吸引不精准人群，询盘很多但成交差，系统沉淀出来的人群标签可能会变差，后续推荐流量也会更不精准。

## 千人千面的逻辑

千人千面意味着：同一个关键词，不同买家、不同国家、不同历史行为，看到的搜索首页可能不同。

影响个性化结果的标签包括：

- 买家国家和地域。
- 性别、年龄、身份或采购类型等人群特征。
- 历史搜索关键词。
- 浏览、点击、收藏、对比等行为。
- 询盘、TM 沟通和订单记录。
- 偏好的类目、款式、价格带、供应商类型。
- 产品标签和店铺标签。

产品标签和买家标签重合越大，产品越容易被推荐给这个买家，也越容易获得点击、询盘和订单。

## 排名观察的注意点

运营自己搜索关键词时，看到的结果不一定等于所有买家的结果。差异可能来自：

1. 千人千面导致买家标签不同。
2. 国家、语言、地域、终端不同。
3. 广告位、资源位和实时竞争状态变化。
4. 产品标签和店铺标签与搜索者历史行为的匹配不同。

所以检查排名时，要结合目标国家、目标买家画像、终端和数据后台，而不是只用一次搜索截图判断成败。

## Agent 使用提示

诊断买家喜好度时优先看：

1. 曝光是否足够。
2. 点击率是否低于行业或店铺平均。
3. 点击后的询盘率是否断层。
4. 询盘后的订单转化是否断层。
5. 当前来访国家、访客类型、搜索词和产品目标人群是否一致。
6. 是否长期用不精准流量拉低了产品和店铺标签。

## 使用边界

千人千面和标签匹配是课程抽象后的运营判断框架，不等于平台公开的精确算法。实操时以当前后台可见的人群、国家、搜索词、访客和转化数据为准。
