# 交易力与保障力星等级提升

交易力和保障力是店铺星等级的后半段能力分。交易力回答“店铺是否能把商机变成交易”，保障力回答“店铺是否能稳定履约并让买家放心”。如果商品力和营销力已经有基础，但星级仍上不去，通常要回头查交易和履约这两个板块。

## 交易力指标

交易力主要由站内交易、站外交易、复购率和支付转化率构成。课程里特别强调，交易额是交易力星级的主因素，很多店铺星级低不是因为没有流量，而是交易力没有跟上。

| 指标 | 运营含义 | 诊断问题 |
| --- | --- | --- |
| 站内交易额 | 平台内真实成交和支付能力。 | 询盘是否能转成订单，订单是否走平台。 |
| 站外交易额 | 课程时点下可折算部分交易贡献。 | 老客户是否有真实业务但没有线上沉淀。 |
| 复购率 | 客户重复购买和业务粘性。 | 老客户是否被定期激活。 |
| 支付转化率 | 访客、商机到支付的效率。 | 产品、价格、话术、信任和交易方式是否卡住成交。 |

站外交易、交易力口径和折算比例属于平台时点规则，实操前要以当前阿里巴巴国际站后台为准。

## 交易力提升路径

交易力不能只靠运营改页面，也不能只靠业务跟客户聊天。更稳的路径是把“精准商机、业务转化、真实交易线上化、老客户复购”连起来。

```text
提升询盘质量
-> 提升业务转化话术和报价能力
-> 推动真实订单在线成交
-> 激活老客户复购
-> 沉淀交易额、买家数、支付转化率和复购率
-> 拉升交易力星级
```

可执行动作：

1. 复盘高质量询盘来源，保留能带来真实订单的关键词、国家、产品和推广计划。
2. 对业务常见卡点做话术训练，例如 MOQ、价格、样品、交期、定制能力和付款方式。
3. 对已有真实需求的老客户，引导其通过平台完成合规线上交易，让交易数据沉淀到店铺。
4. 在大促、节假日和行业旺季前做老客户复购触达。
5. 计算真实线上交易手续费和直通车获客成本，不只看单次费用。

这里的“线上化”只适用于真实客户、真实订单和真实履约。不能为了星级制造虚假订单、虚假买家或虚假评价。

## 星级流量层级

课程给出的经验判断是：零星到一星、二星的差距明显，一星/二星到三星以上又会出现新的流量和买家质量差异。因此最低目标不是“完美五星”，而是先避免零星，再从二星向三星突破。

星级提升的现实顺序通常是：

```text
零星店铺：先修短板，避免基础能力不达标
一星/二星店铺：优先冲三星，争取更多流量和买家质量
三星以上店铺：按四个能力分的最低项做月度维护
```

因为星级规则是“四个能力分取短板”，交易力不足会拖住全店，即使商品力和营销力较好，也可能无法体现到最终星级。

## 保障力指标

保障力偏履约和买家信任，课程中重点提到：

| 指标 | 运营含义 | 常见问题 |
| --- | --- | --- |
| 轨迹上网及时率 | 成交后物流轨迹是否及时有效。 | 单号填写慢、物流国家/发货地不一致、轨迹异常。 |
| 买家评价分 | 买家对商品和服务的真实反馈。 | 邀评不足、服务承接差、低分评价未复盘。 |
| 风险健康分 | 店铺履约和风险综合健康状态。 | 发货率低、逾期发货、纠纷和投诉。 |
| 发货率/按时发货率 | 是否按约定时间真实发货。 | 发货周期设置过短、订单逾期、内部履约协同差。 |

课程里把发货率视为保障力的基础门槛，并提到低于某些阈值可能导致星级严重受损。具体阈值和影响口径要以当前后台为准，但“发货率是底线”这条原则可以保留。

## 保障力提升路径

保障力先查具体指标，再做流程修复：

```text
查看保障力短板
-> 定位是轨迹、评价、风险健康还是发货率问题
-> 修复物流、评价、履约和订单流程
-> 月末前复核预测星级
-> 固化业务和仓储协作节奏
```

轨迹上网及时率的动作：

1. 成交后及时填写真实物流单号。
2. 保证物流国家、发货国家和订单信息一致。
3. 对轨迹异常订单及时跟进物流服务商，保留真实凭证。

买家评价分的动作：

1. 业务在客户满意后主动邀请真实评价。
2. 把低分评价拆成产品质量、交期、沟通、包装和售后问题。
3. 对满意客户建立复购和评价提醒机制，但不能用虚假交易或利益诱导制造评价。

风险健康分和发货率的动作：

1. 起草订单时给出真实且充足的发货周期。
2. 每周查看逾期风险订单，提前协同生产、仓储和物流。
3. 对已发生的异常订单做真实物流补录、履约说明和客户沟通。
4. 每月 20 日后跟进星级预测，23-25 日前重点检查发货率和风险项。

## 月度检查节奏

保障力和交易力都不适合等月末最后一天才处理。可按月度节奏：

| 时间 | 检查动作 |
| --- | --- |
| 每周 | 查看交易额、支付转化率、发货率、轨迹和评价异常。 |
| 每月 20 日前 | 查看星级预测和四个能力分短板。 |
| 每月 23-25 日 | 对发货率、风险健康分和交易力短板做最后纠偏。 |
| 月末 | 确认真实订单、真实履约和老客户复购沉淀完成。 |
| 次月 5 日左右 | 复盘星级更新结果，记录本月有效动作。 |
