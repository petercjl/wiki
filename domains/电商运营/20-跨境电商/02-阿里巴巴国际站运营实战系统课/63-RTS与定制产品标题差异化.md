# RTS与定制产品标题差异化

RTS 和定制产品标题不能完全一样，因为两类买家的搜索意图不同。标题决定流量精准性，也会影响询盘转化率。

## 底层差异

RTS 买家通常更明确：

1. 找具体款式。
2. 找能直接下单的产品。
3. 更关注款式、规格、颜色、功能等明确属性。
4. 标题越精准，点击和下单意图越容易匹配。

定制买家通常更开放：

1. 找某一类产品。
2. 初期未必确定颜色、功能、型号或风格。
3. 需要供应商承接多款式、多方案或多场景需求。
4. 标题要覆盖产品类型和常规通用词。

## RTS标题写法

RTS 标题偏“款式精准”。

适合包含：

1. 明确款式词。
2. 精准属性词。
3. 具体颜色、功能、规格。
4. 能被主图直接展示的特征。
5. 买家直接搜索后可以马上判断是否下单的词。

不适合：

1. 把多个互斥款式塞进一个标题。
2. 用过宽通用词吸引不精准点击。
3. 标题描述和现货 SKU 不一致。

## 定制标题写法

定制标题偏“产品类型覆盖”。

适合包含：

1. 产品类型词。
2. 大词和精准词组合。
3. 多颜色、多型号、多风格的通用表达。
4. 买家可能会进一步沟通确认的词。
5. 详情页可以补充的多款式能力。

例如同一系列手机壳，定制链接可以覆盖 A/B/C 多个型号；RTS 则更适合按 A 型号、B 型号、C 型号拆不同链接。

## 决策表

| 问题 | RTS链接 | 定制链接 |
| --- | --- | --- |
| 买家是否直接下单 | 是 | 通常不是 |
| 标题重心 | 款式与精准属性 | 产品类型与多样需求 |
| 词类型 | 精准词、长尾词 | 大词、精准词、通用词 |
| 主图要求 | 与标题特征高度一致 | 展示代表款并在详情页补充能力 |
| 多型号处理 | 拆链接更稳 | 同类型号可合并覆盖 |

## 运营边界

1. RTS 标题不要为了流量写不承接的款式词。
2. 定制标题可以更广，但不能跨产品类目。
3. 多型号只有在同一产品或同一类产品下才适合合并。
4. 标题、主图、属性、详情页必须保持可解释一致。
5. 直通车推广时，RTS 和定制链接的目标词、国家和人群要分开判断。
