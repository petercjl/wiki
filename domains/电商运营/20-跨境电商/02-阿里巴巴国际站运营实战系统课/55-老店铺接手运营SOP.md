# 老店铺接手运营 SOP

老店铺接手不能直接套新店动作。课程把老店分成两类：曾经做得很好但闲置几年的 TOP 店铺，以及效果普通、数据长期一般的老店。两类店都要先诊断，再决定是否盘活原资产、重做定位或按新店方法重新启动。

## 适用场景

1. 接手长期闲置的 TOP 店铺。
2. 接手效果一般的普通老店铺。
3. 从 A 行业切换到 B 行业，不熟悉产品和买家。
4. 店铺有历史数据，但不知道哪些资产还能用。
5. 老方法失效，需要重新建立运营框架。

课程提醒：方法会过期，但分析思路可以迁移。接手老店时，不要机械复制过去的动作，要根据当前行业、产品、店铺数据和公司资源重新推导。

## 主线流程

```text
学习产品和公司 -> 重做店铺定位 -> 诊断综合数据 -> 盘点资源资产 -> 制定阶段计划 -> 推广与真实交易信号沉淀 -> 复盘迭代
```

## 第一步：补产品知识和公司资源

跨行业接手时，运营先要补三类知识：

1. 产品知识：材料、功能、应用场景、价格带、MOQ、交期、质量标准。
2. 公司优势：工厂、贸易、服务、证书、案例、供应链、销售支持。
3. 业务资源：现有客户、历史订单、重点市场、业务最想要和最不想要的询盘。

如果不懂产品，就容易用错关键词、选错国家、推广错误款式，最终给业务带来不想要的询盘。

## 第二步：重做店铺定位

老店铺的定位不能默认沿用历史状态。要重新确认：

1. 当前最适合的赛道是什么。
2. 店铺应主推哪些产品线。
3. 目标国家和目标人群是否变化。
4. 历史店铺标签是否符合当前目标。
5. 公司现在是否有资源支持原来的方向。

如果目标国家是美国、欧洲等高质量市场，但当前询盘长期来自非目标国家，说明店铺标签和流量入口可能已经偏移。

## 第三步：诊断店铺综合数据

老店要先判断它处于行业平均、行业优秀，还是低于行业平均。

诊断维度：

1. 曝光、点击、访客、询盘/TM、订单主链路是否断开。
2. 流量问题、转化问题还是订单问题最严重。
3. 目标国家、询盘国家和访客国家是否一致。
4. 产品是否还有排名、询盘和订单沉淀。
5. 店铺综合实力、星等级、优品/爆品数量是否支持当前目标。

诊断结果要写进运营计划，而不是只停留在“感觉店铺不行”。

## 第四步：盘点历史资产和资源

老店铺往往不是从零开始，重点是判断哪些资产还能用，哪些资产应该清理。

盘点清单：

| 资源 | 要看什么 |
| --- | --- |
| 产品总量 | 有多少有效产品、沉寂产品、重复产品、低质量产品。 |
| 关键词覆盖 | 主推品类的关键词是否覆盖，是否有排名词。 |
| 优品/爆品/准爆品 | 数量、类目分布、与行业平均/优秀差距。 |
| 橱窗/顶展/问鼎等资源 | 是否绑定到真正主推和高转化产品。 |
| 直通车预算 | 日预算、计划类型、推广品类、询盘成本和转化结果。 |
| 历史询盘 | 国家、人群、产品、业务反馈和成交情况。 |
| 历史订单 | 是否有真实线上交易可沉淀到有效产品。 |

如果资源少，要判断公司能否补资源；如果不能补，就要选择更窄的突破口，不要做大而全计划。

## 第五步：和老板、业务一起制定计划

老店接手不能运营一个人埋头做。课程强调要和老板、业务沟通，因为运营目标必须和公司要的客户、产品和订单一致。

计划至少包含：

1. 阶段目标：询盘量、询盘质量、订单或某个品类突破。
2. 时间节点：每天、每周、每月分别做什么。
3. 产品范围：先做哪些品类，哪些暂时不做。
4. 推广预算：为什么要投、投哪里、预期看什么指标。
5. 业务协同：哪些询盘需要业务反馈，哪些订单可以线上化沉淀。

如果公司不想卖某些产品，运营不要为了询盘量去发和推这些产品。否则短期数据可能变好，长期会增加业务内耗。

## 第六步：重新启动推广和真实交易信号

老店铺如果历史数据已经沉寂，可以用新计划、新品和直通车重新启动。

动作顺序：

1. 先选择 1-3 个主推品类。
2. 整理或新发一批高质量链接。
3. 以目标国家和目标人群建推广计划。
4. 用数据筛选潜力产品和拉偏产品。
5. 对有效产品沉淀真实询盘、TM、订单和履约信号。

信保或线上交易只应承接真实业务订单。运营要和业务确认哪些产品、哪些客户、哪些订单适合回到平台沉淀数据，再向老板申请预算和资源。

## 老店类型判断

| 店铺类型 | 优先动作 |
| --- | --- |
| 曾经优秀但闲置 | 先盘点历史爆品、排名词、订单和国家标签，再判断是否能恢复。 |
| 普通老店 | 先找主链路短板，再选择 1-3 个品类重新突破。 |
| 行业切换店 | 先补产品知识和公司资源，不能直接按旧行业经验推广。 |
| 标签混乱店 | 先清理低质量产品和推广流量，重新锁定国家、人群和产品。 |
| 资源不足店 | 缩窄目标，集中预算和人力做单点突破。 |

## 接手老店避坑

1. 不要一接手就大批量改爆品标题。
2. 不要把历史有价值链接当作无效链接全部重做。
3. 不要沿用旧计划而不看目标国家、人群和询盘质量。
4. 不要只追询盘量，忽略业务是否愿意承接。
5. 不要脱离公司资源做过大的运营规划。
