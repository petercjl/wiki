# 店铺星等级价值与指标框架

店铺星等级决定店铺权重和流量分配层级。课程把星等级放在店铺增长的底层指标里：星级越高，平台给店铺的起始流量、买家层级、活动机会和推广效率越可能更好。

## 提升星等级的价值

星等级提升的好处：

1. 流量分配更多：更高星级通常意味着更高的起始流量权益。
2. 店铺权重更高：同等条件下，产品排名更容易靠前。
3. 买家质量更好：平台买家和卖家分层后，高层级店铺更容易匹配高质量买家。
4. 活动和场景流量更多：星级常是报名活动和系统圈品的条件之一。
5. 直通车成本更容易下降：店铺和单品权重提高后，推广排名和数据积累效率更好。

课程中的流量数字只是示例，不能作为固定规则。真正判断要看当前行业和后台权益说明。

## 双赛道框架

星等级分定制赛道和快速交易赛道。两个赛道都看四个能力分，但部分指标不同。

| 能力 | 定制赛道 | 快速交易赛道 |
| --- | --- | --- |
| 商品力 | 全店优品数、橱窗优品占比。 | 全店优品数、橱窗优品占比，并关注 RTS 可交易商品数量。 |
| 营销力 | 营销流量指数、商机指数、商机转化率、平均回复时间、RFQ 营销能力等。 | 营销流量指数、RTS 商机转化率、平均回复时间、极速回复率等。 |
| 交易力 | 站内外交易额、支付转化率、复购率等。 | 站内外交易额、RTS 支付转化率、复购率、RTS 站内交易买家数等。 |
| 保障力 | 风险健康分、买家评价分、轨迹上网及时率等。 | 风险健康分、买家评价分、轨迹上网及时率、到货保障覆盖率等。 |

如果店铺主做定制，就先记定制赛道，再关注 RTS 赛道中特有的变化指标；如果主做 RTS，则反过来。

## 星级评定规则

课程强调几个容易被忽视的规则：

1. 双赛道取高星级：定制赛道和快速交易赛道哪个星级高，店铺最终星级就取哪个。
2. 四个能力分取最低：单一赛道内，商品力、营销力、交易力、保障力中最低的能力分决定该赛道星级。
3. 月末数据决定下月：每月最后一天的综合数据决定下个月星级，课程口径为美国时间。
4. 后台显示滞后：星级通常在北京时间每月 5 日左右更新展示。
5. 类目口径重要：星级按店铺内产品数量最多的二级类目参与评定。

这意味着运营不能只拉高一个指标。只要一个能力分明显短板，整个赛道星级就会被拉低。

## 分值门槛

课程给出的星级能力分门槛：

| 星级 | 能力分门槛 |
| --- | ---: |
| 1 星 | 60 分 |
| 2 星 | 70 分 |
| 3 星 | 80 分 |
| 4 星 | 85 分 |
| 5 星 | 90 分 |

这些门槛应以当前后台为准。实操上，运营至少要在每月 20 日前看一次预测星级和四项能力短板，25 日到月底持续跟进，避免月底掉星影响下月流量。

## 诊断方法

星等级诊断不要只看总星级，要拆到赛道、能力分和小指标：

```text
看定制赛道星级
看快速交易赛道星级
-> 取高者作为当前店铺主星级
-> 在目标赛道内找最低能力分
-> 在最低能力分里找最容易补的小指标
-> 20 日前制定补分动作
-> 月底前每日跟踪
```

如果店铺预测星级突然下降，优先检查最低能力分是否因为退款、发货、评价、回复时间、交易额或推广商机转化变化而被拉低。
