# 新店铺三阶段运营框架

新店铺运营要有阶段感。课程把新店和长期无人打理的老店，都放进一条主线：先做询盘量，再做询盘质量，最后做询盘数量、质量和订单的均衡。

适用对象：

1. 6 个月以内的新店。
2. 刚接手的 3-12 个月店铺。
3. 一年以上但长期没人打理、数据沉寂的老店。
4. 不知道每天该做什么、没有运营节奏的店铺。

## 运营终极目标

目标不是单纯让询盘变多，而是让高质量询盘达到行业优秀，并最终完成交易额目标。

因此阶段目标不能错：

```text
阶段 1：先建立询盘量和店铺数据
阶段 2：提升询盘质量和订单转化
阶段 3：在质量稳定前提下扩大数量，并持续做订单
```

## 阶段一：询盘量优先

新店开始时缺少数据，要先通过询盘、TM、点击和访客建立基础权重。

但询盘量优先不等于低价乱引流。课程特别提醒，低价引流容易带来非目标国家和低质量买家，短期询盘多，长期会造成店铺标签混乱。

阶段一要做：

1. 明确目标国家和主推品类。
2. 发布足够数量的高质量产品。
3. 用新品、直通车、活动和场景流量获取初始反馈。
4. 价格和 MOQ 保持在真实可承接范围。
5. 不为了询盘量牺牲店铺标签。

## 阶段二：询盘质量优先

当询盘量起来后，重点转向询盘质量。这个阶段询盘总数可能下降，因为非目标国家、低意向买家和低质量流量被筛掉了。

质量提升的判断不是“询盘越多越好”，而是：

1. 目标国家询盘占比提高。
2. L1+ 或更高质量买家占比提高。
3. 目标产品、目标品类的询盘增长。
4. 业务反馈客户更容易沟通。
5. 订单转化率开始提升。

如果询盘质量看似提高但订单没有任何改善，要进一步判断是运营人群仍不准，还是业务回复、报价和跟进能力不足。

## 阶段三：数量、质量和订单均衡

当质量稳定后，再扩大有质量的询盘数量。不能为了追求数量，把店铺重新拉回第一阶段的低质量询盘状态。

阶段三要做：

1. 持续监控询盘质量。
2. 放大已验证高质量产品、关键词和国家。
3. 与业务协同提升订单转化。
4. 让业务反馈客户需求趋势，反哺选品和页面表达。
5. 用真实信保订单和线上交易数据强化有效产品。

运营和业务需要共同复盘：哪些产品带来高质量询盘，哪些客户需求正在变化，哪些业务问题影响成交。

## 时间预期

课程给出的整体周期为 3-6 个月，常见情况下每个阶段约 2 个月。国际站是批发类平台，成交和数据积累需要时间，不能用即时零售平台的节奏判断效果。

如果产品优势强、资源充足、业务承接好，节奏可能更快；如果产品、国家、词和业务都需要调整，则要给足测试和积累周期。

## 新店运营时间轴

| 周期 | 重点 |
| --- | --- |
| 第 1 月 | 明确品类、国家、产品结构，发布高质量产品，启动基础推广和活动。 |
| 第 2 月 | 筛选潜力链接，提升询盘量，观察国家和人群是否偏移。 |
| 第 3-4 月 | 清理低质量流量，提升目标国家和目标买家询盘占比。 |
| 第 5-6 月 | 放大高质量词、产品和国家，与业务共同提升订单转化。 |
