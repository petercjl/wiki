# 词品匹配与关键词收集表

真正的词品匹配不是“关键词里有 T-shirt，产品也是 T-shirt”这么简单，而是买家在某个国家、某个场景下搜索这个词时，平台展示的产品是否符合他的款式、属性、价格和需求偏好。

## 什么是真正的词品匹配

课程把关键词定义为买家和产品之间的桥梁。一个买家可能搜索多个词，一个词也可能匹配多个产品，一个产品也可以承接多个关键词。

```text
目标国家买家
-> 搜索关键词
-> 平台召回多个产品
-> 买家更偏好其中某些款式/属性/价格
-> 这些产品才是真正匹配这个词
```

如果美国买家搜索关键词 A，实际更偏好产品 B/C 的款式，而店铺把关键词 A 投给产品 D/E，即使字面上都属于同类产品，也可能是错误匹配。

## 错误匹配的常见表现

- 产品词和关键词表面一致，但目标国家买家不点击。
- 点击率可以，但询盘、TM、订单很弱。
- 直通车花费不低，却没有形成关键词排名和产品权重。
- 标题覆盖很多词，但主图、属性、详情页不能承接这些词。
- 用全球热门词投目标国家市场，却没有国家偏好验证。

词品匹配要同时看关键词偏好和产品偏好。关键词偏好可来自关键词指数、国家筛选、买家参谋和市场参谋；产品偏好可来自同行高转化产品、榜单、交易产品、访客行为和历史询盘。

## 关键词收集原则

站内关键词优先，站外关键词补充。Google Trends、Google 搜索、Amazon 等外部工具可以帮助判断趋势和款式，但国际站运营最终要看阿里巴巴国际站内部是否有搜索、点击、询盘和交易承接。

## 站内关键词来源

| 来源 | 用法 |
| --- | --- |
| 关键词指数 | 输入词根，选择近 30 天，导出相关词，并按搜索指数、点击率、卖家规模指数筛选。 |
| 引流关键词 | 找出已经带来曝光、点击或询盘，但还没有被设置为产品关键词的词。 |
| 买家参谋 | 按行业、产品和买家偏好收集国家、人群和需求词。 |
| 访客详情 | 查看访客全站偏好关键词；相关且有指数的词可进入词库。 |
| 产品 360 / 产品参谋 | 找到产品或推广产生的高流量、未覆盖、可补充关键词。 |
| 市场参谋 | 从类目、国家、热销产品标题和买家偏好中提取词。 |
| 定向产品和属性 | 产品属性、材质、工艺、功能等可作为关键词补充。 |
| 国际站前台和 APP 下拉词 | 用核心词搜索，收集系统建议词和买家常搜表达。 |
| 产品发布推荐词 | 发布产品时平台推荐的词，可补充到产品标题和关键词库。 |
| 数据概览推荐词 | 后台推荐词可作为补充来源。 |
| 直通车和顶展建议词 | 推广创建和优化时推荐的词，用指数和词品匹配再过滤。 |

## 关键词表字段

建议把关键词收集到统一表格，至少包含以下字段：

| 字段 | 作用 |
| --- | --- |
| 关键词 | 原始英文关键词。 |
| 词根 | 由哪个核心词或同义词根拓展而来。 |
| 词类型 | 行业词、核心词、精准词、长尾词。 |
| 来源 | 关键词指数、买家参谋、访客详情、直通车等。 |
| 搜索指数 | 判断需求规模。 |
| 点击率 | 判断买家点击意愿。 |
| 卖家规模指数 | 判断竞争强弱。 |
| 目标国家 | 词在哪些国家更值得使用。 |
| 匹配产品 | 对应哪类产品、款式或链接。 |
| 使用位置 | 标题、K1/K2/K3、属性、详情页、直通车、屏蔽词。 |
| 备注 | ASR 校正、风险、测试结果、复盘结论。 |

## 去重和分组

关键词收集完后要按词根、品类、国家、产品类型和使用场景去重分组。一个混乱词库会让标题、发品和推广互相打架；一个清晰词库则能让不同产品覆盖不同关键词入口。

## 与产品和页面回写

关键词不是只放进表格里。选定关键词后，要回写到产品标题、K1/K2/K3、属性、主图、详情页模块、旺铺分组和直通车计划中。只有词、品、图、属性和页面一致，才会形成稳定转化。
