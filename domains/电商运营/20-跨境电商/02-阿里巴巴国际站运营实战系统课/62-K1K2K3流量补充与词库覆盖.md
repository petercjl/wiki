# K1K2K3流量补充与词库覆盖

K1/K2/K3 不是标题的重复栏，而是产品链接的关键词补充层。它们要在保持产品精准匹配的前提下，补足标题没有覆盖的关键词。

## 两个原则

1. 补充标题：标题里已有 A/B/C/D，K1/K2/K3 可以补充 E/F/G，但仍然围绕同一核心产品词。
2. 精准匹配产品：每个词都要能被产品主图、属性、详情页和买家需求承接。

如果 K1/K2/K3 写得越长但越不精准，后续直通车和自然搜索可能出现高曝光低点击、高点击低询盘或无效消耗。

## K1K2K3写法

执行步骤：

1. 明确产品核心词。
2. 每个 K1/K2/K3 都包含产品核心词或能明确指向产品。
3. 从关键词指数中选择 1-7 个词组合成一个 K。
4. 三个 K 之间尽量不同，不要一模一样。
5. 与标题保留部分关联，但要增加标题没有覆盖的词。
6. 优先选择有搜索指数且点击率大于等于 0 的词。
7. 冷门行业词少时，可适当使用 0 点击率但相关性强的词。

示意：

```text
标题：A属性 + B属性 + C核心词
K1：D属性 + E属性 + C核心词
K2：F场景 + G规格 + C核心词
K3：H人群 + I功能 + C核心词
```

## 三个不要

1. 不要随意造词。运营想到的词不等于买家会搜的词。
2. 不要为了写满 128 字符而复制同行长标题。
3. 不要 K1/K2/K3 三个框完全一样。

三个框都一样，本质只覆盖了一个关键词组合，浪费了补充位。

## 我的词库应用流程

CH108 提供了用“我的词库”补足标题和 K1/K2/K3 的方法。

流程：

1. 按品类或产品线创建词库分组。
2. 导入已整理关键词，或手动添加关键词。
3. 等待数据更新后查看搜索指数、效果和使用状态。
4. 重点找 T 和 K 都没有用过的词。
5. 剔除无搜索指数、弱相关、无承接能力的词。
6. 将可用词整理到表格。
7. 再按标题组合方法选款、组标题、写 K1/K2/K3 和推广。

这里的 T 通常代表标题用过，K 代表 K1/K2/K3 使用过。具体字段口径以后端当前页面为准。

## 词库筛选标准

优先选择：

1. 有搜索量。
2. T/K 未使用。
3. 与当前款式强匹配。
4. 指标表现优于平均或同行参考。
5. 能补充标题没有覆盖的流量池。

谨慎选择：

1. 没有搜索指数的词。
2. 与当前产品图不匹配的词。
3. 仅有营销感但无买家搜索证据的词。
4. 与标题重复过多的词。

## 引流关键词和我的词库的区别

课程将两者作为相似方法使用：

1. 引流关键词：更偏店内已有关键词表现，范围较窄。
2. 我的词库：同时包含店内和平台维度，可覆盖范围更宽。

如果要系统扩词，优先用我的词库；如果要诊断当前店铺已有流量词，优先看引流关键词。
