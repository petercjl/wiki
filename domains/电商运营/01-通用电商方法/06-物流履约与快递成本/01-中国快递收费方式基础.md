# 中国快递收费方式基础

## 第一原则：快递价格是动态规则

这页记录的是中国快递收费的基本结构，不是任何快递公司的长期价目表。快递公司价格、计重单位、体积重系数、增值服务费、偏远附加费、电子面单/隐私面单收费和月结折扣会随时间、地区、产品、线路、账号、平台和合同变化。

因此，页面中涉及顺丰、中通、圆通、申通、中国邮政、中通国际等公司的规则，只能作为“公开资料中的示例口径”，用于理解不同计费机制。实际寄件、报价、对账、投诉和成本核算，必须以寄件当日快递公司官方价目、当地网点确认、平台电子面单后台、ERP/打单系统账单、月结合同和监管部门最新规定为准。

## 快递费用由什么决定

快递收费通常由这些变量共同决定：

| 变量 | 含义 | 商家要核验什么 |
| --- | --- | --- |
| 始发地和目的地 | 同城、省内、跨省、偏远、港澳台、国际线路价格不同。 | 线路分区、偏远附加费、是否可达。 |
| 服务产品 | 标快、特快、经济件、生鲜、冷运、大件、快运、国际件等。 | 产品名称、时效、限重限尺寸、是否计泡。 |
| 实际重量 | 商品加外包装、填充物、冰袋、纸箱等的称重结果。 | 不要只用商品净重，必须样本实称。 |
| 体积重量 | 大而轻的包裹按体积折算重量。 | 长宽高、抛比系数、是否取大。 |
| 计费重量进位 | 不足一个计费单位如何向上取整。 | 首重、续重单位、小数保留规则。 |
| 增值服务 | 保价、包装、上门、冷链、易碎、签回单等。 | 是否单独计费，是否已含在合同价中。 |
| 面单相关成本 | 电子面单、隐私面单、打单系统、耗材和打印设备成本。 | 是否按单收费，是否被藏在平台或仓配服务费里。 |

监管口径上，快递企业应公示服务种类、服务地域、服务时限、资费标准、投诉处理等信息。对于知识库和经营分析，最重要的是保存“核验路径”，而不是把某一日价格写成固定事实。

## 基础运费：首重和续重

国内个人寄件和普通小件常见公式是：

```text
运费 = 首重价格 + 续重重量 × 续重单价
```

示例：

```text
首重 1kg 以内 12 元
续重每 1kg 4 元
计费重量 3kg
运费 = 12 + (3 - 1) × 4 = 20 元
```

这个例子只解释公式。实际首重是多少、续重按 1kg/0.5kg/0.1kg 还是其他单位计算，要看快递公司、线路、产品和账号合同。

## 计费重量：实际重量和体积重量取大

很多快递产品会按下面的逻辑确定计费重量：

```text
实际重量 = 包装后称重 kg
体积重量 = 长 cm × 宽 cm × 高 cm ÷ 抛比系数
计费重量 = max(实际重量, 体积重量)
```

体积重量也叫泡重、抛重、计泡。常见抛比系数有 `8000`、`6000`、`5000`、`3000` 等。系数越小，折算出来的体积重量越大，泡货越容易被按体积重收费。

例子：

```text
包裹尺寸：60cm × 40cm × 30cm
实际重量：5kg
抛比系数：6000
体积重量 = 60 × 40 × 30 ÷ 6000 = 12kg
计费重量 = 12kg
```

这个包裹不是因为“真的有 12kg”，而是因为占用运输空间，按体积重量计费。

## 公司规则示例只用于理解机制

不同快递公司和不同产品的计费单位、体积重系数和进位方式差异很大，而且会变化。下面只作为机制示例，不作为长期标准。

| 示例维度 | 可能出现的规则差异 | 经营含义 |
| --- | --- | --- |
| 计费单位 | 首重不足 1kg 按 1kg；续重可能按 1kg、0.5kg、0.1kg 或阶梯单位计算。 | 同样 1.1kg 的包裹，在不同规则下费用可能不同。 |
| 体积重系数 | 同一公司国内件、港澳台件、国际件、冷运件、大件可能使用不同抛比。 | 泡货、保温箱、生鲜、抱枕、棉服类 SKU 要单独测算。 |
| 产品线 | 标快、特快、经济件、冷链、大件、快运的收费结构不同。 | 不能只比较“哪家公司便宜”，要比较产品和时效。 |
| 地区差异 | 偏远地区、县乡网点、港澳台、国际线路可能有附加费或不可达。 | 运费模板和包邮策略要按地区分层。 |
| 商家合同 | 月结价、平台价、电子面单价、网点协议价可能不同。 | 商家成本应以账单和合同为准，而不是官网散客价。 |

当要沉淀某家公司规则时，应另建“日期化快照”，例如 `顺丰国内标快计费规则-2026-07-01`，并在标题和正文中标注快照日期、来源 URL、适用地区、产品和核验人。

## 附加费用

快递费用可能不止基础运费：

| 费用 | 说明 | 商家核算提醒 |
| --- | --- | --- |
| 保价费 | 用户声明价值并购买保价服务。 | 高客单、易损品要把保价策略纳入毛利模型。 |
| 包装费/耗材费 | 纸箱、气泡膜、木架、泡沫箱、冰袋、保温袋等。 | 生鲜、易碎、大件品类不能只算运费。 |
| 上门取件费 | 部分场景或特殊地址可能加收。 | 商家仓发货通常看合同，个人寄件看下单页。 |
| 偏远地区附加费 | 偏远、县乡、港澳台、国际偏远地址可能加收。 | 运费模板、包邮区域和售后补寄都要分区。 |
| 超长超重附加费 | 单边过长、三边和过大、单件超重可能加收或转快运。 | 家具、家电、长条件、设备类 SKU 要提前判断。 |
| 退回/拒收/改址/拦截 | 逆向物流和操作可能产生费用。 | 售后成本要把拒收率、退货率和拦截失败率算进去。 |
| 特殊品类附加费 | 生鲜、冷链、易碎、液体、带电、国际清关等。 | 需要核验禁限寄、包装要求和赔付规则。 |

## 面单费

面单费是电商物流成本里容易漏算的一项。它指生成、购买、打印、使用快递面单相关的成本，不一定在账单里直接叫“面单费”。

常见表现形式：

| 形式 | 说明 | 常见位置 |
| --- | --- | --- |
| 电子面单余额/单号成本 | 商家通过快递公司、平台或服务商获取电子面单号。 | 平台电子面单后台、快递月结账户、服务商账户。 |
| 隐私面单服务费 | 平台为保护消费者信息提供手机号/地址脱敏面单。 | 淘宝、京东、拼多多、抖音等平台后台或服务账单。 |
| 打单软件/API 服务费 | 使用打单软件、ERP、开放接口批量取号和打印。 | 旺店通、快递100、打单软件、三方仓系统等。 |
| 面单纸和打印耗材 | 热敏纸、标签纸、碳带、打印头损耗。 | 仓库耗材、办公用品、仓配服务费。 |
| 打印设备折旧维护 | 热敏打印机、扫码枪、维护和备机成本。 | 仓库固定资产或低值易耗品。 |
| 代发/仓配内含成本 | 第三方仓可能把面单操作放进打包费、操作费或单票服务费。 | 仓配合同和月度账单。 |

商家可以用这个公式做单票核算：

```text
单票面单相关成本 =
电子面单/隐私面单服务费
+ 打单系统/API费
+ 面单纸/耗材
+ 打印设备折旧维护
+ 仓配/代发操作中包含的面单成本
```

面单费是否显性取决于业务模式。个人寄件通常感知不到这项；电商商家、仓配代发和多平台店群必须单独确认，否则会低估单票履约成本。

## 快递、快运、物流和同城配送

| 类型 | 典型场景 | 计费特点 |
| --- | --- | --- |
| 快递 | 小件、文件、电商包裹、个人寄件。 | 首重续重、体积重取大、门到门服务。 |
| 快运 | 30kg 以上、大件、多包裹、重货。 | 按重量/体积、最低一票价、提送货费和装卸费更常见。 |
| 零担物流 | 家具、设备、批量货、B2B 发货。 | 按公斤、方数、线路、装卸和送货方式计价。 |
| 同城即时配送 | 外卖、鲜花、文件、跑腿、小时达。 | 按距离、重量、时段、天气、加急和等待时间计价。 |
| 国际快递 | 跨境包裹、样品、文件。 | 体积重、燃油附加费、清关、关税、偏远费更重要。 |

## 商家单票履约成本模型

商家看物流成本时，不要只看快递账单上的运输费。更完整的公式是：

```text
单票履约成本 =
基础运费
+ 体积重/超重/超长差额
+ 面单相关成本
+ 包材和包装人工
+ 增值服务
+ 退换货逆向成本
+ 异常件损耗
+ 仓配操作费
```

SKU 级核算建议：

1. 对每个核心 SKU 做包装后实称，记录长、宽、高、实际重量、包材方案。
2. 用至少 30-100 单样本对比 ERP 出库重量、快递账单计费重量和实际支付金额。
3. 把泡货 SKU、易碎 SKU、生鲜保温 SKU、大件 SKU 单独建成本参数。
4. 把面单费、包材、仓配操作费和退货逆向成本分摊到单票。
5. 每月按快递公司、地区、重量段、SKU、退货原因做复盘，更新包邮门槛和运费模板。

## Agent 使用规则

当用户问“快递怎么收费”“为什么这个包裹这么贵”“商家物流成本怎么算”“面单费是什么”“这个品类适合快递还是快运”时，先按以下顺序回答：

1. 先声明具体价格和公司规则是动态的，需要以最新官方/合同/后台为准。
2. 判断业务类型：个人寄件、商家发货、仓配代发、快运大件、同城配送还是国际件。
3. 计算实际重量、体积重量和计费重量。
4. 拆基础运费、附加费、面单费、包材、逆向物流和异常成本。
5. 如果是商家经营问题，输出 SKU 级成本核算表和月度校准动作。
