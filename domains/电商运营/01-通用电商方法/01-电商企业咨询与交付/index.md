# 咨询逻辑：电商企业辅导与咨询交付体系

这组页面来自 XMind 脑图“咨询大纲”，主题不是单纯的电商运营技巧，而是一套面向电商企业主的咨询、谈单、带教、交付和经营认知引领体系。

它适合给 Agent 做三类事情：

- 诊断电商企业咨询场景：客户处在什么阶段、核心障碍是什么、应该如何建立共识。
- 生成咨询交付方案：开班、一阶、二阶、三阶、进阶、毕业分别应该抓什么。
- 调用经营模型：企业基因、产品基因、品类基因、价格基因、消费基因、增长拆解、战略框架和赚钱等级。
