# Agent 使用模板：电商企业咨询诊断

## 适用场景

当用户要求 Agent 诊断电商企业、生成咨询方案、设计陪跑交付节奏、处理客户满意度或拆解增长战略时，优先读取本组页面。

## 推荐读取顺序

1. 先读 [[domains/视觉制作/05-小红书风格AI生图/index|咨询逻辑：电商企业辅导与咨询交付体系]]。
2. 如果是谈单场景，读 [[03-谈单五大铁律|谈单五大铁律]]。
3. 如果是陪跑交付，读 [[06-咨询交付节奏|咨询交付节奏]]。
4. 如果客户不满意，读 [[05-满意度与预期管理|满意度与预期管理]]。
5. 如果要做战略判断，读 [[07-专业体系与基因模型|专业体系与基因模型]]、[[08-增长引领体系|增长引领体系]]、[[09-战略框架与赚钱等级|战略框架与赚钱等级]]。
6. 如果进入老板认知和长期经营，读 [[10-创始人人生引领|创始人人生引领]]。

## 诊断问题清单

- 企业当前属于工厂型、拿货型、代理型还是品牌型？
- 产品更接近标品、半标品还是非标品？
- 品类是强势品类、中度品类还是弱势品类？
- 当前增长瓶颈在访客数、客单价、利润率、利润额还是利润周期？
- 客户处于开班、一阶、二阶、三阶、进阶还是毕业阶段？
- 客户的不满来自事实结果不足，还是心理预期过高？
- 老板现在卡在执行、布局、组织、效率、资源还是心念？

```markdown
## 3. 预期管理

## 7. 下一步行动清单
```

## 注意事项

- 不要只给运营动作，要先建立共识。
- 不要被客户情绪化描述带走，要追问事实、依据、证据和数据。
- 不要把所有客户按同一套节奏交付，要根据阶段和状态区分。
- 不要把短期增长等同于长期确定性。
