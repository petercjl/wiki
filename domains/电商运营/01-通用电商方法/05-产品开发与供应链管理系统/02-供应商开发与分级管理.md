# 供应商开发与分级管理

供应链能力不是简单找到更低采购价，而是让产品能够稳定、快速、可控、可迭代地交付。供应商管理的目标，是把“凭感觉找货”变成“用候选池、评分表和分级机制管理供给”。

## 供应链四个能力

课程把供应链核心能力压缩成四个问题：

| 问题 | 含义 | 经营影响 |
| --- | --- | --- |
| 快不快 | 是否能快速响应爆款和市场变化 | 影响放量速度和补货节奏 |
| 稳不稳 | 品质、交期、服务是否稳定 | 影响复购、口碑和售后 |
| 有没有 | 货源是否稳定、是否容易断货 | 影响销售机会和平台数据 |
| 变不变 | 是否能跟上产品迭代 | 影响长期竞争力 |

扫地机器人从随机碰撞到自动清洗的迭代，说明供应链不能只复制旧款，还要跟上市场升级。

## 源头工厂、贸易商与定制能力

不要把所有供应商都简单分成“源头”和“中间商”。不同供应商适配不同业务。

| 类型 | 特征 | 优势 | 风险/限制 |
| --- | --- | --- | --- |
| 源头工厂 | 通常专注单一或有限品类 | 成本、工艺、定制和深度合作空间 | 产品线窄，未必适合多品类组合 |
| 贸易商 | 能整合多地、多品类资源 | 产品线丰富，适合多品类组合 | 定制和长期质量控制可能弱 |
| 产业带供应商 | 集中在特定品类和区域 | 定制、打样、协同效率较高 | 需要判断真实产能和稳定性 |

如果某供应商声称能集合全国各地食品并深加工，它不一定是源头工厂，更可能是资源整合型贸易商。贸易商并非天然不好，关键看业务是否需要多品类组合。

定制化服务通常更依赖工厂或产业带。没有生产能力或产业聚集基础的中间商，长期稳定提供专款专供的难度更高。

## 供应商开发七步

```text
信息收集
-> 初筛
-> 资质审核
-> 实地考察
-> 样品测试
-> 小批量试单
-> 全链路合作
```

### 1. 信息收集

先建立候选池，不要只看当前一个供应商。候选池至少记录：

- 供应商名称和联系人。
- 供应品类。
- 产品覆盖范围。
- 所在地和是否方便考察。
- 是否支持现货、定制、OEM/ODM。
- 发票、资质、合规能力。

课程中提到要识别不同供应商覆盖范围，例如 A 家有 ABCDE，B 家只有 AAEF。这个动作的本质是避免单一供应商依赖。

### 2. 初筛

初筛标准包括：

- 实力强。
- 货品齐全。
- 价格合理。
- 品质稳定。
- 距离近，方便实地考察。

距离近不是绝对条件，但对早期合作、打样、纠纷和深度协同有价值。

### 3. 资质审核

资质审核要看营业执照、生产许可、检测报告、发票能力、品牌授权、合规资料和历史合作记录。食品、母婴、医疗、消防、工业品等品类尤其不能只看报价。

### 4. 实地考察

即使采用一件代发，有机会也应实地走访。实地考察的价值不只是看厂房，还包括建立信任、理解产能、判断管理水平，并为后续议价、优先排产和深度合作打基础。

### 5. 样品测试

样品不是老板看一眼觉得可以就结束。要测试材质、尺寸、包装、破损、功能、用户反馈和页面表达是否一致。

### 6. 小批量试单

小批量试单验证的不只是产品好不好，还包括交期、破损率、客服反馈、退货原因、仓库处理、供应商响应速度和真实利润。

### 7. 全链路合作

全链路合作前，要明确价格阶梯、质量标准、售后责任、补货周期、产能保障、账期、授权权益和退出机制。

## 供应商评估八个维度

| 维度 | 要看什么 |
| --- | --- |
| 价格 | 是否有成本竞争力，而不是单次低价 |
| 品质 | 原料、工艺、成品稳定性 |
| 交付能力 | 交期、爆单应对、产能弹性 |
| 开发配合度 | 打样速度、改版响应、沟通质量 |
| 规模 | 产能、人员、设备、管理成熟度 |
| 合规性 | 资质、检测、品控、行业要求 |
| 发票能力 | 是否能提供合规票据和结算 |
| 风险控制 | 是否有替代方案、售后机制、异常处理 |

评分表不能一套权重打所有行业。灭火器等安全责任强的品类要重售后和合规；服饰更重交付及时性、面料稳定和款式迭代；食品更重资质、批次和效期。

## 供应商四级管理

| 等级 | 定义 | 管理方式 |
| --- | --- | --- |
| 战略供应商 | 高度协同、稳定配合、不愿流失 | 深度合作、长期规划、共同开发 |
| 核心供应商 | 主力供货，支撑核心产品 | 定期评估，重点维护 |
| 合格供应商 | 可供非核心 SKU 或补充品 | 保持合作，不做关键依赖 |
| 临时供应商 | 品质达标但合作不稳定 | 应急使用，限制关键业务 |

分级的作用不是贴标签，而是决定资源投入、议价方式、备选方案和风险控制。

## 跨境采购渠道差异

课程提到，小规模跨境卖家未必偏好 1688。原因包括单件运费高、退货机制复杂、中转仓时效等问题。

拼多多可能成为小规模跨境卖家的采购渠道，因为包邮、仅退款便捷、平台自动比价使采购价格持续下降。这个结论不应机械套用，但它提醒商家：供应链渠道选择要结合订单模型、退货机制、物流时效和售后成本，而不是只看“是不是源头”。

## 诊断问题

1. 供应商候选池是否覆盖多个来源？
2. 每个供应商覆盖哪些产品线？
3. 哪些供应商是真工厂，哪些是贸易商，哪些有产业带定制能力？
4. 核心产品是否依赖单一供应商？
5. 供应商是否按战略、核心、合格、临时分级？
6. 评分表是否按行业调整权重？
7. 关键供应商是否有备选方案？
8. 供应商的发票、资质、售后、合规能力是否足够？
