# 产品类型与爆款运营系统

本目录基于 **山峰组256期6-15课程**，把三段课程材料重构为一套电商运营判断系统：先识别企业和产品的优势，再按产品类型与行业结构选择指标和流量打法，接着按生命周期推进爆款递增，最后把方法落到具体店铺的链接、主图、客服、活动和执行排期。

## 使用边界

来源是三份 AI 整理后的课程纪要，不是逐字原文。正式页保留明确出现的知识点、案例和数字锚点，并在此基础上做结构化理解。涉及具体平台工具、H2/H3、IPA、流量金卡、悟空推广等名词时，需要结合当前后台复核。

## 阅读顺序

1. [[01-经营定位与核心优势|经营定位与核心优势]]：先明确企业靠什么赢，而不是直接学打法。
2. [[02-产品类型与行业模型|产品类型与行业模型]]：判断标品、非标品、半标品、高频、高客单、ToB 的不同逻辑。
3. [[03-产品生命周期与阶段指标|产品生命周期与阶段指标]]：按新品期、成长期、爆款期、衰退期切换目标。
4. [[04-爆款递增与数据化放大|爆款递增与数据化放大]]：用 S/A/B 产品梯队、潜力款挖掘、预算递增和复盘把爆款放大。
5. [[05-ToB定制与高客单信任业务|ToB 定制与高客单信任业务]]：处理询单、客服、信任和长决策链。
6. [[06-电锅炉壁挂炉店铺执行案例|电锅炉壁挂炉店铺执行案例]]：把理论落到关键词、主图测试、客服、竞品复制、顺手买一件和排期。
7. [[07-Agent使用模板：产品类型与爆款运营诊断|Agent 使用模板：产品类型与爆款运营诊断]]：给未来 Agent 做店铺诊断和运营方案生成的入口。

## 核心判断链

```text
企业优势 -> 产品类型 -> 行业结构 -> 生命周期阶段 -> 指标选择 -> 流量端口 -> 组织执行 -> 复盘迭代
```

如果只看最后的投放动作，容易把标品打法用到非标品，把新品期用爆款期指标考核，把 ToB 定制当作静默成交商品。课程真正有价值的地方，是把“产品是什么”和“阶段在哪里”放到“怎么推广”之前。
