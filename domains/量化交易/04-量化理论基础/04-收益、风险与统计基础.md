# 收益、风险与统计基础

量化交易用统计语言描述收益和风险。Empyrical、ffn、Pyfolio 这类工具的存在说明：策略研究不能只看最终收益率，而要把收益路径转成可比较的指标。

## 收益口径

| 口径 | 含义 | 用途 |
| --- | --- | --- |
| 简单收益 | `(P_t - P_{t-1}) / P_{t-1}` | 日常绩效展示。 |
| 对数收益 | `log(P_t / P_{t-1})` | 时间加总和统计建模更方便。 |
| 累计收益 | 净值从起点到终点的变化 | 看总结果。 |
| 年化收益 | 把不同周期收益换算到年 | 比较不同回测区间。 |
| 超额收益 | 策略收益减基准收益 | 判断是否超过被动持有。 |
| 主动收益 | 相对基准或风险模型的偏离 | 归因和基金评价。 |

收益必须和频率匹配。日收益、周收益、月收益不能混用同一个波动率或夏普公式。

## 风险口径

| 风险 | 含义 | 适用场景 |
| --- | --- | --- |
| 波动率 | 收益标准差 | 衡量净值抖动。 |
| 最大回撤 | 从峰值到谷底的最大跌幅 | 衡量投资者最痛路径。 |
| 下行波动 | 只统计负收益或低于目标的波动 | 更关注亏损方向。 |
| VaR | 给定置信度下的潜在损失分位数 | 风险限额。 |
| CVaR | 超过 VaR 后的平均尾部损失 | 尾部风险。 |
| Beta | 对市场或因子的敏感度 | 市场暴露和对冲。 |
| 跟踪误差 | 相对基准的波动 | 指数增强和主动管理。 |

Riskfolio-Lib 支持多种风险度量，包括方差、下行风险、VaR、CVaR、EVaR、RLVaR 等。对个人策略研究而言，不需要一开始用最复杂的风险度量，但要理解：不同风险指标对应不同损失观。

## 常见绩效指标

| 指标 | 公式直觉 | 解释边界 |
| --- | --- | --- |
| Sharpe | 超额收益 / 波动率 | 假设收益分布相对稳定；对尾部风险不充分。 |
| Sortino | 超额收益 / 下行波动 | 更重视亏损波动。 |
| Calmar | 年化收益 / 最大回撤 | 适合看趋势策略的回撤效率。 |
| Alpha/Beta | 相对基准的截距和斜率 | 依赖基准选择。 |
| 信息比率 | 主动收益 / 跟踪误差 | 衡量相对基准的主动能力。 |
| 胜率 | 盈利交易次数占比 | 必须结合盈亏比。 |
| 盈亏比 | 平均盈利 / 平均亏损 | 必须结合胜率和成本。 |

## 统计直觉

量化统计中最容易误解的点：

- 均值不稳定：历史平均收益可能被少数极端日支配。
- 方差不固定：市场波动会聚集，平静期后可能突然放大。
- 相关性会变化：危机时资产相关性可能上升。
- 样本量重要：一年回测的高 Sharpe 不可靠。
- 分布非正态：金融收益有尖峰厚尾，极端亏损比正态假设更常见。
- 指标不是目标：最大化 Sharpe 不等于可实盘。

## Agent 使用模板

评价一个回测时，先输出：

1. 收益：累计、年化、超额。
2. 风险：波动、最大回撤、回撤持续时间。
3. 风险调整：Sharpe、Sortino、Calmar 或信息比率。
4. 路径：年度/月度收益、最大回撤区间。
5. 交易：换手、成本、滑点、订单失败。
6. 稳健性：分市场环境、成本敏感性、样本外。
