## 量化理论基础

- [[domains/量化交易/04-量化理论基础/index|量化理论基础]]：从 GitHub 开源量化项目中沉淀完整基础框架，覆盖市场、数据、信号、组合、执行、回测、评估和实验纪律。
- [[domains/量化交易/04-量化理论基础/01-量化交易全景与研究对象|量化交易全景与研究对象]]：把量化交易理解为从数据到信号、组合、执行、回测和监控的系统工程。
- [[domains/量化交易/04-量化理论基础/02-市场、资产与交易制度|市场、资产与交易制度]]：梳理股票、ETF、期货、期权、债券、外汇、商品等资产的制度和策略边界。
- [[domains/量化交易/04-量化理论基础/03-数据体系与特征工程|数据体系与特征工程]]：说明行情、基本面、宏观、文本、盘口等数据如何点时正确地变成特征。
- [[domains/量化交易/04-量化理论基础/04-收益、风险与统计基础|收益、风险与统计基础]]：建立收益、波动、回撤、VaR/CVaR、Sharpe、Sortino、Calmar 等统计口径。
- [[domains/量化交易/04-量化理论基础/05-策略类型与收益假设|策略类型与收益假设]]：按趋势、动量、均值回复、价值、事件、套利、机器学习等收益假设分类策略。
- [[domains/量化交易/04-量化理论基础/06-因子、信号与Alpha研究|因子、信号与 Alpha 研究]]：用收益分组、IC、换手和分层分析验证 alpha 信号。
- [[domains/量化交易/04-量化理论基础/07-组合构建与仓位管理|组合构建与仓位管理]]：覆盖等权、信号加权、风险预算、均值方差、HRP、约束和仓位模型。
- [[domains/量化交易/04-量化理论基础/08-交易执行、成本与市场冲击|交易执行、成本与市场冲击]]：解释订单、手续费、滑点、市场冲击、容量和执行取舍。
- [[domains/量化交易/04-量化理论基础/09-回测系统与真实性检查|回测系统与真实性检查]]：建立事件驱动、账户、订单、成本、交易制度和数据可见性的回测检查清单。
- [[domains/量化交易/04-量化理论基础/10-绩效评估、归因与报告|绩效评估、归因与报告]]：形成收益、风险、交易、持仓、归因和稳健性的回测报告结构。
- [[domains/量化交易/04-量化理论基础/11-过拟合、数据泄漏与样本外验证|过拟合、数据泄漏与样本外验证]]：覆盖未来函数、幸存者偏差、参数搜索、purge、embargo 和 walk-forward。
- [[domains/量化交易/04-量化理论基础/12-量化研究工作流与实验纪律|量化研究工作流与实验纪律]]：定义假设、实验记录、版本管理、失败归档和策略决策流程。
- [[domains/量化交易/04-量化理论基础/13-市场状态识别与策略失效|市场状态识别与策略失效]]：把固定策略在某些年份失效的问题转成市场状态、分布漂移和策略收益假设边界的诊断框架。
- [[domains/量化交易/04-量化理论基础/14-状态自适应策略与动态资产配置|状态自适应策略与动态资产配置]]：整理 regime filter、strategy router、expert models、dynamic allocation、no-trade regime 和现金替代资产。
- [[domains/量化交易/04-量化理论基础/15-策略失效监控与滚动验证|策略失效监控与滚动验证]]：建立主评估周期、分年度/分状态拆解、walk-forward、状态策略验证和失效决策树。
