# 杉居品牌风格系统

本页用品牌策略方法沉淀杉居的心向感知、ABC 视觉分层和边界规则。

## 心向感知

### 品牌名称

“杉居”天然指向自然与居家：

- 杉：木、自然、清新、浅木、温和。
- 居：家、日常、空间、陪伴、生活秩序。

这个名字不适合高冷设计师路线、强促销路线或粗犷户外路线。它更适合“自然、温和、可托付的料理生活感”。

### 价值短句

`live gently.` 是语气系统，不只是英文装饰。中文表达应围绕：

- 温柔地生活。
- 把生活放轻一点。
- 有光，有厨房，有猫，也有你。
- 每一份热爱，都被安心托付。

### 品类情绪

杉居围绕厨房工具系列产品建立品牌，包括烧烤盘、调味罐、烘焙工具、收纳和餐厨小物。基础情绪不是“高级”，而是：

- 干净
- 安心
- 温柔
- 松弛
- 有陪伴
- 可被托付

室内厨房是高频入口，户外烧烤、露台料理、庭院和野餐分享是同一套工具和情绪的外延。

### 形象气质

> 明亮、轻柔、会照顾料理细节的生活陪伴者。

杉居不是：

- 高冷极简。
- 日式侘寂。
- 奶油风网红家。
- 黑白灰高级公寓。
- 强烟火气家庭厨房。
- 粗犷露营烧烤品牌。
- 强宠物品牌。

## ABC 视觉分层

### A 层：品牌态度

A 层表达轻生活态度：

> 生活本该这样温柔。  
> 光与厨房，有猫，有你，有生活。

A 层画面可使用：

- 明亮厨房远景。
- 露台或庭院轻料理远景。
- 人物与猫同处一个空间。
- 品牌名和短句在大留白区域轻声出现。
- 不放强卖点、促销、大标题压迫。

### B 层：使用场景

B 层是杉居最强的层级，负责把厨房工具放回用户生活：

- 清晨厨房：准备早餐或烘焙。
- 午后轻烘焙：面包出炉，猫在旁边等待。
- 傍晚露台：烧烤盘上是轻煎蔬菜、面包或肉类，调味罐有序摆在浅木桌上。
- 周末庭院/野餐：厨房工具被带到户外，料理和分享仍然保持清爽秩序。
- 厨房收纳：台面干净，工具和食物被温柔组织。

B 层的关键词是关系：人与厨房、人与户外料理场、人与猫、人与食物、产品与这些关系。

### C 层：功能卖点

C 层必须被 B 层包住，不要做成参数表。

| 功能事实 | 直接卖点 | 杉居式表达 |
| --- | --- | --- |
| 烧烤盘适合多场景 | 室内外两用 | 从厨房到露台，一盘接住刚好的烟火气 |
| 调味罐便携/密封 | 户外可带 | 把喜欢的味道，整齐带到周末的风里 |
| 收纳容量 | 大容量 | 把台面还给光，把小物安放好 |
| 防烫/隔热 | 隔热耐高温 | 怡到好处的温度，安心托付每一份热爱 |
| 易清洁 | 好擦洗 | 烘焙后的厨房，也能很快回到清爽 |
| 天然材质 | 环保自然 | 和浅木、面包、猫一起，留在家的温柔质感里 |

## 场景到情景

场景层：

- 在明亮厨房做轻烘焙。
- 从烤箱取出刚出炉的面包。
- 在露台用烧烤盘做一顿轻松晚餐。
- 把调味罐带到庭院、野餐桌或户外烧烤场景。

情景层：

- 我把普通日子照顾得有一点温柔。
- 我的厨房工具不只留在厨房，也陪我把周末搬到户外。
- 户外也可以不粗糙，烧烤也可以干净、有序、温柔。
- 简单的幸福，不需要很宏大。
