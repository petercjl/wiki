# 摄影、包装与接触点标准

## 问题定义

视觉标准不能只停留在 LOGO、色彩和字体。电商中真正影响消费者感知的，是图片素材、产品细节、包装体验和所有品牌接触点。

原书第 6.8 至 6.12 节重点讨论：品牌摄影、产品摄影、品牌包装、品牌故事和品牌接触点。

## 品牌摄影标准

品牌摄影负责表达品牌生活方式和情绪。它不是“拍得漂亮”，而是让目标客户产生代入感，觉得品牌懂自己。

### 标准项

| 标准项 | 需要明确 |
| --- | --- |
| 目标人群 | 年龄、性别、城市、消费层级、审美偏好 |
| 模特 | 气质、妆发、动作、表情、是否固定或可替换 |
| 场景 | 室内/室外、城市/自然、生活方式、文化背景 |
| 道具 | 可重复出现的品牌元素和生活物件 |
| 光线 | 自然光、棚拍、硬光、柔光、明暗关系 |
| 构图 | 留白、主体位置、视角、景深 |
| 色调 | 主色、环境色、后期调色 |
| 文案策划 | 每组照片要表达的故事和消费理由 |

### 判断标准

- 老客户能否一眼识别这是你的品牌？
- 即使换摄影师、换模特，风格是否仍一致？
- 图片是否让消费者幻想自己进入这个场景？
- 摄影是否支撑品牌定位，而不是只为好看？

## 产品摄影标准

产品摄影负责细节展示和购买证据。它通常出现在主图后几张、详情页细节、规格说明和对比图中。

### 标准项

- 拍摄角度；
- 光线方向；
- 背景；
- 尺寸比例；
- 产品属性表；
- 材质细节；
- 功能细节；
- 修片原则；
- 同品类 SKU 的统一角度。

产品摄影最怕“乱”：同一类产品角度不同、光线不同、修片风格不同、属性信息不统一，会削弱专业感。

## 品牌包装标准

包装是线上购物中消费者收到货后的第一体验。它影响客单价感知、惊喜感、复购和评价。

### 包装标准包含

1. 外包装识别：品牌名、LOGO、主色、材质。
2. 开箱路径：打开顺序是否顺畅，是否有仪式感。
3. 内部结构：防护、分层、收纳、赠品位置。
4. 感官体验：重量、触感、气味、声音。
5. 赠品策略：是否制造超值感，而非低质堆料。
6. 售后卡/说明书：是否继续重复品牌记忆。

原书提到“价格心理物理学”：消费者收到商品时，重量感、包装质感和赠品会影响“值不值”的感受。正式使用时应理解为体验设计，而不是简单加重包装。

## 品牌故事标准

品牌故事有两个目的：

1. 让客户相信品牌；
2. 让客户相信自己用了产品也能实现理想状态。

好的故事不是品牌自传，而是把卖点、信任和消费者愿望合并起来。

故事检查问题：

- 是否和产品真实优势有关？
- 是否和目标客户愿望有关？
- 是否能自然带出卖点？
- 是否比参数说明更容易被记住？
- 是否能被短视频、详情页、客服、包装重复使用？

## 品牌接触点标准

品牌接触点是一切消费者能看到品牌的场景。原书列举了店铺类、流量类、活动专题类、SNS、线下等触点。

今天可以扩展为：

- 搜索列表；
- 推荐流；
- 广告素材；
- 店铺首页；
- 商品详情页；
- 直播间；
- 短视频；
- 私域社群；
- 包裹和售后卡；
- 客服话术；
- 线下展陈；
- AI 生成素材库。

## Agent 使用提示

Agent 做视觉落地方案时，应把摄影、包装和触点纳入同一张标准表，不要只改页面：

1. 品牌摄影是否有固定场景和符号；
2. 产品摄影是否有统一细节标准；
3. 包装是否支撑价格带和复购体验；
4. 接触点是否重复同一套品牌资产；
5. AI 生图是否会破坏既有摄影标准。
