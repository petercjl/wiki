# 商业短视频摄影与运镜语法

本页不是电影术语百科，而是给 AI 商业短视频使用的摄影语法速查表。使用顺序是：先按 [[domains/视觉制作/06-AI视频/03-导演式镜头设计方法|导演式镜头设计方法]] 判断镜头任务，再从本页选择景别、机位、构图、运镜、光线和剪辑关系。

## 景别语法

| 中文 | 英文 prompt 可用词 | 主要功能 | 商业短视频用法 |
| --- | --- | --- | --- |
| 大远景/极远景 | extreme wide shot, extreme long shot | 建立世界、规模、孤独、距离 | 城市、山野、厂房、门店、家庭空间开场 |
| 远景/全景 | wide shot, long shot, full shot | 人与环境关系、动作完整性 | 展示家居空间、通勤、运动、搬家、户外场景 |
| 中远景 | medium wide shot, cowboy shot | 人物身体和环境兼顾 | 职场走廊、厨房操作、人与产品同框 |
| 中景 | medium shot | 日常动作、对话、任务推进 | 多数生活流镜头的基础景别 |
| 中近景 | medium close-up | 表情和动作平衡 | 情绪广告、人物思考、轻微表情变化 |
| 近景/特写 | close-up | 情绪、反应、关键物件 | 人脸、手、产品按钮、杯口、屏幕 |
| 大特写 | extreme close-up, macro close-up | 细节、感官、决定性瞬间 | 材质、液体、屏幕、眼神、机械结构 |
| 插入镜头 | insert shot | 交代关键物件或动作 | 手机取景界面、手扣安全带、杯子放下 |
| 主观镜头 | POV shot | 代入人物视角 | 手机拍照、开门、拿起产品、看孩子 |
| 过肩镜头 | over-the-shoulder shot | 关系、对话、观看对象 | 母亲看孩子、客户看方案、用户看产品 |

### Katz 补充：景别不是大小，而是观众距离

景别变化同时改变三件事：

- 空间理解：观众是否知道人物在哪里。
- 情绪距离：观众和人物是旁观、靠近，还是侵入隐私。
- 剪辑逻辑：观众能否把近景/插入镜头理解成前一个广阔空间的一部分。

AI 视频分镜不要只写“近景更有质感”。更稳的写法是：

```text
远景：建立世界和任务环境
中景：保留身体动作和人物关系
近景：进入表情、手部和产品动作
插入镜头：交代关键物件、按钮、屏幕、触点或证据
```

如果一条短片全是中近景，观众会失去空间和节奏；如果全是远景，观众会失去亲密和产品证据。

## 机位语法

| 机位 | 情绪/意义 | 适合场景 | AI 生成注意 |
| --- | --- | --- | --- |
| 平视 | 真实、自然、共情 | 生活方式、人物日常、亲子关系 | 最稳定，商业片常用 |
| 低机位 | 高级、力量、压迫、仰视 | 汽车、家电、建筑、高端产品 | 不要过低，否则容易夸张 |
| 高机位 | 孤独、秩序、脆弱、被观察 | 办公室、餐桌、家庭全局 | 适合建立空间关系 |
| 俯拍 | 图形感、秩序、桌面信息 | 食品、手作、办公桌、儿童画画 | 产品容易变平，要控制透视 |
| 仰拍 | 权威、宏大、陌生感 | 高楼、产品英雄镜头、人物力量感 | 人脸和建筑易畸变，少用 |
| 侧面 | 克制、内心、过渡 | 车窗、窗边、独处 | 适合情绪片 |
| 背影 | 距离、进入、未说明的故事 | 城市、通勤、窗边、离开 | 常用于开场或身份悬念 |
| 倾斜机位 | 不稳定、焦虑、失衡 | 悬疑、危机、强冲突 | 商业广告慎用 |

## 构图语法

| 构图 | 功能 | 商业短视频用法 |
| --- | --- | --- |
| 三分法 | 稳定但不死板 | 人物看向画面留白处，暗示思考和方向 |
| 中心构图 | 仪式感、产品权威、秩序 | 产品英雄镜头、品牌收束 |
| 对称构图 | 高级、秩序、空间感 | 酒店、家居、汽车内饰、城市建筑 |
| 框中框 | 被空间包裹、观察、隔离 | 门框、窗框、电梯、车窗 |
| 前景遮挡 | 真实感、偷窥感、空间深度 | 办公室植物、门框、厨房器具前景 |
| 负空间 | 孤独、安静、呼吸感 | 高端产品、人物独处、片尾情绪 |
| 引导线 | 方向、速度、空间纵深 | 道路、走廊、桌面、城市街道 |
| 浅景深 | 注意力集中、广告质感 | 人物表情、产品细节 |
| 深焦 | 世界完整、环境信息多 | 门店、工厂、家庭全景 |

## 运镜语法

| 运镜 | 英文 prompt 可用词 | 叙事功能 | 适合用法 |
| --- | --- | --- | --- |
| 静止 | static shot, locked-off camera | 克制、观察、真实 | 情绪停顿、产品质感、家庭安静 |
| 平摇 | pan left/right | 扫描空间、发现关系 | 从环境扫到人物，从产品扫到使用者 |
| 俯仰摇 | tilt up/down | 揭示高度、从细节到整体 | 从手部产品上摇到人物表情 |
| 推近 | slow dolly in, slow push in | 靠近内心、强调发现 | 人物意识到、情绪升起、产品揭示 |
| 拉远 | slow dolly out, pull back | 留白、结束、人物回到世界 | 片尾、孤独、释然 |
| 跟拍 | tracking shot, follow shot | 行动、任务、通勤、生活流 | 人物走路、家庭活动、运动产品 |
| 横移 | lateral tracking, truck left/right | 空间展开、平行观察 | 办公室、厨房、货架、街道 |
| 升降 | crane up/down, boom up/down | 层级变化、开阔感 | 城市、汽车、空间揭示 |
| 环绕 | arc shot, orbit shot | 人物处在情绪中心 | 概念片可用，生活广告少量使用 |
| 手持 | handheld camera | 真实、紧张、即时 | 纪实感、冲突、运动场景 |
| 变焦 | zoom in/out | 镜头焦距变化、强调或抽离 | AI 中容易假，谨慎使用 |
| 推拉变焦 | dolly zoom | 心理失衡、强烈戏剧化 | 商业短视频除非有明确冲突，否则不用 |

## 光线和色彩

| 类型 | 观感 | 用法 |
| --- | --- | --- |
| 自然窗光 | 真实、柔和、生活感 | 家庭、办公、女性生活方式 |
| 黄昏光 | 时间流逝、记忆感、高级感 | 情绪广告、城市、家庭收束 |
| 冷色办公光 | 压力、效率、距离 | 职场、高端商务、科技产品 |
| 暖色家庭光 | 亲密、安全、放松 | 亲子、家居、食品、情绪转折 |
| 逆光/轮廓光 | 高级、距离、神秘 | 产品英雄镜头、人物侧影 |
| 低调光 | 戏剧、质感、悬念 | 高端科技、香水、珠宝 |
| 高调光 | 清爽、干净、亲和 | 母婴、食品、日化、小红书风格 |

商业短视频可以用冷暖对比构建叙事：冷光表达外部任务和压力，暖光表达家庭、亲密和解决之后的状态。

## 剪辑与镜头关系

| 关系 | 功能 | 示例 |
| --- | --- | --- |
| 建立镜头 -> 人物镜头 | 先给世界，再给人 | 城市黄昏 -> 女性背影 |
| 人物视线 -> 被看对象 | 建立关系和动机 | 母亲看向地毯 -> 孩子画画 |
| 动作 -> 插入镜头 | 让产品自然入场 | 拿起手机 -> 手机取景界面 |
| 冷色 -> 暖色 | 表示身份和情绪切换 | 办公室 -> 家庭客厅 |
| 远景 -> 特写 | 从世界进入内心 | 街头人群 -> 眼神 |
| 静止 -> 运动 | 从压抑到行动 | 窗边静止 -> 跟拍出门 |
| 运动 -> 静止 | 从行动到情绪落点 | 跟拍回家 -> 坐下看孩子 |

## 连续性与轴线

Katz 的连续性方法提醒：轴线不是教条，而是为了让观众保持空间方向。AI 视频虽然常常按单镜头生成，但剪辑时仍然会遇到方向混乱。

| 规则 | 用途 | AI 视频执行方式 |
| --- | --- | --- |
| line of action / 180-degree rule | 保持人物左右、行进方向和视线方向一致 | 分镜表记录人物 A/B 的站位、视线方向、运动方向 |
| pivot shot | 用一个转头、转身、看向新对象的镜头建立新轴线 | 需要换方向时，先加“人物看向新对象”的桥接镜头 |
| reestablishing shot | 回到熟悉空间，修复观众方向感 | 多场景或多人镜头后，用中远景/全景重新交代空间 |
| triangle system | 对话或人物关系的基础 coverage | 准备 master two-shot、OTS、POV single、profile 等组合 |

商业短片通常不需要完整电影式 coverage，但需要知道每个镜头在空间中的方位。尤其是汽车行驶、人物走向产品、母亲看孩子、用户看屏幕这类动作，方向一乱，情绪就会断。

## 场面调度与画面深度

Katz 把前景、中景、背景视为导演调度的三个层。它们不只是“画面有层次”，而是可以替代剪辑、转移注意、隔离人物或连接概念。

| 深度用法 | 作用 | AI 视频例子 |
| --- | --- | --- |
| 前景遮挡 | 真实感、窥视感、人物被环境包裹 | 办公室植物/门框遮挡人物侧脸 |
| 前景产品 + 背景人物 | 产品与生活关系同框 | 桌面杯子前景，疲惫用户在背景放松 |
| 前景人物 + 背景关系 | 让观众贴近一个人的心理 | 母亲在前景，孩子在暖光背景玩耍 |
| 中景相遇 | 让关系从分离变为连接 | 两个角色从画面两侧进入同一中景 |
| 背景信息 | 解释世界、阶层、行业或时间 | 城市天际线、工厂产线、家庭杂物 |

深度调度不等于必须 deep focus。可以让前景或背景略虚，只要观众知道它承担什么叙事功能。

## 移动摄影的任务表

| 运镜 | Katz 式任务 | AI 视频使用边界 |
| --- | --- | --- |
| pan / tilt | 覆盖超出固定画框的空间、跟随动作、连接两个兴趣点、暗示逻辑关系 | 适合从产品扫到使用者、从环境扫到人物；不要无意义扫全场 |
| cross-pan | 在两个移动主体之间转移注意力 | 适合活动现场、多人生活流；AI 中要简化主体数量 |
| crane / boom | 从整体进入细节，或从细节回到整体；制造“进入世界”的感觉 | 适合开场、收束、空间揭示；不要只为宏大感使用 |
| tracking | 跟随人物行动、探索空间、连接故事点 | 适合通勤、运动、逛街、家庭动线；要标注人物和相机的相对速度 |
| tracking in/out | 改变观众与主体的亲密程度 | 推近代表靠近内心/发现；拉远代表回到世界/结束/疏离 |

移动镜头要写“为什么动”。如果原因只有“更电影感”，先改成静止镜头。

## 商业品类的常用镜头策略

| 品类 | 镜头策略 |
| --- | --- |
| 高端手机/相机 | 先生活记忆，后产品取景；多用城市、家庭、手部、屏幕插入镜头 |
| 汽车 | 空间关系、家庭动作、道路跟拍、车内外切换；功能不要只做参数展示 |
| 服饰 | 人物状态和身体动作优先；用生活场景反推产品气质 |
| 食品饮品 | 感官特写 + 情绪场景；不要只拍液体和包装 |
| 家居家电 | 空间秩序变化；使用前后的生活摩擦要能看见 |
| 礼品/珠宝/香水 | 关系和仪式感；产品是关系的承载物 |

## AI 生成时的反坑

- 不要让所有镜头都是 shallow depth of field close-up。
- 不要每个镜头都 cinematic camera movement。
- 不要把 pan、tilt、dolly、zoom、orbit 全写进一个 prompt。
- 不要只写“电影感”，要写景别、机位、运动方向和速度。
- 不要在同一镜头里同时要求复杂手部动作和精准产品结构。
- 不要把产品特写放太早，除非产品本身就是悬念。
