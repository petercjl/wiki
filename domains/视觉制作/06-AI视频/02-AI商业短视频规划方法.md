# AI商业短视频规划方法

AI 商业短视频的优势不在于把硬广做得更快，而在于把产品卖点翻译成更具体的人、场景、冲突、情绪和世界。拿到产品后，第一步不是写 prompt，也不是列镜头，而是判断这个产品应该被观众如何感知。

罐头 Visuals 的 7 个案例共同说明：好广告不是“产品展示 + 旁白卖点”，而是先找到产品在人生活中的角色，再让产品自然进入一个可相信的故事。

## 核心流程

```text
产品资料
-> 产品/主题诊断
-> 问题类型分类
-> 检索相似案例
-> 卖点翻译成叙事命题
-> 设计人物、场景、冲突和产品入场
-> 分镜图片确认
-> 图生视频和剪辑取舍
-> 输出可复盘的案例资产
```

这条流程里，AI 生成只位于后半段。前半段的产品判断、案例类比和叙事命题，决定短视频是否有商业价值。

## 第一步：判断产品要解决什么感知问题

不要把产品直接归类为“手机、汽车、饮品、服装”。短视频规划更应该按“传播问题”分类：

| 问题类型 | 适合案例 | 判断问题 |
| --- | --- | --- |
| 高客单情绪价值 | [[domains/视觉制作/06-AI视频/90-案例库/01-OPPO-Find-X9-Ultra：把影像能力翻译成记忆保存|OPPO Find X9 Ultra]] | 用户为什么愿意为参数之外的意义付费？ |
| 功能明确但表达同质化 | [[domains/视觉制作/06-AI视频/90-案例库/02-五菱星光560：功能卖点转生活冲突广告|五菱星光560]] | 功能解决的是哪一个日常摩擦？ |
| 节日和关系情绪 | [[domains/视觉制作/06-AI视频/90-案例库/03-母亲节广告：多人物群像与生活动作表达|母亲节广告]] | 节日情绪能否用生活动作表达，而不是口号表达？ |
| 世界观和概念感 | [[domains/视觉制作/06-AI视频/90-案例库/04-意识残留：世界观预告片与基调图|意识残留]] | 有没有值得展开的冲突、规则和人物动机？ |
| 服饰和生活方式 | [[domains/视觉制作/06-AI视频/90-案例库/05-美国运动品牌：产品气质先行与双重生活叙事|美国运动品牌]] | 产品穿在谁身上，代表什么生活状态？ |
| 情绪放松和软广 | [[domains/视觉制作/06-AI视频/90-案例库/06-低度果酒：压力铺垫与自我批准的轻松|低度果酒]] | 产品给用户一个什么心理许可？ |
| 传统文化和节日资产 | [[domains/视觉制作/06-AI视频/90-案例库/07-端午安康：传统节日的时间纵深与史料美术|端午安康]] | 能否跳出符号堆砌，写出时间、人物和文化关系？ |

这个分类比品类分类更重要。一个保温杯可能是功能问题，也可能是亲情问题；一件衣服可能是服饰广告，也可能是自我表达广告；一个地方特产可能是美食展示，也可能是地域文化短片。

## 第二步：把卖点翻译成故事命题

短视频里的卖点不能停留在商家语言。它要经历四次翻译：

```text
产品事实 -> 用户场景 -> 人物任务 -> 故事命题
```

示例：

| 产品事实 | 用户场景 | 人物任务 | 故事命题 |
| --- | --- | --- | --- |
| 手机影像能力强 | 家人、旅行、成长瞬间 | 留住以后会舍不得忘的画面 | 影像能力是记忆保存能力 |
| 汽车空间大 | 小家庭搬家、带娃、兼顾工作 | 把凌乱生活重新装进秩序 | 大空间不是参数，是家庭摩擦的解决方案 |
| 低度果酒 | 下班后疲惫、社交耗尽 | 给自己一个不用解释的放松时刻 | 微醺是一种自我批准 |
| 传统节日 | 现代人只记得表层符号 | 从历史和家庭关系中重新看节日 | 节日不是符号，而是代代相传的情感秩序 |

这里可以调用 [[domains/视觉制作/01-电商视觉基础/03-卖点转炸点方法|卖点转炸点方法]] 和 [[domains/品牌策略/03-产品战略与大单品/03-品牌产品营销与拓品策略/10-内容营销资产化与内容中台|内容营销资产化与内容中台]]。短视频的命题本质上也是内容根：谁、什么场景、什么任务、什么情绪、什么产品价值。

## 第三步：设计人物，而不是设计商品展示

案例里的产品都不是孤零零出现的。手机在父女、老人、旅行和成长瞬间里出现；汽车在夫妻争吵、搬家、夜晚工作和家庭秩序里出现；运动服在家庭生活和运动释放之间出现。

人物设计要回答：

- 这个人为什么需要这个产品？
- 他原本的生活里有什么缺口、摩擦或愿望？
- 他和产品之间是工具关系、身份关系、情绪关系，还是文化关系？
- 产品出现前后，人物状态有没有变化？

如果人物没有变化，短视频就容易变成产品摆拍。如果产品不能改变人物状态，它就只是道具。

## 第四步：用冲突组织镜头

好看的镜头不等于有效镜头。镜头要服务下面至少一种功能：

| 镜头功能 | 作用 | 案例启发 |
| --- | --- | --- |
| 建立人物 | 让观众知道谁在经历这件事 | OPPO、运动品牌 |
| 建立处境 | 让观众理解生活里的问题 | 五菱、低度果酒 |
| 建立情绪 | 让观众进入人物感受 | 母亲节、低度果酒 |
| 建立世界规则 | 让观众相信这个世界 | 意识残留、端午安康 |
| 产品入场 | 让产品成为解决、连接或象征 | OPPO、五菱、运动品牌 |
| 收束命题 | 让观众记住一句核心意义 | 低度果酒、端午安康 |

规划镜头时，要先写每个镜头承担的任务，再写画面。不能先追求漂亮画面，再事后解释它有什么用。

## 第五步：先做分镜图片，再做视频

这些案例反复出现一个规律：图像阶段不是临时素材，而是导演工作台。分镜图片用于确认人物、场景、构图、情绪、产品一致性和美术方向。

AI 视频生成前至少要准备：

| 资产 | 用途 |
| --- | --- |
| 人物设定图 | 保证角色一致性，尤其是多镜头、多场景广告 |
| 产品参考图 | 保证商品外观、比例、颜色和关键结构不乱 |
| 场景参考图 | 保证画面不是泛泛的“好看场景” |
| 基调图 | 确认项目的整体气质和冲突中心 |
| 分镜图 | 把故事拆成可生成、可剪辑的画面单位 |
| 史料/证据包 | 用于文化、历史、专业产品或强信任品类 |

提示词不是创意本身。提示词是把已经想清楚的镜头表达出来。

## 第六步：把产品入场设计成答案，而不是插播

硬广的常见问题，是产品突然占满画面、字幕开始报卖点。更好的方式是让产品承担叙事功能：

- OPPO：产品是记忆保存的能力。
- 五菱：产品是家庭摩擦的解决方案。
- 母亲节：品牌或主题应退到情绪背后，让生活动作成为表达。
- 意识残留：产品若适合概念片，应成为世界规则的一部分。
- 运动品牌：产品是人物两种生活状态之间的连接。
- 低度果酒：产品是用户允许自己放松的仪式。
- 端午安康：产品或品牌如果进入，应成为文化传承中的现代连接点。

检查标准：把产品拿掉后，故事是否塌掉？如果不塌，说明产品只是贴片；如果塌掉，说明产品已经进入叙事结构。

## 第七步：导演式取舍

AI 会生成很多漂亮镜头，但广告不需要每个漂亮镜头。取舍标准是：

- 是否推进人物任务？
- 是否强化产品价值？
- 是否让观众更理解冲突？
- 是否建立可信世界？
- 是否和前后镜头形成节奏？
- 是否只是炫技？

越是 AI 生成，越需要导演思维。因为生成成本下降后，最大的问题不是素材不够，而是素材过多、风格过散、故事变松。

## Agent 化时的主线

未来把这套方法做成 skill 时，主 Agent 不应该直接输出脚本，而应按节点推进：

1. 读取产品资料和品牌资料，提取产品事实、证据、限制和目标人群。
2. 判断传播问题类型，并从 [[domains/视觉制作/06-AI视频/90-案例库/index|AI商业短视频案例库]] 检索相似案例。
3. 输出 2-3 个创意命题，每个命题必须说明借鉴了哪个案例、借鉴了什么、哪些不能照抄。
4. 选择或推荐一个命题后，展开人物、场景、冲突、产品入场和镜头任务。
5. 生成分镜表、参考资产清单、提示词方向和制作风险。
6. 完成后把新项目回写成新案例，进入案例库复利。

Skill 的关键不是“自动写一个视频脚本”，而是让 Agent 先拥有案例记忆，再像策划一样完成判断。

## 误用边界

- 不要把案例库当模板库。案例只能提供类比，不能机械替换产品名。
- 不要把情绪片理解成慢镜头、暗光和文艺旁白；情绪必须来自人物关系和生活动作。
- 不要把功能产品拍成参数展示；功能要落到具体摩擦。
- 不要把文化片拍成符号堆叠；文化要有时间、人物、史料和现代连接。
- 不要跳过分镜图片阶段直接生成视频；没有分镜，AI 很容易把片子做散。
- 不要让漂亮镜头主导广告；商业短视频最终要让观众理解产品为什么值得被需要。
