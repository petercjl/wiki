## 案例结论

这条运动品牌广告不是先找一个帅气运动员，再把衣服套上去；它的关键是先确定产品气质，再让人物从产品气质里长出来。

成片的核心命题是：

```text
一个人有两种生活
-> 家庭里的慢、安静、温柔
-> 球场上的原始、释放、力量
-> 一件舒适运动服可以适配两个部分的自我
```

这让服装不只是“运动时穿的功能产品”，而是一个成年人在家庭责任和自我燃烧之间切换时的身体容器。它适合迁移到运动服、家居服、内衣、鞋履、通勤运动两用产品等品类。

## 产品/主题诊断

| 维度 | 项目判断 |
| --- | --- |
| 品类 | 面向美国市场的运动服饰/运动故事 |
| 广告路线 | 人物状态广告，而不是参数广告 |
| 核心人群 | 有家庭生活，同时仍保留运动自我和竞技欲望的成年人 |
| 情绪命题 | 我不只有一种身份，舒适服装要适配完整的我 |
| 产品角色 | 连接家庭慢生活和运动力量感的舒适载体 |
| 制作重点 | 产品气质图、角色反推、角色一致性、运动镜头、AI 音乐版权规避 |

## 构思路径复原

### 1. 不从功能词开始，而从“人的两种状态”开始

运动服饰很容易写成：

```text
透气
速干
弹力
舒适
专业运动
```

这条片没有这样做。它先写了一个人物状态：

```text
I live two lives
one slow, quiet, wrapped in family warmth and gentle mornings
the other raw, unchained, where I chase my fire on the court
```

功能卖点“舒适”没有消失，而是被放在最后：“the comfort that fits every part of me”。这比直接说“舒适运动裤”更高级，因为它把舒适解释为“能容纳多重身份”。

### 2. 生活段和运动段形成反差

成片里有两个世界：

| 世界 | 画面 | 作用 |
| --- | --- | --- |
| 慢生活 | 郊区清晨、家庭、车内、孩子、温和对话 | 建立责任、亲密、柔软的一面 |
| 运动场 | 跑步机、篮球、汗水、身体发力、球场 | 建立力量、释放、竞技的一面 |

这两个世界不是割裂的。广告真正要卖的是：同一个人可以在两种状态之间切换，而服装可以适配这种切换。

### 3. 先确定产品感觉，再生成人物

`ch08` 里最重要的制作节点是：因为产品面向美国市场，博主一开始没有先生成角色，而是先确定产品感觉，做了一张跑步机中景产品图。

然后他把这张产品图喂给 AI，让 AI 根据产品气质“脑补”角色长相。这个顺序很关键：

```text
错误顺序：先生成一个好看人物 -> 再让人物穿产品
本案例：先确定产品感觉 -> 由产品图反推人物 -> 再扩展角色资产
```

对服装广告来说，产品气质决定人物气质。角色不是随机模特，而应该像这个品牌会选择的人。

### 4. 用白底图和三视图解决角色一致性

当确认角色后，博主会创建角色白底图和三四角图，后续在控制角色一致性时直接链接。

这说明 AI 服装广告不能只靠一张好图。至少需要：

- 角色正面/侧面/不同角度。
- 产品穿着状态图。
- 运动场景图。
- 生活场景图。
- 统一的人物脸、体型、发型和年龄感。

### 5. 画布模式把图片、视频和音乐串成生产线

讲解里推荐使用无限画布模式：图片节点生成完后，不用退出软件，直接在图片加号里创建视频节点；音乐也可以在画布节点中创建音频。

这个案例的工作流是：

```text
图片节点
-> 角色/产品图
-> 视频节点
-> 选择视频模型与画面比例
-> 输入运动提示词
-> 音频节点
-> AI 音乐生成
-> 剪辑整合
```

### 6. AI 音乐用于商业版权规避，但仍要抽卡

博主说自己大多做商业广告，如果找现成音乐会很麻烦，所以直接用 AI 音频生成以避免版权问题。这里他使用 `11 Music V3`，并把主角人设、风格和音乐节奏输入进去。

但音乐也不是一次成功。他大约抽了七次，最后从两条里面选一条可用素材合成。

## 原片故事结构

| 段落 | 画面/旁白 | 情绪功能 | 方法意义 |
| --- | --- | --- | --- |
| 开场生活 | 郊区清晨、家庭氛围 | 建立慢生活和温柔基底 | 先让人物像真实成年人 |
| 家庭片段 | 孩子、车内、家庭互动 | 展示责任和亲密关系 | 让运动员不是单一肌肉符号 |
| 运动唤醒 | 跑步机、身体准备、球场 | 切入力量和释放 | 与慢生活形成反差 |
| 运动爆发 | 篮球、跑步、身体发力 | 建立产品的运动属性 | 功能通过动作表达 |
| 收束产品价值 | comfort fits every part of me | 把两种生活连接到舒适 | 产品价值从“穿着舒服”变成“适配完整自我” |

## 博主讲解中的方法节点

### 节点 1：工具不是核心，画布是生产结构

这次用到的是 Lib TV，但博主强调的仍然是画布模式。画布的价值在于把产品图、角色图、场景图、视频节点和音频节点放在同一个项目结构中，便于追踪和复用。

### 节点 2：模型按镜头需求选择

跑步镜头用了 Lib 的 Nano Pro；视频生成阶段可以根据画面选择 Seedance、KlingView 等主流模型，并设置画面比例。这里的原则不是“固定模型”，而是按镜头运动、画面比例和稳定性选择模型。

### 节点 3：产品图可以反推角色

对服装广告特别有价值：当产品已经有明确市场、版型和气质时，不一定先找模特。可以让产品图成为角色生成的约束，让人物更像“这个品牌的人”。

### 节点 4：角色一致性要提前资产化

确认角色后立刻创建白底图和三四角图，这是后续大量镜头稳定生成的基础。否则运动广告一旦切换生活、跑步、篮球、车内等场景，人物很容易变成不同人。

### 节点 5：商业广告音乐要把版权当制作条件

AI 音乐不是因为“方便好玩”，而是因为商业广告找现成音乐会有版权和授权麻烦。音乐生成时要输入主角人设、风格和节奏，再通过抽卡挑可用素材。

## 可迁移灵感

### 适合迁移到哪些产品

- 运动服、瑜伽服、运动内衣、运动裤。
- 通勤运动两用鞋、休闲跑鞋。
- 家居服、内衣、无尺码服饰。
- 男士基础款、轻户外服装。
- 面向海外市场的服饰品牌片。

### 可迁移结构

```text
产品气质
-> 目标市场和人物状态
-> 人物的两种生活/两种身份
-> 产品如何连接这两种状态
-> 先生成产品气质图
-> 由产品图反推角色
-> 角色白底图和三视图
-> 生活镜头 + 运动/使用镜头
-> 产品价值一句话收束
```

### 迁移示例

| 产品 | 不该只讲 | 可以转译成 |
| --- | --- | --- |
| 通勤跑鞋 | 轻、软、缓震 | 早上挤地铁，晚上跑五公里，同一双鞋接住两种节奏 |
| 运动内衣 | 支撑、防震、透气 | 她不是为了被看见而运动，而是为了重新掌控身体 |
| 家居服 | 亲肤、柔软 | 工作结束后，人终于从角色里退出来 |
| 轻户外外套 | 防风、防泼水 | 城市和山野之间，一个周末就能切换身份 |
| 男士基础 T 恤 | 棉感、版型 | 父亲、丈夫、球友、自己，都不用换一件衣服来证明 |

## 反向拆解：如果做成硬广会怎样

错误做法可能是：

- 开头直接拍模特跑步和打球。
- 旁白只说透气、舒适、弹力、适合运动。
- 人物只有肌肉和汗水，没有生活身份。
- 每个镜头都像健身房素材，缺少家庭反差。
- 先生成随机帅哥，再强行换装。

这种做法的问题是，服装只剩功能，没有人。这个案例的优势是把“舒适”放进一个人的完整生活里。

## Agent 调用模板

当 Agent 拿到服饰、鞋履、运动或生活方式产品时，可以调用本案例：

1. 产品气质是什么？不要先找随机模特。
2. 目标市场的人有哪些两种生活/身份？
3. 产品如何连接这两种状态？
4. 第一张产品气质图应该是什么？
5. 能否由产品图反推角色？
6. 角色一致性需要哪些白底和角度资产？
7. 生活镜头和使用镜头如何形成反差？
8. 最后一日句产品价值如何收束？

输出结构：

```text
产品：
目标市场：
产品气质：
人物两种状态：
产品连接方式：
第一张产品气质图：
角色资产：
生活镜头：
功能/运动镜头：
旁白主线：
音乐风格：
硬广感检查：
```

## 误用边界

- 不要把运动服广告只做成肌肉、汗水和慢动作。
- 不要先生成随机模特再硬套产品。服装品类要先有产品气质。
- 不要忽略目标市场。美国市场、国内市场、女性运动、青少年运动的角色气质不同。
- 不要用 AI 音乐一次抽到就直接用，商业片至少要按风格、节奏和版权要求筛选。
- 不要让生活镜头和运动镜头各拍各的，最后必须用产品价值把两者连接起来。
