## 案例结论

这条母亲节广告没有用“妈妈我爱你”“感谢母亲”这类口号，而是用一组组生活动作让观众自己感到母爱：

```text
母亲送菜
-> 母亲催喝鸡汤
-> 母亲嘴硬但惦记孩子
-> 母亲学会点外卖
-> 最后把“照顾你”换成一种更现实的方式
```

它的核心不是节日祝福，而是把母亲的爱还原成普通人生活里的笨拙、操心、嘴硬、学习和妥协。节日情绪片要打动人，不能只靠煽情文案，要靠具体动作和关系细节。

## 主题诊断

| 维度 | 项目判断 |
| --- | --- |
| 主题 | 母亲节情绪广告 |
| 广告路线 | 节日情绪群像片 |
| 核心人群 | 成年子女、高中生、怀孕白领、加班年轻人，以及各自背后的母亲 |
| 情绪命题 | 母亲未必会说爱，但会用食物、操心和笨拙学习来照顾你 |
| 结构 | 多人物片段并列，最后用“母亲学会点外卖”收束 |
| 制作重点 | 人物关系设计、生活动作、镜头减法、真实感取舍 |

## 构思路径复原

### 1. 不写“母亲伟大”，先找母爱的生活动作

节日广告最容易变成口号：辛苦了、谢谢你、母爱伟大。但这条片子没有直接解释母爱，而是把母爱拆成动作：

- 从老家带很多吃的。
- 觉得城里买不到自己种的菜。
- 催怀孕女儿喝鸡汤。
- 早上五点起来摘蔬菜。
- 说话嘴硬，关心绕着说。
- 学会点外卖，承认让孩子自己做饭不现实。

这些动作比文案更有记忆点，因为观众能从自己的生活里找到相似经验。

### 2. 用群像覆盖不同母子关系

`ch13` 里博主说，他把剧情拆成四个板块，并花了比后期制作更长的时间做人物关系设计。片中角色都是身边的普通人：

| 人物关系 | 情绪来源 | 具体动作 |
| --- | --- | --- |
| 农村母亲与成家儿子 | 母亲怕孩子在城里吃不好 | 带很多吃的、送菜 |
| 严厉母亲与高中女儿 | 爱被包在要求和嘴硬里 | 承诺考好就给、开玩笑又惹生气 |
| 照顾怀孕女儿的妈妈 | 担心女儿身体和工作压力 | 催喝鸡汤、质疑工作太辛苦 |
| 关心加班年轻人的母亲 | 想照顾但接受现实限制 | 学会点外卖，让孩子准时吃饭 |

群像结构的好处是：它不是押宝一个主角，而是用多个母亲样本扩大共鸣面。

### 3. 让最后一个动作完成情绪反转

片尾“母亲学会点外卖”是很好的收束，因为它不是传统意义上的煽情大动作，而是一个小小的行为变化：

```text
过去：妈让你自己做饭
现实：你工作忙，自己做饭不现实
转变：妈学会点外卖
情绪：她仍然在照顾你，只是换了一种方式
```

这个动作让母亲不只是“付出者”，也变成一个愿意学习、适应孩子生活的人。情绪因此更真实。

## 博主讲解中的方法节点

### 节点 1：画布是工作模式

`ch06` 里博主说，画布不是某一家 AI 软件的优势，而是一种工作模式。母亲节这种多人物、多场景片，如果不用画布，人物关系和镜头逻辑会乱。

对案例库来说，画布对应的是：

- 角色板块。
- 场景板块。
- 分镜图。
- 人物一致性节点。
- 音乐和剪辑节点。

### 节点 2：提示词要简单精准

博主反对“提示词越长越好”。例如“女儿抱紧骑电动车的母亲”“关车门”这类提示词本身可以很简单，关键是你知道要拍什么。

提示词不是创意，提示词是把已经想清楚的镜头表达出来。

### 节点 3：导演思维和分镜比工具重要

博主说，不应该优先学 AI 工具，而要积累审美和导演思维，学习分镜头制作和如何剪辑好故事。AI 只是缩短影视质量差距，但讲好故事的差距还很大。

### 节点 4：镜头减法

他举了父亲得知母亲五点起来摘菜后的镜头设计：

错误新手做法：

```text
人物开车回去
-> 和母亲聊感人台词
-> 手接过蔬菜
-> 放回车里
-> 正面人物特写
```

博主做法：

```text
一个镜头里出现玉米、蔬菜、母亲在掰
-> 车出现
-> 后备箱一关
```

两个镜头观众就明白了。这里的经验是：情绪不一定靠台词和特写，有时一个信息密度足够的动作镜头更有力量。

### 节点 5：真实感允许不完美

`ch13` 里博主说，有些图看着好看，但不真实，因为现实光线下人物面部不可能那么清楚。他宁愿调整提示词，让人物关系变大、画面不那么清楚，也要更接近现实拍摄效果。

这对 AI 视频很重要：真实感不是每张脸都清楚、每个物体都漂亮，而是画面符合现实拍摄逻辑。

### 节点 6：角色一致性

母亲节片涉及多个角色。博主会创建虚拟角色，在节点里关联角色图片，例如小女孩站在这里，前景虚焦也用这个女孩。角色一致性能提高影片完整度。

### 节点 7：音乐按段落生成

博主会根据影片段落生成不同背景音乐。多人物情绪片不是一条音乐铺到底，而是根据情绪段落设计音乐。

## 可迁移灵感

### 适合迁移到哪些产品/主题

- 母亲节、父亲节、重阳节、春节返乡、中秋团圆。
- 食品、滋补品、礼盒、家电、健康产品。
- 外卖平台、即时零售、社区团购、智能手机教学类内容。
- 面向家庭关系的品牌片。

### 可迁移结构

```text
节日主题
-> 不写口号
-> 找 3-4 组普通关系
-> 每组用一个具体生活动作表达情感
-> 末尾用一个现代生活里的小转变收束
```

### 迁移示例

| 产品/主题 | 不该只讲 | 可以转译成 |
| --- | --- | --- |
| 母亲节礼盒 | 感恩妈妈、送礼首选 | 妈妈总把好的留给你，这次换你记得她爱吃什么 |
| 智能手机适老功能 | 字大、操作简单 | 父母学会视频、外卖、付款，其实是在靠近孩子的生活 |
| 滋补品 | 营养、配方 | 子女终于把“多注意身体”还给父母 |
| 洗碗机 | 解放双手 | 年夜饭后不用让妈妈一个人留在厨房 |
| 社区团购 | 便宜方便 | 远方子女也能帮家里买到今天要吃的菜 |

## 反向拆解：如果做成硬广会怎样

错误做法可能是：

- 直接喊“母亲节快乐”。
- 镜头里孩子抱着妈妈哭。
- 用大段旁白解释母爱伟大。
- 产品强行出现在每个画面里。
- 每个场景都拍得干净、漂亮、过度摆拍。

这种做法的问题是情绪太“被安排”。母亲节项目的优势在于，它让观众先看到熟悉的生活动作，再自己得出“这是爱”。

## Agent 调用模板

当 Agent 拿到节日、亲情、家庭关系类产品时，可以调用本案例：

1. 主题是什么？不要先写口号。
2. 这个主题在真实生活中有哪些小动作？
3. 能不能找 3-4 组普通人物关系？
4. 每组关系用哪个动作表达，不用解释？
5. 哪个动作可以作为最后的情绪反转？
6. 哪些镜头可以合并，避免口水话？
7. 画面是否真实到像生活，而不是像棚拍？

输出结构：

```text
节日主题：
核心情绪：
人物关系组：
每组生活动作：
最后反转动作：
镜头减法方案：
角色一致性要求：
音乐段落：
真实感检查：
硬广感检查：
```

## 生成提示词方向

### 农村母亲送菜

```text
竖屏真实电影广告画面，清晨农村院子里，一位六十多岁的母亲弯腰整理刚摘下的蔬菜和玉米，旁边停着一辆旧车，后备箱半开，画面有泥土、竹筐和清晨雾气，柯达胶片质感，真实生活抓拍，不要过度清晰，温暖克制
```

### 加班年轻人接电话

```text
竖屏电影广告画面，深夜办公室，一个二十多岁的年轻人坐在电脑前接母亲电话，桌上有外卖盒和文件，窗外城市灯光虚焦，人物疲惫但克制，柯达胶片质感，真实室内弱光，浅景深
```

### 母亲学点外卖

```text
竖屏真实生活广告画面，一位中年母亲戴着老花镜坐在餐桌旁，认真看手机教程学习点外卖，桌上放着写满步骤的纸条和一杯水，表情笨拙又专注，家庭暖光，柯达胶片质感，真实不摆拍
```

### 图生视频运动

```text
subtle handheld camera, the mother slowly taps the phone screen while checking handwritten notes, warm kitchen light, quiet emotional mood, realistic ordinary family scene, no exaggerated acting
```

## 误用边界

- 不要把节日情绪片做成口号合集。必须有具体生活动作。
- 不要把所有母亲都拍成同一种温柔形象。真实母亲可能嘴硬、严厉、笨拙、操心。
- 不要为了画面漂亮牺牲真实光线和生活质感。
- 不要用太多台词解释情绪。能用动作表达，就删掉口水话。
- 不要把群像片做成碎片堆砌，每个片段都要服务同一个主题。
