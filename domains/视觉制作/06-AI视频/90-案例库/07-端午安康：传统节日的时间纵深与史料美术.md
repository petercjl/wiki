## 案例结论

这条端午短片的核心不是“端午安康”四个字，也不是粽子、龙舟、艾草的节日元素堆叠，而是把端午拆成穿越不同时代的人生片段：

```text
端午不止粽子
-> 有家国记忆
-> 有亲情和孝道
-> 有战乱后的回家
-> 有青年人的奔赴
-> 有现代家庭的团圆
```

它真正的创意是给节日做“时间纵深”：同一个节日不是一个静态符号，而是被不同历史时期的人反复经历、解释和传承。

| 维度 | 项目判断 |

| 可迁移任务 | 端午、中秋、春节、七夕、重阳等传统文化节日品牌片 |

## 构思路径复原

### 1. 不把节日当元素，而把节日当历史容器

传统节日广告最容易变成元素清单：

```text
端午
-> 粽子
-> 龙舟
-> 艾草
-> 祝福语
```

这条片子反复强调“端午不止粽子”。它把节日当成一个能容纳多种情绪的历史容器：有人记录，有人思亲，有人回家，有人奔赴，有人抗争，有人现代团圆。

### 2. 用多时代片段承接节日厚度

成片不是讲一个主角，而是用多个时代人物构成群像：

| 片段 | 可能承载的主题 | 方法意义 |
| --- | --- | --- |
| 古代记录者 | 后人该知道端午不止粽子 | 用一句话提出整片命题 |
| 母子追逐和粽子 | 孝道、亲情、生活烟火 | 让节日从宏大文化落到家庭 |
| 战乱后的衣裳 | 回家、珍重、平安 | 用衣服承载“终于回来了” |
| 看龙舟的青年 | 河流不止，人也不能停 | 把龙舟精神转成青年奔赴 |
| 家国和抗争片段 | 屈原、家国、意气 | 让端午有精神底色 |
| 现代家庭 | 海外/异地归家、陪父母过节 | 让传统节日回到当下生活 |

这类结构适合文化短片，因为它不依赖一个完整剧情，而依赖主题连续性。

### 3. 用一句重复命题把碎片串起来

“端午不止粽子”是整条片的钉子。没有这句话，多时代片段容易散；有了这句话，每个片段都在回答“还是什么？”

```text
端午不止粽子
-> 还有家国
-> 还有亲情
-> 还有回家
-> 还有奔赴
-> 还有传承
```

### 4. 现代段负责把文化拉回今天

如果全片只停留在古代和历史，就容易变成文化展示。现代家庭段让观众意识到：端午不是博物馆里的传统，它仍然发生在今天的家庭关系里。

现代段的价值是把“文化传承”从抽象词变成行动：回来陪家人过节。

## 博主讲解中的方法节点

### 节点 1：五步制作流程

该影片的制作流程分为：

```text
资料查询和脚本构思
-> 专项训练图片生成
-> 图生视频
-> AI 音乐制作
-> 剪辑合成
```

和普通情绪广告相比，端午片多了一个特别重的前置环节：资料查询和史料美术。

### 节点 2：历史细节片最难的是资料查验和图片生成

博主明确说，涉及历史细节的影片，最难的是资料查验和图片生成。很多粗制滥造 AI 视频之所以浮夸和错误，是因为没有做专项训练和美术，大量使用公共库素材。

这给传统文化片一个重要边界：AI 不是随便生成“古风”。传统文化广告越想高级，越不能泛古风。

### 节点 3：古画、壁画和人物资料是视觉依据

讲解里提到：

- 唐朝小孩形象取自古画。
- 唐朝女性挎包从壁画还原。
- 东汉时期建筑风格和服装需要还原。
- 杜甫段尽量使用杜甫本人脸部资料。

这说明“真实感”不是只靠画质，而是靠来源依据。历史片的美术要能回答：这个服装、建筑、道具从哪里来？

### 节点 4：专项训练是精品化流程

博主认为，专项训练让影片真实感提升，也是未来 AI 视频精品化的流程。

对案例库来说，这个节点非常关键：如果未来我们给品牌做传统文化节日短片，要准备的不是一句 prompt，而是一套史料包、人物包、服饰包和场景包。

## 可迁移灵感

### 适合迁移到哪些产品/主题

- 端午、中秋、春节、七夕、重阳、清明等传统节日品牌片。
- 茶、酒、糕点、礼盒、国风服饰、文创产品。
- 老字号品牌、地方特产、非遗工艺。
- 文化旅游、博物馆、城市宣传片。
- 有历史和产地资产的食品、滋补品、香氛、家居品牌。

### 可迁移结构

```text
传统节日/文化主题
-> 不先堆元素
-> 找一句反元素命题
-> 拆成 3-5 个时代/人生片段
-> 每段表达一个文化情绪
-> 用现代片段收回当下
-> 用史料美术保证可信
```

### 迁移示例

| 产品/主题 | 不该只讲 | 可以转译成 |
| --- | --- | --- |
| 中秋礼盒 | 月饼、团圆、送礼 | 月亮照过不同年代的离别和归来 |
| 春节年货 | 红色、喜庆、全家福 | 一桌年夜饭如何接住每个归家的人 |
| 茶品牌 | 古法、茶香、产地 | 一杯茶如何穿过师徒、山场、书房和现代办公室 |
| 老字号糕点 | 百年传承、口味 | 每一代人都用同一种甜，记住不同的人 |
| 城市文旅 | 名胜古迹 | 一条河/一座城在不同年代里如何改变普通人的命运 |

## 反向拆解：如果做成硬广会怎样

错误做法可能是：

- 只拍粽子、龙舟、艾草和祝福语。
- 统一套“古风滤镜”，不分朝代和地域。
- 人物服装、建筑、道具没有史料依据。
- 全片只有漂亮古装，没有现代观众的情绪入口。
- 只讲文化大词，不落到具体人物和生活动作。

这种做法的问题是，节日只剩符号，没有时间深度。端午项目的优势是用多个时代证明：节日不是元素，而是反复被人经历的文化记忆。

## Agent 调用模板

当 Agent 拿到传统节日、文化品牌、老字号、文旅或非遗产品时，可以调用本案例：

1. 这个节日/文化主题最容易被简化成什么元素？
2. 能否写出一句“反元素命题”？
3. 这个主题能拆成哪 3-5 个时代或人生片段？
4. 每段表达哪一种文化情绪？
5. 现代段如何把文化拉回今天？
6. 需要哪些史料、古画、壁画、服装、建筑、人物参考？
7. 哪些地方不能用泛古风公共素材？
8. 产品或品牌应该在文化片中承担什么角色？

输出结构：

```text
节日/文化主题：
反元素命题：
时代/人生片段：
每段文化情绪：
现代连接段：
史料参考清单：
角色/服装/建筑要求：
产品入场方式：
音乐风格：
误用边界：
```

## 生成提示词方向

### 古代记录者

```text
竖屏历史电影画面，古代书房中一位文人坐在木案前记录端午习俗，桌上有竹简、毛笔、粗陶杯，窗外江雾微光，服饰和建筑参考东汉时期，克制写实，不要泛古风仙侠，不要夸张特效
```

### 唐代母子与粽子

```text
竖屏真实历史生活画面，唐代街巷或庭院中，一个小孩穿着参考古画的儿童服饰奔跑，母亲拿着粽子在后面笑着追，人物服饰和发髻参考唐代壁画，温暖生活感，真实电影质感，不要现代化道具
```

### 战乱后归家

```text
竖屏历史电影画面，战乱后归来的男子站在旧宅门口，手中拿着一件干净衣裳，背景是烟尘散去的街巷，表情克制，主题是回家与平安，低饱和写实摄影，服装道具符合时代，不要戏曲化
```

### 现代端午归家

```text
竖屏真实生活方式画面，现代年轻女孩回家和父母一起吃端午饭，桌上有粽子和家常菜，室内自然光，父母带一点调侃和开心，温暖真实家庭气氛，节日感克制，不要广告摆拍
```

## 误用边界

- 不要把传统节日片做成元素堆叠。必须有反元素命题。
- 不要用泛古风替代史料美术。不同朝代的服装、建筑、道具要区分。
- 不要只拍古代，现代观众需要当下入口。
- 不要把文化片做成知识讲解，要通过人物和动作表达文化情绪。
- 不要因为 AI 能生成古装，就忽略资料查验和专项训练。
