## 案例结论

这条广告把汽车“大空间/能装”的功能卖点，转译成一个三四五线小家庭会真实遇到的生活矛盾：

```text
进货装不下
-> 多跑一趟浪费油钱
-> 夫妻争吵
-> 孩子在中间感到紧张
-> 更大的车解决工作和家庭两类需求
```

它的关键不是展示车有多大，而是让观众先感到“装不下”会带来的家庭成本、情绪成本和经营成本。产品出现后，功能卖点才变成生活改善。

## 产品诊断

| 维度 | 项目判断 |
| --- | --- |
| 产品 | 五菱星光560 |
| 品类 | 家用/商用兼顾的汽车 |
| 核心卖点 | 空间大、能装、兼顾工作和生活 |
| 目标人群 | 三四五线个体经商户小家庭，夫妻共同经营，有孩子，有装货和家庭出行需求 |
| 广告路线 | 功能问题解决型产品故事 |
| 核心结构 | 一个问题的争吵，一个产品去解决问题 |
| 真实感关键 | 美术设计和生活信息密度，而不是长提示词 |

## 构思路径复原

### 1. 先问产品给谁用

博主在 `ch10` 和 `ch12` 都重复了同一个起点：首先要确定产品是给谁用的。

这款车没有被设计给抽象的“年轻家庭”或“城市精英”，而是具体到：

- 三四五线城市或县城。
- 个体经商户。
- 夫妻共同经营。
- 有孩子。
- 日常既要进货、搬货，也想家庭露营和生活出行。

人物越具体，剧情越容易长出来。

### 2. 把“大空间”变成夫妻矛盾

如果直接讲大空间，很容易变成参数广告。但博主把它转成一段家庭争吵：

- 妻子抱怨丈夫“又拉多了”。
- 货里有啤酒、大米、纸等生活化物品。
- 多跑一趟油钱都够进两箱纸。
- 妻子提到“当初叫你买个大点的车，你非说没用”。
- 孩子出现，让争吵带上家庭关系压力。

这个设计让空间卖点不再是抽象尺寸，而是经营成本、时间成本、夫妻关系和孩子感受。

### 3. 产品不是展示，而是转折

产品出场前，问题已经被观众理解：

```text
车装不下
-> 要多跑一趟
-> 多花油钱
-> 夫妻争执
-> 孩子被卷入气氛
```

产品出场后，解决的不是“装货”一个点，而是：

- 生意进货能装。
- 家庭露营也有着落。
- 妻子发现丈夫不是固执，而是准备了惊喜。
- 家庭气氛从争吵变为和解。

这就是“功能卖点 -> 生活价值”的转换。

### 4. 用三个人完成剧情

博主明确说，爸爸、妈妈、孩子三个人就能交代清楚剧情。

三个人的角色分工是：

| 角色 | 剧情功能 |
| --- | --- |
| 妻子 | 提出真实问题：装不下、油钱、当初没买大车 |
| 丈夫 | 承接压力和转折：先被抱怨，后面拿出更大的车 |
| 孩子 | 让争吵有家庭感，也让“露营”成为生活希望 |

这是功能广告里非常值得借鉴的角色设计：不要堆很多人物，只用最少角色让问题、冲突和解决都成立。

## 原片故事结构

| 段落 | 画面/台词 | 叙事功能 | 产品价值 |
| --- | --- | --- | --- |
| 开场冲突 | “你就不能过来搭把手”“你这明显又拉多了” | 直接进入问题，不铺垫 | 用户马上知道有装载压力 |
| 具体成本 | “多跑一趟的油钱都够进两箱纸了” | 把空间不足转成经营成本 | 大空间不是爽点，是省钱省事 |
| 旧选择被质疑 | “当初叫你买个大点的车，你非说没用” | 暗示产品决策曾经错误 | 新产品纠正旧痛点 |
| 孩子介入 | “爸，妈”“你快进去写作业” | 争吵变成家庭氛围问题 | 产品改善家庭情绪 |
| 转折 | “谁告诉你拉不下的” | 产品即将出现 | 解决方案进入 |
| 价值证明 | “这车可真能装啊” | 卖点被角色说出 | 功能由剧情验证 |
| 生活延展 | “咱们去露营是不是也有着落了” | 从工作延展到生活 | 能工作，也能生活 |

## 制作流程节点

### 节点 1：美术概念场景设计

博主把这个项目流程拆成三个板块：美术概念场景设计、剧情图片生成、视频生成。其中 `ch12` 特别强调，美术设计是 AI 视频真实感的关键。

这里的美术不是“画面漂亮”，而是让场景里包含足够多生活信息：

- 夫妻搬货的矛盾点。
- 街上路人。
- 小区商铺。
- 刚刚走过的货车。
- 车牌、招牌、第三方品牌信息的合规处理。

一个场景要包含多个信息，观众潜意识会判断它像不像真实生活。

### 节点 2：第一张美术基调图

为了出第一张美术基调图，博主用了“海量战术”，反复用提示词试探。第一张图的作用不是单纯好看，而是确立：

- 这个家庭所处的城市层级。
- 店铺/街道/货物的生活质感。
- 车辆和人物的关系。
- 画面的真实度与干净度平衡。

### 节点 3：剧情图片生成

在剧情图片阶段，需要围绕已经确定的人物关系和冲突生成分镜。博主特别提到产品一致性难，所以会单独给产品资料打区域，用链接节点生成对应模型。

对汽车广告来说，产品不能每个镜头都变形，否则观众会立刻出戏。

### 节点 4：图生视频

图生视频阶段不是重新创作剧情，而是让分镜图中的角色完成动作。博主说“你让角色该做什么就做什么”，这说明核心创作已经在前面的定位、场景和分镜完成了。

### 节点 5：焦段和摄影基本功

`ch12` 里提到，可以把焦段信息给 AI：例如 50mm 中远景、75mm 近景。提示词本质上是把“该拍摄的镜头”变成文字；不懂摄影，就会弱于有实拍基础的人。

## 真实感方法

### 1. 真实感不是提示词问题

博主明确说，AI 视频做不到真实度，不单单是提示词的问题，背后是美术设计。

也就是说，错误方向是问：

```text
这个提示词怎么写？
```

正确方向是问：

```text
这个人应该在哪个真实环境里？
环境里有哪些符合他生活的细节？
哪些信息会让观众潜意识相信？
哪些东西会破坏合规或真实逻辑？
```

### 2. 场景信息密度

一张图里要承载多个信息：

- 夫妻正在搬货。
- 货物和经营有关。
- 街边有商铺。
- 旁边有路人。
- 环境不像摄影棚，而像真实县城/社区街道。

这些不是装饰，而是剧情证据。

### 3. 真实质感和干净画面的平衡

博主提到，画面物料要找到“真实质感”和“干净画面”之间的平衡。真实不等于乱，干净也不等于假。商业广告要既能让人相信，又不能脏乱到影响产品价值。

### 4. AI 创作者判断高于 AI

车牌、招牌、第三方品牌信息、场景逻辑，都需要创作者判断。不能把 AI 当成自动导演，AI 只能承担部分工作，最核心判断仍在人。

## 可迁移灵感

### 适合迁移到哪些产品

- 汽车、货车、露营车、后备箱空间相关产品。
- 收纳箱、置物架、搬家工具、推车。
- 家电：洗烘套装、洗碗机、扫地机等解决家庭争吵的产品。
- 母婴大件：婴儿车、安全座椅、儿童学习桌。
- 小商户工具：扫码设备、打包机、打印机、货架。

### 可迁移结构

```text
具体人群
-> 日常摩擦
-> 两个人因摩擦产生争吵
-> 产品解决摩擦
-> 功能价值变成关系和生活价值
```

### 迁移示例

| 产品 | 不该只讲 | 可以转译成 |
| --- | --- | --- |
| 洗碗机 | 洗得干净、省水 | 谁洗碗引发的家庭疲惫和争吵被解决 |
| 收纳柜 | 容量大、材质好 | 家里总找不到东西、老人孩子被绊倒的问题被解决 |
| 扫地机 | 吸力强、避障好 | 下班后不用再为地面和宠物毛崩溃 |
| 打包机 | 效率高 | 小商户晚上少加班，孩子不用等父母太晚回家 |
| 儿童安全座椅 | 安全认证 | 夫妻关于孩子出行安全的焦虑被具体安放 |

## 反向拆解：如果做成硬广会怎样

错误做法可能是：

- 开头直接拍车外观和内饰。
- 字幕写“大空间、强动力、全家出行”。
- 展示后备箱能装多少箱东西。
- 用模特露营摆拍。

这种做法的问题是：卖点都对，但没有真实生活压力。观众知道车能装，却不一定感到“我需要它”。五菱这个项目的优势是先让观众感到“装不下真的烦”，再让产品解决。

## Agent 调用模板

当 Agent 拿到功能明确的产品时，可以调用本案例：

1. 产品给谁用？必须具体到城市层级、家庭结构、工作方式。
2. 这个产品解决的日常摩擦是什么？
3. 这个摩擦会让谁和谁发生争吵？
4. 争吵里能不能出现具体成本？例如钱、时间、体力、孩子感受。
5. 产品出现后，除了功能解决，还能带来什么关系改善？
6. 场景里需要哪些生活信息让观众相信？

输出结构：

```text
目标人群：
生活摩擦：
争吵双方：
具体成本：
产品转折：
工作价值：
生活价值：
关键场景美术：
分镜表：
产品一致性要求：
真实感检查：
```

## 生成提示词方向

### 美术基调图

```text
竖屏真实商业广告画面，三四五线城市小区街边商铺门口，一对三十多岁的夫妻正在搬货，地上有啤酒箱、大米、纸箱和日用品，旁边停着家用车，街上有路人和刚驶离的小货车，环境真实但画面干净，县城生活质感，50mm中远景，低饱和自然光，电影广告摄影
```

### 冲突镜头

```text
竖屏电影广告画面，妻子站在一堆货物旁边皱眉看向丈夫，丈夫手里拿着纸箱有些尴尬，孩子从旁边探出头，背景是小区商铺和停在路边的家用车，真实生活冲突，75mm近景，浅景深，自然光
```

### 产品解决镜头

```text
竖屏真实汽车广告画面，五菱星光560后备箱打开，啤酒箱、大米、纸箱整齐装入车内，夫妻表情从争吵转为惊讶和缓和，孩子站在旁边开心看着，真实县城家庭生活场景，商业广告质感，干净但不失生活感
```

### 图生视频运动

```text
subtle handheld camera, the wife points at the pile of goods, the husband pauses with a box in his hands, the child looks between them, natural street ambience, realistic family tension, no exaggerated acting
```

## 误用边界

- 不要把所有功能产品都硬套夫妻争吵。只有当产品确实解决家庭、工作或关系摩擦时才适用。
- 不要为了戏剧冲突把人物写得太夸张。这个案例的真实感来自日常小矛盾，不是狗血剧情。
- 不要只复制“能装货”。真正可迁移的是“功能卖点进入具体生活成本”。
- 不要忽略美术设计。没有真实场景和生活信息密度，故事会变成 AI 影视棚摆拍。
