## 案例结论

这条低度果酒软广的核心不是“酒好喝”“适合聚会”，而是把产品放进年轻人的压力叙事里：

```text
从小到大，轻松总被别人承诺
-> 高考完就好了
-> 毕业以后就好了
-> 工作以后就好了
-> 但轻松好像一直由别人定义和批准
-> 一杯轻微醺，变成自己给自己的日常放空
```

它真正卖的不是酒精刺激，而是“轻松这件事不用等别人批准”。这让低度酒从饮用产品变成一种克制的自我照顾仪式。

## 产品/主题诊断

| 维度 | 项目判断 |
| --- | --- |
| 品类 | 低度果酒/微醺饮品软广 |
| 广告路线 | 情绪软广，而不是口味或酒精度硬广 |
| 核心人群 | 从考试、毕业到职场一路被目标追赶的年轻人 |
| 情绪命题 | 轻松不必等项目结束、别人批准或人生阶段结束，今天也可以自己给 |
| 产品角色 | 温柔和解、日常放空、自我批准的载体 |
| 制作结构 | 脚本定位 -> AI 生图 -> 图生视频 -> 定制 AI 音乐 -> 剪辑合成 |

## 构思路径复原

### 1. 先和品牌方确认“不能做空洞推销”

博主在讲解里明确说，年轻人不太喜欢空洞的产品推销，能够戳中情绪的故事才是好广告。

这一步决定了这条片不会走传统饮品广告路线：

```text
口味清爽
果香浓郁
低度微醺
朋友聚会
年轻人都爱喝
```

而是先问：这个品类在年轻人生活里解决什么情绪？

### 2. 把品牌诉求翻译成情绪命题

品牌方想要表达的是：

- 轻微醺。
- 缓压力。
- 拒绝过度饮酒。
- 日常放空。
- 自愈。

博主没有把这些词直接写进旁白，而是转成一句更有故事感的命题：

```text
原来轻松这件事，不用等到谁批准。
```

这是本案例最值得迁移的地方：品牌词不能直接当脚本，必须翻译成用户能感到“这说的是我”的情绪判断。

### 3. 前半段铺压力，后半段给和解

讲解里说，他把年轻人从小到大的压力铺垫在影片前半段；前半段积攒所有情绪压力，后半段用一杯微醺果酒温柔和解。

结构可以拆成：

| 段落 | 情绪功能 | 画面/台词方向 |
| --- | --- | --- |
| 学生阶段 | 从小就被告知“以后就好了” | 教室、高考、家长/老师式鼓励 |
| 职场阶段 | 工作后仍然被目标追赶 | 加班、项目、同事说“做完就轻松了” |
| 觉察阶段 | 发现轻松一直由别人定义 | “我们的轻松，好像一直都是他们说” |
| 转向阶段 | 轻松也可以自己给 | 户外、朋友、湖边、产品出现 |
| 和解阶段 | 微醺不是逃避，而是短暂自愈 | 饮用、放空、自然光、温柔氛围 |

### 4. 用“目标群体看到自己的影子”作为脚本检查

博主提到，他反复打磨情节，希望目标群体能够看到自己的影子。

这给短视频规划一个很实用的检查标准：脚本不是看文案是否优美，而是看用户是否会觉得“这不就是我吗？”

### 5. 产品在后半段进入，不抢情绪铺垫

这条片前半段并不急着展示产品，而是先建立压力共鸣。产品在后半段作为和解和放空的道具出现。

这种入场方式适合低度酒、茶饮、香氛、零食、家居小物等情绪消费品：如果太早卖产品，观众会防备；先建立生活困境，产品才像自然解决方式。

## 原片故事结构

| 段落 | 台词/画面 | 情绪功能 | 方法意义 |
| --- | --- | --- | --- |
| 外部承诺 | “高考完就松了”“毕业以后就好了”“工作以后就好了” | 建立长期被延期的轻松 | 用人生阶段串联压力 |
| 职场压力 | “项目做完，下半天就轻松了” | 说明工作后仍在被催促 | 从学生压力延伸到职场压力 |
| 主体觉察 | “我们的轻松，好像一直都是他们说” | 让情绪从抱怨变成洞察 | 找到一句能击中用户的话 |
| 自我给予 | “也可以是自己给的” | 情绪转向 | 产品价值开始出现 |
| 产品和解 | 湖边、朋友、小酌、果酒 | 温柔放空 | 低度酒成为自我批准的仪式 |
| 主题收束 | “轻松这件事，不用等到谁批准” | 形成记忆点 | 品牌诉求变成金句 |

## 博主讲解中的方法节点

### 节点 1：五步制作流程

整条片被拆成五个核心步骤：

```text
脚本定位
-> AI 生图
-> 图生视频
-> 定制 AI 音乐
-> 剪辑合成
```

这说明即使是情绪软广，也不能跳过脚本定位。AI 生成只是后续执行，核心创意在前面。

### 节点 2：创意阶段跳出传统推销

博主主动和品牌方沟通，年轻人不喜欢空洞产品推销。这里不是单纯迎合审美，而是判断内容平台上的用户防备心：越像硬广，越难被年轻人接受。

### 节点 3：品牌内核要转成剧情结构

品牌内核是“轻微醺、缓压力、拒绝过度饮酒、日常放空、自愈”。剧情结构是“前半段压力、后半段和解”。这一步就是广告策划的核心翻译。

### 节点 4：情绪片也要反复打磨情节

情绪广告不能只靠抽象文案。博主反复打磨情节，让目标群体看到自己的影子。对微醺/治愈产品来说，情节越具体，广告越不像空泛鸡汤。

### 节点 5：产品价值要克制

这个案例强调拒绝过度饮酒，产品不是“喝到释放”，而是“一杯微醺的温柔和解”。克制感让品牌更高级，也降低酒类广告的冒犯风险。

## 可迁移灵感

### 适合迁移到哪些产品

- 低度酒、果酒、预调酒、无醇/低醇饮品。
- 茶饮、咖啡、气泡水、功能饮料。
- 香氛、精油、沐浴、睡眠产品。
- 轻零食、甜品、夜间放松产品。
- 面向年轻职场人的治愈系消费品。

### 可迁移结构

```text
品牌功能词
-> 翻译成用户情绪命题
-> 前半段铺压力来源
-> 中段出现主体觉察
-> 后半段用产品完成温柔和解
-> 用一句金句收束品牌内核
```

### 迁移示例

| 产品 | 不该只讲 | 可以转译成 |
| --- | --- | --- |
| 低度果酒 | 好喝、果香、低度 | 轻松不用等别人批准，今晚可以自己给 |
| 香氛 | 留香、前中后调 | 下班后把工作从房间里请出去 |
| 睡眠喷雾 | 助眠、草本 | 不必把白天的问题带进枕头里 |
| 气泡水 | 0 糖、清爽 | 忙到喘不过气时，给自己一个小气泡 |
| 小甜品 | 低卡、好吃 | 今天不需要很成功，也可以被奖励 |

## 反向拆解：如果做成硬广会怎样

错误做法可能是：

- 开头直接拍产品瓶身和倒酒。
- 旁白重复“低度、果香、微醺、治愈”。
- 用聚会狂欢表达喝酒。
- 把缓压力拍成逃避现实。
- 过度强调酒精刺激，削弱“拒绝过度饮酒”的品牌边界。

这种做法会让品牌落回普通饮品广告。这个案例的优势是，产品只是一杯，但它承接了年轻人长期被延期的轻松。

## Agent 调用模板

当 Agent 拿到低度酒、饮品、香氛、放松类产品时，可以调用本案例：

1. 品牌方给的功能词是什么？
2. 这些词背后的年轻人情绪是什么？
3. 用户的压力从哪里来，能否具体到人生阶段或日常场景？
4. 哪一句洞察能让用户觉得“这说的是我”？
5. 产品应在什么时候进入，避免像硬广？
6. 产品是解决问题、陪伴放空，还是完成仪式？
7. 需要哪些负向边界，例如不过度饮酒、不逃避现实、不制造焦虑？

输出结构：

```text
产品：
品牌功能词：
目标人群：
压力来源：
核心洞察句：
前半段压力铺垫：
后半段产品和解：
产品入场时机：
金句收束：
负向边界：
画面风格：
音乐情绪：
```

## 生成提示词方向

### 压力铺垫：教室

```text
竖屏真实电影广告画面，高中教室里，一个女生坐在课桌前低头写题，黑板上有倒计时和复习重点，老师站在讲台前，窗外是下午光线，画面克制真实，浅景深，青春压力感，不夸张表演
```

### 职场压力：加班

```text
竖屏写实商业广告画面，年轻女性坐在夜晚办公室工位前，电脑屏幕亮着，桌上有文件和水杯，同事从旁边走过，窗外城市灯光虚焦，疲惫但克制，真实职场压力，电影感低饱和
```

### 湖边放空：微醺和解

```text
竖屏真实生活方式广告画面，两个年轻女性坐在湖边草地野餐垫上，树荫和自然阳光，一瓶清爽低度果酒和透明酒杯放在桌布上，人物轻松聊天微笑，绿色清新治愈感，克制微醺，不要狂欢，不要夜店
```

### 产品特写

```text
竖屏商业广告特写，透明玻璃杯中倒入浅黄色低度果酒，气泡和果香感，旁边有绿色水果和白色桌布，湖边自然光，清爽治愈，真实摄影质感，品牌瓶身干净可见
```

## 误用边界

- 不要把低度酒拍成纵酒或逃避现实。
- 不要只讲“低度、好喝、果香”，要先找到用户情绪命题。
- 不要把年轻人压力拍成夸张卖惨，真实细节更有共鸣。
- 不要让产品过早入场破坏共情，产品应在和解段出现。
- 不要忘记“拒绝过度饮酒”的克制边界，微醺广告更适合温柔自愈而不是狂欢。
