## 案例结论

这不是一个“手机拍照好”的广告，而是一个“高价影像手机为什么值得进入高质量人群生活”的广告。

博主的核心判断是：**OPPO Find X9 Ultra 这种高价手机，用户购买时不只在意参数，更在意产品能不能承载自己的身份、生活和情绪价值。** 因此广告不能停留在漂亮 AI 模特、手机外观和参数展示，而要构建一个观众相信的世界：这个主角是谁，她为什么需要这部手机，她有哪些工作、家庭和情绪片段值得被保存。

最终，影像能力被翻译成一句情绪命题：

```text
每一次按下快门，都是在说：我不想忘记这一刻。
你舍不得忘记的，它记得。
```

## 产品诊断

| 维度 | 项目判断 |
| --- | --- |
| 产品 | OPPO Find X9 Ultra |
| 品类 | 高端影像旗舰手机 |
| 核心卖点 | 影像能力、哈苏资产、随身记录 |
| 价格带 | 高价位。博主明确说，如果只是四五千价位，漂亮 AI 模特概念图可能就够；但这个价位需要情绪价值。 |
| 目标人群 | 公司里管事的高质量职业人群，同时有家庭和孩子。 |
| 广告路线 | 高价身份/记忆型情绪广告 |
| 产品入场方式 | 先建立人物世界和人生片段，最后让手机成为保存记忆的容器。 |

## 构思路径复原

### 1. 先否定“漂亮 AI 模特图”

博主在 `ch15` 里提到，他一开始做过一些 OPPO 手机风格概念图，觉得也蛮好看。但问题是：这款手机定价很高，只靠漂亮 AI 模特并不能支撑购买理由。

这个判断很重要。它说明短视频规划不是“先做一张好看的图”，而是先判断：

- 产品价格带是否要求更高的情绪价值？
- 目标人群是否需要身份感和世界感？
- 好看画面是否能解释为什么这个产品值得买？

如果答案是否定的，就要从概念图退回产品定位。

### 2. 产品定位决定广告路线

博主明确说，第一步是产品定位，而且是在生图之前完成。对这款 OPPO 手机，他的判断是：

```text
高价影像旗舰
-> 用户在意的不只是配置
-> 更适合概念性、情绪性传递
-> 需要人物小传和可信世界
```

这就是这个项目最值得学的地方：AI 生成不是第一步，产品定位才是第一步。

### 3. 给人物写小传，而不是只生成模特

这条广告里的主角不是“一个漂亮女性”。她被设计成：

- 高质量都市职业女性。
- 至少是公司里管事的人。
- 工作中会被评价、被审视、被要求准备好。
- 工作后有独处的安静。
- 生活里有家庭和小孩。

观众看到她的工作碎片，会潜意识脑补她的职业身份；看到她回到家庭，会理解她除了职业角色，还有母亲和家庭角色。这样手机出现时，产品不再是孤立道具，而是进入了一个完整的人物世界。

### 4. 构建可信世界，让观众脑补故事

博主说“以前广告实拍怎么拍，现在就怎么样去做，AI 这个流程千万不要变”。这句话是这个项目的方法核心。

所谓可信世界，不是把背景做高级，而是让每个画面都能回答：

- 这个人是谁？
- 她在哪里？
- 她刚经历了什么？
- 她为什么有这个情绪？
- 她为什么需要记录？

原片用几个碎片建立世界：

- 城市黄昏和高楼：高压都市生活。
- 车内沉默：工作之后的切换和疲惫。
- 街头人群一瞥：关系或记忆的微妙瞬间。
- 办公场景和方案评价：职业身份与被审视感。
- 独处阅读：忙碌之后只属于自己的安静。
- 孩子画画和搭积木：家庭身份与亲子柔软。

### 5. 让手机在“情绪已经成立”后出现

原片不是开头就展示手机，而是先让观众感到这些片段都“不会重来”。当旁白说出 `These moments won't replay` 后，产品才承担“把它们留住”的角色。

产品入场链路是：

```text
工作/关系/家庭片段
-> 这些瞬间不会重来
-> 人会舍不得忘记
-> 手机把它们留在离心最近的地方
-> Find X9 Ultra / Hasselblad in your pocket
```

这就是它不像硬广的原因。产品不是被强行展示，而是在观众已经接受“我有需要保存的东西”之后出现。

## 导演与镜头语言拆解

这条片子的导演感来自“外部世界到内部关系”的镜头递进，而不是某个孤立的炫技运镜。

```text
城市/高楼/冷色
-> 职业女性背影、侧脸、街头和办公室
-> 独处窗边
-> 家庭暖光、孩子、地毯、手部动作
-> 手机取景界面
-> 品牌字卡
```

| 镜头段落 | 镜头任务 | 摄影选择 | 运镜/节奏 | 作用 |
| --- | --- | --- | --- | --- |
| 城市黄昏 | 建立高端都市世界和时间流逝 | 远景、城市天际线、冷暖交界 | 稳定或极慢运动 | 先给产品所属的生活世界 |
| 跑步/街头/职业形象 | 建立人物身份和外部压力 | 背影、中近景、侧脸、街头人群 | 跟拍或轻微推进 | 让她成为具体的人，而不是 AI 模特 |
| 办公室和窗边 | 建立克制、被审视、独处 | 中景、中近景、前景遮挡、窗光 | 静止或慢推 | 把职业压力转成内心状态 |
| 阳台/城市过渡 | 从外部身份切到私人空间 | 侧脸、负空间、城市虚化 | 慢节奏留白 | 为家庭段落做情绪缓冲 |
| 家庭暖光 | 建立亲子关系和柔软状态 | 中景、低角度生活视线、暖色、浅景深 | 多用静止，人物自然动 | 让“值得记录的瞬间”成立 |
| 手机取景 | 产品作为动作答案 | POV / 插入镜头、手部、屏幕 | 静止或轻微稳定运动 | 手机不是贴片，而是在保存记忆 |
| 片尾收束 | 价值命题落点 | 黑场字卡、产品名 | 节奏停下 | 把影像能力压成“它记得” |

可迁移的导演结构：

```text
先让观众相信人物世界
再让观众理解人物情绪
再让产品进入一个已经成立的动作
最后用品牌句收束产品价值
```

如果用 [[domains/视觉制作/06-AI视频/03-导演式镜头设计方法|导演式镜头设计方法]] 复盘，本片至少包含三组对比：

- 远近对比：城市远景到家庭近景，观众从世界进入关系。
- 冷暖对比：职场/城市冷感到家庭暖光，人物身份完成切换。
- 动静对比：外部通勤和街头移动，到家庭静止观察，情绪落地。

这说明高端情绪广告的“运镜”不是越多越好，而是镜头关系要能让观众感到人物生活正在转换。

## 博主讲解中的方法节点

### 节点 1：brief 和剧情

`ch01` 里博主说，基本是先 brief 和剧情，把自己当作广告导演。这里的 brief 不只是客户需求，而是将产品、人群、情绪和广告目标放到一起判断。

### 节点 2：分镜图片

不是先生成视频，而是根据想象的剧情设计相应分镜头图片。也就是说，AI 视频前面要有一个分镜图片阶段，用图片确认人物、场景、情绪和构图。

### 节点 3：先出图再做视频

先出图能提高控制力。对这种真实广告，人物身份、场景质感和画面连续性都不能完全交给图生视频模型随机发挥。

### 节点 4：音乐、配音、剪辑和声音设计

`ch15` 把流程拆成五步：分析产品与全系列图片、图生视频、AI 原创音乐、AI 配音、剪辑和声音设计。这个顺序说明他把声音也当成广告完整度的一部分，而不是最后随便配乐。

### 节点 5：产品定位在生图之前

这是最核心节点。没有定位，后面生成再漂亮也可能方向错。

## 可迁移灵感

### 可以借鉴的创意结构

```text
高价产品
-> 用户不只买功能
-> 需要一个匹配价格的人物世界
-> 让产品进入这个世界中的关键情绪时刻
-> 产品成为情绪价值的承载物
```

### 可以迁移到的产品

- 高端手机、相机、运动相机。
- 高价手表、首饰、香水、礼品。
- 家庭影像设备、云相册、相册打印服务。
- 婚礼摄影、旅拍、毕业照、纪念日影像服务。
- 高端家居、智能家电、车载影像系统。

### 迁移示例

| 产品 | 不该只讲 | 可以转译成 |
| --- | --- | --- |
| 高端香水 | 香味、留香时间 | 某个阶段的自己被气味保存 |
| 高端首饰 | 材质、工艺 | 关系里的承诺被随身带着 |
| 旅行箱 | 容量、轮子 | 每一次出发和归来都被装进去 |
| 婚鞋 | 舒适、防磨脚 | 人生仪式里的某一步被记住 |
| 高端相机 | 像素、镜头 | 家庭、成长、旅行和关系的记忆容器 |

## 反向拆解：如果做成硬广会怎样

错误做法可能是：

- 开头大特写手机。
- 讲哈苏、长焦、夜景、像素、芯片。
- 让 AI 模特拿着手机摆拍。
- 用大量产品参数做字幕。
- 最后说“记录美好生活”。

这种广告的问题是：所有高端手机都能这么说，观众不会记住它。这个 OPPO 项目的构思优势在于，它先让观众相信这个人物的生活，再让产品成为这个生活里合理出现的记忆容器。

## Agent 调用模板

当 Agent 拿到高价手机、相机或其他高客单情绪价值产品时，先按以下问题调用本案例：

1. 这个产品是否高价到不能只靠功能说服？
2. 目标用户是谁？她/他在生活里扮演哪些角色？
3. 能否写出 80-120 字人物小传？
4. 这个人物有哪些“舍不得忘记”的瞬间？
5. 产品应该在哪个情绪节点出现？
6. 最后的品牌记忆句是什么？

输出结构：

```text
产品定位：
目标人物小传：
可信世界设定：
三个记忆片段：
产品入场时刻：
片尾品牌句：
分镜表：
生图提示词：
图生视频提示词：
声音设计方向：
硬广感检查：
```

## 误用边界

- 不适合低价走量产品直接套用。低价产品通常更需要明确利益点和问题解决。
- 不适合没有情绪承载点的弱关注品类，除非先找到强人生场景。
- 不要把“情绪广告”理解为慢镜头、暗光、文艺旁白。关键是产品能力和人物生活之间有真实关系。
- 不要只复制都市女性和孩子。真正要复制的是“高价产品 -> 可信人物世界 -> 情绪节点 -> 产品入场”的结构。
