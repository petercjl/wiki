## 模型定位

Doubao-Seedance-2.0 系列是豆包大模型团队在火山方舟上的专业级多模态视频创作模型。控制台在 clipping 中标记为“全面开放”，核心定位是让创作者把文字、图片、视频、音频作为参考输入，生成、编辑或延长视频。

它不是只做“文生视频”的单一模型。更关键的能力是参考一致性：从参考素材里继承人物动作、镜头语言、物品细节、材质、音色、视效风格、运镜和角色特征，并把这些约束带入生成结果。

## 场景适配

clipping 明确把 Seedance 2.0 放在三个核心场景里：

| 场景 | 适配原因 | Agent 处理方式 |
| --- | --- | --- |
| 商业广告 | 需要产品细节、材质、风格、镜头和视听质感共同稳定。 | 把产品主体、替换对象、卖点动作、镜头语言和音效分别写成约束。 |
| 影视制作 | 需要复杂场景、运镜、多镜头叙事和角色一致性。 | 不要只写剧情，要写镜头路径、连续性、人物比例、物理规律和剪辑边界。 |
| 社交媒体营销 | 需要快速生成具有视觉冲击和音画完整度的内容。 | 更适合短视频片段、广告氛围片、产品展示和平台素材测试。 |

## 三个 demo 的可复用 prompt 模式

### 1. 多模态参考生成

原 demo 结构：

- 参考素材：视频 1、图像 1、图像 2、图像 3、音频 1。
- 目标：根据视频 1 的人物动作和镜头语言，生成图像 1 和图像 2 的打斗场面。
- 背景：打斗背景使用图像 3。
- 风格：打斗过程模仿像素游戏。
- 音频：背景音乐来自音频 1，并随打斗动作加入打斗音效。

原 prompt：

```text
参考@视频1的人物动作和镜头语言，生成@图像1和@图像2的打斗场面，打斗背景是@图像3，打斗的过程模仿像素游戏，背景音乐是@音频1中的音乐，随着打斗动作还有打斗音效。
```

可迁移规则：每个参考素材必须有明确职责。视频负责动作和镜头，图片负责主体和背景，音频负责音乐或音效，文字负责风格与叙事目标。

Agent 模板：

```text
参考@视频1的[动作/节奏/镜头语言]，使用@图像1的[主体A]和@图像2的[主体B]生成[互动场景]；
场景背景采用@图像3的[环境]；
整体风格为[风格]；
背景音乐参考@音频1，并在[关键动作]处生成匹配音效；
保持[主体特征/材质/比例/镜头连续性]。
```

### 2. 视频编辑定向替换

原 demo 结构：

- 参考素材：视频 1、图像 1。
- 目标：把视频 1 礼盒中的香水替换为图像 1 中的面霜。
- 保持项：动作和运镜不变。

原 prompt：

```text
将@视频1礼盒中的香水替换成@图像1中的面霜，动作和运镜不变
```

可迁移规则：视频编辑 prompt 的核心不是重新生成整段视频，而是精确指定“替换对象、参考对象、不可变因素”。如果不可变因素不写，模型可能同时改变动作、构图、镜头或场景。

Agent 模板：

```text
将@视频1中[位置/容器/人物手中]的[原对象]替换成@图像1中的[新对象]；
保持[动作/运镜/人物姿态/场景光线/背景/时长]不变；
新对象需要符合原视频的[透视/比例/材质反光/遮挡关系]。
```

### 3. 视频延长一镜到底

原 demo 结构：

- 参考素材：视频 1、音频 1。
- 目标：延长视频 1，以一镜到底方式从厨房门进入客厅，再从窗户推出到室外孩子放烟花。
- 氛围：新年喜庆、红灯笼、年味。
- 声音：背景音乐参考音频 1，背景语音为“新春快乐，阖家幸福，马年吉祥”。
- 约束：全程无剪辑断点、运镜丝滑、速度均匀、视觉连贯、沉浸、人物比例符合现实世界物理规律。

原 prompt：

```text
延长@视频1，一镜到底运镜，全程无剪辑断点，新年喜庆氛围感拉满；开篇@视频1画面，自然衔接慢拉镜头匀速穿过厨房门，顺滑过渡到客厅，一对夫妻正在客厅门口贴福字，镜头无缝摇移至贴窗花的客厅窗户处，紧接着慢推镜头从窗户向外穿出，流畅衔接孩子们在室外空地上放烟花的场景；全程运镜丝滑连贯、速度均匀不卡顿，画面融入红灯笼等新年元素，烘托浓厚过年氛围；背景音乐参考@音频1，背景语音为：“新春快乐，阖家幸福，马年吉祥”，整体保证一镜到底的视觉连贯性、沉浸感，年味与氛围感双重拉满，人物比例符合现实世界物理规律。
```

可迁移规则：延长视频时，要把“起点画面、镜头路径、空间过渡、场景事件、氛围元素、音频、语音、连续性和物理约束”一次写完整。尤其是一镜到底需求，要明确“无剪辑断点、速度均匀、比例真实”，否则延长段容易出现跳变或空间错乱。

Agent 模板：

```text
延长@视频1，从原画面自然衔接到[新场景路径]；
镜头方式为[一镜到底/慢推/慢拉/摇移/跟拍]，全程[无剪辑断点/速度均匀/不卡顿]；
依次出现：[事件1] -> [事件2] -> [事件3]；
画面加入[氛围元素]，形成[情绪/节日/品牌调性]；
背景音乐参考@音频1，语音内容为：“[旁白/祝福/口播]”；
保持视觉连贯、空间关系清晰、人物比例符合现实世界物理规律。
```

## 生产使用规则

1. 先判断任务属于生成、编辑、延长还是首尾帧生成，再选择 prompt 结构。
2. 多模态输入不是越多越好；每个参考素材都必须有职责，否则会互相干扰。
3. 商品类视频要把产品细节、材质、比例、包装、替换边界和光线关系写清楚。
4. 视频编辑任务必须写“保持不变”的对象，尤其是动作、运镜、人物姿态、背景和光线。
5. 视频延长任务必须写起点、路径、过渡、连续性和物理规律。
6. 需要代码集成时，本页只提供入口线索；必须继续读取 Volcengine Ark 的 Seedance 2.0 API 参考和 SDK 示例。
