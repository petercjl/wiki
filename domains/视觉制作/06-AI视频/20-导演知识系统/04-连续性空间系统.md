# 连续性空间系统

来源锚点：Ch6 Composing Shots: Spatial Connections。

## 连续性空间的目标

连续性空间不是保守规则，而是让观众相信多个镜头发生在同一个可理解空间里。AI 视频分镜尤其需要主动维护空间连续，因为每个镜头可能由模型独立生成。

## 景别系统

| 景别 | 导演功能 | AI 视频使用 |
| --- | --- | --- |
| 远景/全景 | 建立地点、身体动作、人与环境关系 | 开场、空间交代、产品使用环境 |
| 中景 | 日常动作、对话、人物任务 | 生活流、人物使用产品 |
| 近景 | 表情、情绪、关键反应 | 情绪转折、人物内心 |
| 特写/大特写 | 细节、手部、产品证据 | 功能、材质、屏幕、按钮 |
| 插入镜头 | 解释一个关键动作或物件 | 产品入场和剪辑桥 |

景别不是大小选择，而是观众距离选择。过多近景会丢空间，过多远景会丢情绪和证据。

## 轴线系统

Katz 讲 line of action / 180-degree rule 的核心，是保持屏幕方向和视线方向。

| 规则 | 意义 | AI 视频做法 |
| --- | --- | --- |
| 建立轴线 | 确定人物 A/B、运动方向、视线方向 | 分镜表写清左右和视线 |
| 保持同侧拍摄 | 观众不迷失方向 | 同一场景 prompt 固定 left/right |
| pivot shot | 用转头、转身、看向新对象建立新轴线 | 换方向前加桥接镜头 |
| reestablishing shot | 重新交代空间 | 多镜头后加中远景 |

AI 生成常见错误是人物方向在每个镜头里随机变化。解决方法不是多写“consistent”，而是在导演表里固定轴线。

## 三角机位系统

两人对话或关系镜头可以用三角机位组织：

| 机位 | 用途 |
| --- | --- |
| master two-shot | 交代两人关系和空间 |
| angular single | 突出某一方反应 |
| over-the-shoulder | 关系、观看、对话 |
| POV single | 让观众靠近某一方主观视线 |
| profile | 强调对峙、平等、距离 |

AI 视频不一定要全部生成，但至少要知道这些 coverage 选项。

## 视觉识别和叙事推理

观众连接镜头靠两件事：

1. 视觉识别：近景中的手、门把手、产品来自前一镜头的空间。
2. 叙事推理：即使前一镜头没看清门把手，观众也能理解“走到门前 -> 手转门把手”。

AI 视频中，插入镜头要建立在这种因果上。不要突然插入一个产品特写，除非前一镜头已经让观众知道人物为什么要使用它。

## AI 视频检查清单

- 人物左右是否一致。
- 行进方向是否一致。
- 视线是否能接到被看对象。
- 产品特写是否来自已建立空间。
- 是否有必要的 reestablishing shot。
- 是否每个近景都有空间归属。
