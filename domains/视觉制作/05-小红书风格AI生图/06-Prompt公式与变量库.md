# Prompt 公式与变量库

## 通用公式

```text
小红书风格商业种草图片，竖版 3:4，手机端高可读性，主体是【产品/人物/场景】，面向【目标人群】，表达【痛点/结果/场景/卖点】，画面有明确品类信号和使用场景，真实生活感，干净明亮，轻商业感，柔和自然光，高质感产品摄影，画面中心突出一个主视觉，背景简洁不杂乱，适合小红书封面，留出上方或中间文字区域，中文大标题排版清晰醒目，标题文案：【封面文案】
```

## 一键组合模板

```text
小红书风格【笔记类型】图片，竖版 3:4，主题是【品类/产品】，目标人群是【人群】，使用场景是【场景】，核心痛点是【痛点】，核心结果是【结果】。画面需要有明确品类信号和真实场景代入，一个主视觉，干净明亮，真实生活感，高质感产品摄影，手机端高可读性，适合小红书封面。中文标题：【标题】。排版要求：大标题清晰，重点词用红色或黑色加粗，留出文字安全区，少量实用标签。图片逻辑：不是装饰图，而是帮助用户快速理解【为什么点开/为什么收藏/为什么相信】。负面提示：不要杂乱、不要硬广、不要密集小字、不要错误中文、不要低清晰度、不要虚假夸张效果。
```

## 变量库

### 人群词

学生党、上班族、新手妈妈、敏感肌、租房党、养宠家庭、装修人群、通勤族、备婚人群、30岁女性、精致懒人、预算有限但想要品质的人。

### 场景词

通勤、宿舍、办公室、旅行、居家、厨房、浴室、卧室、育儿、换季、约会、健身、露营、搬家、装修、送礼、年货、节日聚会。

### 痛点词

踩坑、不会选、浪费钱、不实用、不舒服、效果不明显、太麻烦、显廉价、收纳混乱、用完闲置、敏感不耐受、没有氛围感。

### 效果词

干净、显高级、省空间、省时间、舒服、好打理、有氛围感、显白、保暖、收纳力强、质感提升、精致、安心、方便、耐用。

### 风格词

真实生活感、干净明亮、轻法式、奶油风、日杂风、韩系清透、极简高级、自然松弛、温柔治愈、精致实用、复古胶片感、高质感商业摄影。

### 图片逻辑词

一个主视觉、明确品类信号、真实使用场景、痛点可视化、前后对比、细节特写、清单式排版、关键词醒目、手机端可读、适合收藏。

## 变量组合建议

为了避免 prompt 像填空题一样机械，建议按下面结构组合：

```text
目标人群 = 身份 + 阶段 + 需求强度
使用场景 = 空间 + 时间节点 + 任务
核心痛点 = 选择痛点 + 使用痛点 + 结果痛点
核心结果 = 功能结果 + 情绪结果 + 展示结果
证据表达 = 细节 + 对比 + 步骤 + 清单
```

## 负面提示词

```text
不要杂乱背景，不要过度广告感，不要硬广海报，不要夸张表情，不要廉价电商详情页，不要文字过多，不要小字密集，不要主体不清，不要过度磨皮，不要虚假效果，不要低清晰度，不要混乱拼贴，不要无关装饰，不要过度滤镜，不要 AI 感，不要错误中文，不要乱码
```
