# 视觉记忆符号与重复机制

## 问题定义

原书反复强调：视觉的终极目标是在买家心中形成强烈的记忆符号。对电商而言，这不是广告口号，而是降低传播成本、提升复购、提高品牌词搜索和支撑客单价的经营动作。

## 核心模型

品牌记忆由三件事构成：

1. **可识别元素**：名字、LOGO、色彩、字体、SLOGAN、品牌活体、摄影符号。
2. **稳定重复**：在不同触点持续出现，而不是每次换风格。
3. **意义绑定**：元素不仅出现，还要和人群、场景、利益、情绪绑定。

只有元素没有意义，会变成装饰；只有意义没有重复，会难以记住；只有重复没有统一，会变成噪音。

## 记忆符号的来源

| 来源 | 例子 | 作用 |
| --- | --- | --- |
| 语言符号 | SLOGAN、品牌口头禅、客服称呼 | 让消费者能复述 |
| 图形符号 | LOGO、品牌活体、标签形状 | 让消费者能识别 |
| 色彩符号 | 主色、系列色 | 让消费者快速扫到 |
| 人物符号 | 模特、创始人、卡通形象 | 让品牌人格化 |
| 场景符号 | 固定生活方式、摄影场景 | 让消费者代入 |
| 体验符号 | 包装、赠品、开箱方式、气味 | 让购买后继续记住 |

## 重复机制

原书用“重复 21 次”和脑白金广告语说明重复的重要性。正式使用时，不必机械相信具体次数，而应保留方法：

- 品牌元素要在足够多触点重复；
- 重复对象必须稳定；
- 重复不是复制粘贴，而是让消费者在不同场景下识别同一个品牌；
- 视觉定位确定后，不应频繁推翻。

## 操作流程

### 1. 选出记忆主符号

每个品牌都要明确一个或几个最重要的记忆锚点：

- 是品牌名？
- 是颜色？
- 是人物？
- 是一句话？
- 是固定场景？
- 是包装体验？

不能同时追求所有元素都成为主符号，否则消费者无法抓住重点。

### 2. 建立触点重复表

| 触点 | 是否出现主符号 | 是否一致 | 是否能理解含义 |
| --- | --- | --- | --- |
| 搜索/推荐主图 |  |  |  |
| 广告入口图 |  |  |  |
| 店铺首页 |  |  |  |
| 详情页首屏 |  |  |  |
| 详情页证据图 |  |  |  |
| 包装 |  |  |  |
| 售后卡/赠品 |  |  |  |
| 客服话术 |  |  |  |
| 私域内容 |  |  |  |

### 3. 区分“重复”和“烦人”

有效重复是让消费者在关键决策点快速识别和理解，而不是把 LOGO 到处放大。

判断方式：

- 是否影响产品信息阅读？
- 是否让页面显得廉价或噪音过多？
- 是否和场景自然融合？
- 是否能帮助消费者复述品牌差异？

## 案例与证据锚点

- 脑白金：通过高频重复让 SLOGAN 几乎无法忘记。
- 茵曼：大辫子、民族风、棉麻场景形成摄影记忆符号，即使换模特和场景，消费者仍能识别。
- 三只松鼠：品牌活体和“主人”称呼进入客服和组织文化，形成可感知人格。
- 相宜本草：绿色主色在搜索和包装中形成识别。

## 与 AI 生图的关系

AI 时代更需要记忆符号，因为 AI 很容易生成“看起来不错但没有品牌资产”的泛风格图。

给 AI 的品牌视觉约束应包括：

- 主色和辅助色；
- 固定人物或品牌活体；
- 场景边界；
- 材质和光线；
- 标志性道具；
- 文案语气；
- 禁止变化的元素。

## Agent 使用提示

当 Agent 被要求“让品牌更有记忆点”时，应输出：

1. 当前已有可继承记忆资产；
2. 需要选择的主符号；
3. 应重复的触点；
4. 不应频繁变化的元素；
5. AI 生图 prompt 中必须固定的品牌资产。
