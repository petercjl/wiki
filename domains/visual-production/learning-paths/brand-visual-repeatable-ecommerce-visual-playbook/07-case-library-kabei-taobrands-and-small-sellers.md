# 案例库：卡贝、淘品牌与中小卖家

## 用法

本页不是案例故事摘抄，而是把原书中的典型案例转成可复用判断规则。Agent 回答电商品牌视觉问题时，可以用这些案例作为模式锚点。

## 案例 1：卡贝——视觉标准提升团队效率

### 原书情境

作者作为卡贝官方旗舰店品牌视觉顾问，两年内帮助其建立视觉标准体系。原书描述：过去美工两天完成的工作，建立标准后半天可完成，效果更好。

### 可复用规则

- 视觉标准能把重复性设计从“每次重新想”变成“按标准组合”。
- 标准不是只给美工用，运营、老板、分销商都要理解。
- 只有标准不够，还要学习、执行和复盘。

### 诊断问题

- 店铺是否有统一视觉标准文档？
- 新活动是否每次从零设计？
- 分销商或渠道素材是否会破坏品牌统一？

## 案例 2：王小帮农家店——品牌感不等于高大上

### 原书情境

王小帮农家店视觉“土”，但符合原生态农产品人群对真实、亲切、纯朴的期待，并持续重复店铺名、LOGO、SLOGAN、字体、颜色、拍摄风格。

### 可复用规则

- 品牌感不是高端感，而是目标人群相信的稳定风格。
- 低客单、农产品、下沉市场或强人设品类，不应盲目做“高级感”。
- 店主/创始人形象本身也可以成为品牌活体。

## 案例 3：德力西爱阁专卖店——代理品牌也要建设店铺品牌

### 原书情境

德力西专卖店对外素材使用“DELIXI 德力西”官方品牌吸引流量，店内强调“德力西爱阁专卖店”，让消费者记住自己的店铺。

### 可复用规则

- 专营店/专卖店不能只替上游品牌做嫁衣。
- 店外靠授权品牌获得信任，店内要沉淀自己的店铺识别。
- 需要区分“引流品牌”和“复购品牌”。

## 案例 4：御道艾灸盒——视觉差异化来自业务理解

### 原书情境

养生器材店铺页面凌乱，甚至穴位标注错误。作者建议团队学习《黄帝内经》，因为卖养生器材至少要懂养生。

### 可复用规则

- 视觉差异化不能脱离品类知识。
- 专业品类的视觉信任来自专业内容、术语准确、场景真实和文化一致。
- 外包视觉如果不懂业务，只能做表层美化。

## 案例 5：茵曼——摄影符号可跨模特和场景延续

### 原书情境

茵曼通过民族风、棉麻、大辫子等摄影元素建立记忆。即使换模特、换场景，只要关键符号不变，消费者仍能识别。

### 可复用规则

- 摄影标准应定义“不变元素”和“可变元素”。
- 模特不一定永远同一人，但气质、发型、场景符号应延续。
- 服饰、家居、生活方式品类尤其依赖摄影记忆。

## 案例 6：三只松鼠——品牌活体进入组织和客服

### 原书情境

三只松鼠不仅有松鼠形象，还把客服称呼、员工花名、品牌人格和消费者互动统一为“主人文化”。

### 可复用规则

- 品牌活体不是吉祥物图片，而是品牌人格系统。
- 活体要进入客服语言、组织文化、内容表达和消费者关系。
- 有争议的语言风格必须与品牌人格一致，否则会显得油腻或刻意。

## 案例 7：阿芙、脑白金、劲酒——一句话绑定定位

### 可复用规则

- “阿芙就是精油”：明确品类心智。
- “今年过节不收礼，收礼只收脑白金”：明确使用场景并通过重复形成记忆。
- “劲酒虽好，可不要贪杯哦”：用关心健康强化品牌好感。

SLOGAN 不应只追求文艺，而要帮助消费者复述品牌差异。

## Agent 使用提示

引用案例时，不要机械搬用案例本身。应先判断用户场景属于哪类模式：

- 团队效率问题 → 卡贝；
- 低价/土味但真实人群 → 王小帮；
- 代理品牌/专卖店 → 德力西爱阁；
- 专业品类信任 → 御道艾灸盒；
- 生活方式摄影 → 茵曼；
- 品牌人格/客服语言 → 三只松鼠；
- SLOGAN/重复 → 阿芙、脑白金、劲酒。
