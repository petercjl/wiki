# 杉居情绪板解析

情绪板原图：`raw/assets/visual-style/minimal-scandinavian/variants/shanju-light-kitchen-living/shanju-moodboard.png`

## 画面结构

情绪板由 4 条横向画面构成，左侧是编号、品牌名、英文小字和中文短句，右侧是明亮厨房中的人物、猫、烘焙动作和食物。

反复出现的文字：

- `SHANJU / 杉居`
- `LIGHT HOME BAKING`
- `CAT & KITCHEN`
- `live gently.`

中文短句：

- 光与厨房 / 有猫 有你 有生活
- 怡到好处的温度 / 安心托付 每一份热爱
- 刚出炉的松软 / 分享给最重要的它
- 简单的幸福 / 生活本该这样温柔

## 空间观察

情绪板主空间是明亮白色厨房：

- 大面积白色墙面、台面和柜体。
- 浅木开放架、浅木凳、浅木托盘。
- 自然窗光，高亮但不刺眼。
- 白色陶瓷、玻璃杯、花瓶、鲜花、绿色植物。
- 烤箱、水槽、台面和食物动作让空间有真实功能。

这张情绪板是杉居的“室内厨房基准”，不是空间边界。杉居的产品如果包括烧烤盘、调味罐、厨房工具和餐厨收纳，应从厨房延展到露台、庭院、野餐桌和户外轻烧烤。

## 人物观察

人物是年轻女性，状态松弛、干净、温柔：

- 浅蓝宽松衬衫。
- 自然盘发。
- 表情轻柔，不强摆拍。
- 行为是倒水、烘焙、吃面包、与猫互动。

她不是网红模特，也不是精致女主人。她代表愿意认真生活、愿意把温柔分给家和猫的人。

## 橘猫观察

橘猫是高记忆资产。它不是装饰道具，而是品牌关系中的第三个角色。

橘猫的作用：

- 情绪：增加温暖、陪伴、松弛。
- 记忆：让用户记住杉居，而不只是记住白色厨房。
- 场景：把“家”具体化为有生命互动的空间。

## 色彩观察

主色：

- 哑光白 / 暖白：厨房、墙面、台面。
- 浅木色：开放架、托盘、凳子。
- 浅蓝：人物衬衫，是区别于普通米白北欧风的重要识别色。
- 鼠尾草绿：手套、植物、小厨具。
- 橘猫暖橙：天然暖色记忆点。
- 面包奶油黄：烘焙和轻料理的食物暖色。

杉居的色彩比通用极简北欧风更有角色。通用北欧风可以只有白木绿；杉居必须保留浅蓝人物、橘猫暖橙和厨房工具的食物暖色。
