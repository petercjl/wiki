# 小红书生图 Prompt 模板库

本页保存可直接调用的 prompt 骨架。使用前先确认：内容场景、封面类型、目标人群、使用场景、核心痛点、核心结果、证据形式。

## 生图前填写表

```text
产品/品类：
目标人群：
使用场景：
核心痛点：
核心痒点：
核心结果：
卖点类型：兴趣型 / 功能型 / 场景型
内容场景：信息流 / 搜索
关键词：品牌产品词、特征词、类目词、场景词、人群词、竞品词
封面标题：
需要呈现的证据：细节、对比、步骤、清单、真人使用、评论反馈
不想要的画面：
```

## 通用商业种草图

```text
小红书风格商业种草图片，竖版 3:4，手机端高可读性，主体是【产品/人物/场景】，面向【目标人群】，表达【痛点/结果/场景/卖点】，画面有明确品类信号和使用场景，真实生活感，干净明亮，轻商业感，柔和自然光，高质感产品摄影，画面中心突出一个主视觉，背景简洁不杂乱，适合小红书封面，留出上方或中间文字区域，中文大标题排版清晰醒目，标题文案：【封面文案】。负面提示：不要杂乱背景，不要硬广海报，不要文字过多，不要错误中文，不要 AI 感。
```

## 信息流种草封面

```text
小红书信息流种草封面，竖版 3:4，真实生活场景，主体是【产品/人物/空间】，目标人群是【人群】，画面呈现【具体场景】里的自然使用瞬间，弱营销感，不像广告摆拍，有兴趣点和轻微反差，产品露出清晰但不硬，融入【热点/季节/生活方式】元素。画面明亮干净，一个主视觉，背景有少量真实生活细节，手机端一眼能看懂。中文大标题：【场景化标题】，标题突出【人群词/场景词/痛点词】，保留文字安全区。负面提示：不要电商详情页，不要强促销，不要品牌硬广，不要复杂拼贴，不要密集小字。
```

## 搜索承接封面

```text
小红书搜索承接封面，竖版 3:4，主题是【品类/产品】，目标用户正在搜索【关键词】，画面直接回答【痛点/选购问题】，包含清晰的产品/方案展示和 3-5 个可读要点。风格真实、干净、可信，像专业博主整理的实用攻略，标题：【答案型标题】，副标题：【利益点/避坑点】，画面中出现【品类词】、【人群词】、【场景词】或【功效词】标签，手机端高可读性，适合收藏。负面提示：不要空泛氛围图，不要只拍美图不讲信息，不要错误中文，不要小字过多。
```

## 痛点型封面

```text
小红书爆款封面风格，竖版 3:4，真实生活场景，主体是【产品/问题场景】，面向【人群】，画面突出【具体痛点】，情绪真实但不夸张，明亮干净，高对比但不刺眼，主体清晰，背景少量生活细节，手机端一眼可读，顶部大字标题：【痛点标题】，副标题：【解决方向】，红色或黑色重点字强调，避免杂乱拼贴
```

## 结果型封面

```text
小红书种草封面，竖版 3:4，主体是【产品】和【使用后结果】，画面展示清晰可见的变化或理想状态，真实自然，不像广告大片，干净背景，柔和光线，质感高级，突出一个核心结果：【结果】，中文标题醒目，标题：【结果型文案】，保留文字安全区，移动端清晰
```

## 场景型封面

```text
小红书生活方式种草图，竖版 3:4，真实【场景】中正在使用【产品】，人物或手部动作自然，场景有代入感，适合【人群】，画面明亮、温暖、干净，产品可见且不生硬，突出【场景需求】，标题区域清晰，中文大标题：【场景标题】，整体真实、有用、可收藏
```

## 清单/攻略型封面

```text
小红书攻略封面，竖版 3:4，信息图和实拍结合，主体是【品类/产品集合】，排版清楚，有 3-5 个要点模块，白底或浅色背景，红色重点标注，黑色清晰中文标题，适合收藏，手机端可读，标题：【攻略标题】，包含关键词：【品类词】、【人群词】、【场景词】，整体像真实博主整理的实用攻略
```

## 对比测评型封面

```text
小红书测评封面，竖版 3:4，左右或上下对比构图，展示【产品A】和【产品B/使用前后/不同方案】，清晰标签区分，真实测评感，桌面自然光，背景简洁，重点突出差异，中文标题醒目：【对比标题】，有小标签：【适合人群】、【核心差异】，画面可信、克制、不夸张
```

## 评论区转化感总结图

```text
小红书总结收藏图，竖版 3:4，主题是【产品/品类】的购买理由总结，目标人群是【人群】。画面用清单式排版总结：适合谁、解决什么、怎么选、注意什么。语气真实克制，不强卖，像博主把踩坑经验整理给朋友。标题：【收藏型标题】，重点标签：【利益点1】、【利益点2】、【避坑点】。背景干净，信息层级清晰，手机端可读。
```

## 行业示例

### 美妆护肤

```text
小红书种草封面图，竖版 3:4，主题是敏感肌换季修护面霜，目标人群是敏感肌女生，使用场景是换季护肤，核心痛点是泛红干痒、不会选修护面霜，核心结果是皮肤稳定、妆前不卡粉。画面展示干净浴室台面上的面霜和自然肤感女性半身，柔和自然光，韩系清透，真实生活感，一个主视觉，手机端高可读性。中文标题：敏感肌换季别乱买面霜。副标题：先看这 3 个修护点。重点词红色加粗，背景简洁，适合小红书封面。不要硬广，不要夸张磨皮，不要错误中文。
```

### 家居收纳

```text
小红书攻略封面，竖版 3:4，主题是租房卧室收纳，目标人群是租房党和学生党，场景是小卧室衣物收纳，核心痛点是空间乱、东西找不到、买了收纳盒也不好用，核心结果是卧室干净、省空间、好拿取。画面为真实小卧室一角，浅色床品、衣柜、收纳盒整齐摆放，干净明亮，日杂风，信息图和实拍结合，标题：租房卧室收纳别再乱买。包含 4 个清单标签：省空间、好拿取、不显乱、不闲置。手机端可读，适合收藏，不要杂乱背景，不要廉价详情页风。
```

### 母婴用品

```text
小红书测评封面，竖版 3:4，主题是宝宝恒温水壶选购，目标人群是新手妈妈，场景是夜间冲奶，核心痛点是温度不准、操作麻烦、夜里手忙脚乱，核心结果是夜奶更省心。画面展示厨房台面上的恒温水壶、奶瓶和柔和夜灯，真实家庭场景，温暖自然光，主体清晰，左右对比小模块：普通水壶 vs 恒温水壶。中文标题：新手妈妈选恒温壶先看这 5 点。排版清晰，红色重点标注，可信测评感，不要夸张广告，不要过度商业海报。
```

### 服饰穿搭

```text
小红书生活方式封面，竖版 3:4，主题是秋冬通勤大衣穿搭，目标人群是 30 岁上班族女性，场景是办公室通勤，核心痛点是穿得臃肿、不显精神、显廉价，核心结果是显高级、保暖、好搭配。画面为真实街拍感，女性穿浅灰色羊毛大衣走在城市街道，干净背景，自然光，轻法式，松弛高级，一个主视觉。中文标题：30岁通勤大衣别乱选。副标题：显高级看这 3 个细节。手机端标题清晰，不要过度摆拍，不要杂乱背景。
```
