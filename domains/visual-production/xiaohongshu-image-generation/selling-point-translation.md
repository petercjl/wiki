# 小红书卖点转译

小红书生图不能直接罗列说明书式卖点。卖点必须转成用户能在画面里看懂的“人群、场景、问题、结果、证据”。

## 转译公式

```text
产品卖点 → 目标人群 → 使用场景 → 原始痛点 → 使用后结果 → 可视化证据 → 图上短句
```

示例：

| 说明书式卖点 | 小红书画面语言 |
| --- | --- |
| 容量大 | 租房小卧室衣柜终于不乱了，一眼能拿到衣服 |
| 恒温准确 | 新手妈妈夜里冲奶不用手忙脚乱 |
| 面霜修护 | 换季敏感肌不再泛红干痒，妆前也不卡粉 |
| 大衣面料好 | 30 岁通勤穿得保暖但不臃肿，看起来更精神 |

## 五类卖点表达

### 1. 兴趣型卖点

用于制造停留和记忆点，适合信息流素材。

```text
小红书兴趣种草图，竖版 3:4，主题是【产品】，突出【有趣/差异化/有记忆点的卖点】，画面有轻微反差或惊喜感，让用户愿意停留。场景是【具体生活场景】，人物状态自然，产品与场景融合，标题：【没想到/原来/谁懂】式表达，整体真实、有记忆点、弱营销感。
```

### 2. 功能型卖点

用于解释产品如何解决问题，适合细节图、证据图、搜索承接图。

```text
小红书功能解释图，竖版 3:4，主题是【产品功能/成分/功效】，画面用细节特写、局部放大、前后对比或步骤示意展示【功能如何解决痛点】。背景干净，文字标注 2-3 个关键证据点，标题：【痛点】看这个【功能点】，风格可信、克制、专业但不冰冷。
```

### 3. 场景型卖点

用于表达产品进入具体生活场景后的价值。

```text
小红书场景种草图，竖版 3:4，主题是【产品】在【使用场景】中的真实体验，目标人群是【人群】，画面突出【场景需求】和【使用后的好处】，人物动作自然，产品露出不突兀。标题：【人群】在【场景】真的需要它，整体生活化、真实代入、可收藏。
```

### 4. 证据型卖点

用于建立可信度。常见证据包括：细节特写、结构示意、材质对比、使用步骤、清单标注、前后对比。

适合图位：组图第 4 张、搜索承接封面、对比测评封面。

### 5. 转化型卖点

用于帮助用户判断“适不适合我”。常见表达包括：适合谁、解决什么、怎么选、注意什么。

适合图位：组图最后一张或评论区转化感总结图。

## 转译检查

一个卖点转译合格，至少要满足：

- 不只说产品属性，还说用户结果。
- 不只说功能强，还说解决哪个痛点。
- 不只说适合谁，还要出现具体场景。
- 不只追求好看，还要有证据或判断依据。
