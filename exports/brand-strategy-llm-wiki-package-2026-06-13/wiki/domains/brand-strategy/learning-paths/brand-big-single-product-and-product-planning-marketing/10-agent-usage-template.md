# 10｜Agent 使用模板：品牌大单品与企划营销诊断

## 适用任务

当 Agent 需要为品牌规划报告输出大单品企划、产品差异化、卖点包装、命名建议、内容策略和直播表达时，应使用本模板。

本模板更偏执行企划，与“心智产品力与大单品战略”路径互补：前者判断大单品是什么，本文判断大单品怎么企划和打穿。

## 输入清单

1. 店铺商品数据和 Top SKU。
2. 核心商品的主图、详情页、标题、价格和卖点。
3. 竞品商品、价格、主图、详情页和内容素材。
4. 用户评论、搜索词、客服问题和达人内容。
5. 供应链、毛利、可改造空间和新品计划。

## 诊断流程

第一步，明确战略目标：

1. 是防守竞品，还是抢新人群。
2. 是做第二增长曲线，还是抬高价格带。
3. 是做销量爆发，还是沉淀长期心智。

第二步，判断产品角色：

1. 流量入口款。
2. 利润款。
3. 形象款。
4. 防守款。
5. 战略大单品。
6. 场景延展款。

第三步，设计差异化：

1. 人群差异。
2. 场景差异。
3. 感官体验差异。
4. 加体验或减负担。
5. 跨品类心智嫁接。

第四步，翻译卖点：

1. 把产品语言转为消费者语言。
2. 把功能卖点转为购买理由。
3. 把抽象优势转为可视证据。
4. 把技术词转为昵称、场景词或结果词。

第五步，设计传播：

1. 主图第一眼卖点。
2. 详情页核心结构。
3. 短视频前三秒动作。
4. 直播间演绎脚本。
5. 小红书种草话题。

第六步，验证放大：

1. 原点用户是否听得懂。
2. 用户是否愿意为差异付费。
3. 内容测试是否能打穿一个卖点。
4. 搜索词和品牌词是否有增长。

## 输出结构

Agent 输出时建议包含：

1. 大单品战略任务。
2. 产品矩阵角色分工。
3. 差异化机会。
4. 核心卖点翻译。
5. 昵称化命名方向。
6. 可视化演绎方案。
7. 详情页与直播话术框架。
8. 用户验证计划。

## Agent 提示词

```text
你是品牌大单品企划顾问。请基于输入的商品数据、视觉素材、竞品和用户反馈，为该品牌提出大单品企划与产品营销方案。

请按以下结构输出：
1. 明确这款大单品承担的战略任务。
2. 判断它在产品矩阵中的角色。
3. 找出当前同质化问题，并提出可感知差异化。
4. 用“加体验/减负担”重写产品创新方向。
5. 把核心卖点翻译成消费者买点。
6. 给出 3-5 个昵称化命名或系列名方向。
7. 给出主图、详情页、短视频、直播间的可视化表达。
8. 给出用户验证和内容测试指标。

要求：不要堆卖点；聚焦一个最可能打穿的核心心智，并说明为什么这个心智能支撑传播和溢价。
```
