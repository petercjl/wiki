# 电商图片营销策划查询入口

当 Agent 要回答以下问题时，先读本入口：

- 帮商品策划主图、详情页、卖点图、证据图、场景图。
- 判断一张电商图为什么不转化。
- 把产品卖点转成电商图片文案和视觉方案。
- 为 AI 生图工具写电商主图/详情页/场景图 prompt。
- 判断不同人群、品类、价格带应该用什么图片策略。

## 推荐输出结构

```text
1. 目标用户与购买情境
2. FABE 卖点转译
3. USP 主卖点
4. 图片组结构：主图/痛点/卖点/对比/场景/证据/规格/促销
5. 每张图的营销任务、核心文案、视觉表达、证据素材
6. AI 生图 prompt
7. QA 检查与风险点
```

## 快速判断句

有效电商图片不是“产品照片 + 漂亮背景 + 几句文案”，而是基于目标用户、购买动机、竞争差异和信任证据设计出来的视觉销售系统。
