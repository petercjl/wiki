# Peter's LLM Wiki

这是 Peter 的个人 LLM Wiki 公共知识库。

路径：`/Users/pechen/wiki`

采用“一库多域”架构：

- `domains/`：长期专业知识域，例如电商运营、视觉制作、品牌策划。
- `shared/`：跨领域共享知识，例如消费者心理、平台规则、AI Agent 工作流。
- `projects/`：具体项目知识，例如 SealSeek、玺承 BI、OpenClaw、Hermes。
- `raw/`：原始资料，只读，不直接修改。
- `inbox/`：Web Clipper、临时笔记和待整理资料。

所有 Agent 在修改前必须先读 `AGENTS.md`、`SCHEMA.md`、`index.md`、`log.md` 最近记录。

Obsidian 可以直接打开本目录作为 Vault。
